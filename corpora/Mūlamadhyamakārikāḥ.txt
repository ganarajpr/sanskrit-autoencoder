na svato nāpi parato na dvābhyāṃ nāpyahetutaḥ / (1.1)
	utpannā jātu vidyante bhāvāḥ kvacana kecana // (1.2)
	catvāraḥ pratyayā hetur ārambaṇam anantaram / (2.1)
	tathaivādhipateyaṃ ca pratyayo nāsti pañcamaḥ // (2.2)
	na hi svabhāvo bhāvānāṃ pratyayādiṣu vidyate / (3.1)
	avidyamāne svabhāve parabhāvo na vidyate // (3.2)
	kriyā na pratyayavatī nāpratyayavatī kriyā / (4.1)
	pratyayā nākriyāvantaḥ kriyāvantaśca santyuta // (4.2)
	utpadyate pratītyemān itīme pratyayāḥ kila / (5.1)
	yāvannotpadyata ime tāvannāpratyayāḥ katham // (5.2)
	naivāsato naiva sataḥ pratyayo 'rthasya yujyate / (6.1)
	asataḥ pratyayaḥ kasya sataśca pratyayena kim // (6.2)
	na san nāsan na sadasan dharmo nirvartate yadā / (7.1)
	kathaṃ nirvartako hetur evaṃ sati hi yujyate // (7.2)
	anārambaṇa evāyaṃ san dharma upadiśyate / (8.1)
	athānārambaṇe dharme kuta ārambaṇaṃ punaḥ // (8.2)
	anutpanneṣu dharmeṣu nirodho nopapadyate / (9.1)
	nānantaram ato yuktaṃ niruddhe pratyayaśca kaḥ // (9.2)
	bhāvānāṃ niḥsvabhāvānāṃ na sattā vidyate yataḥ / (10.1)
	satīdam asmin bhavatītyetannaivopapadyate // (10.2)
	na ca vyastasamasteṣu pratyayeṣvasti tat phalam / (11.1)
	pratyayebhyaḥ kathaṃ tacca bhavenna pratyayeṣu yat // (11.2)
	athāsad api tat tebhyaḥ pratyayebhyaḥ pravartate / (12.1)
	apratyayebhyo 'pi kasmānnābhipravartate phalam // (12.2)
	phalaṃ ca pratyayamayaṃ pratyayāścāsvayaṃmayāḥ / (13.1)
	phalamasvamayebhyo yattatpratyayamayaṃ katham // (13.2)
	tasmānna pratyayamayaṃ nāpratyayamayaṃ phalam / (14.1)
	saṃvidyate phalābhāvāt pratyayāpratyayāḥ kutaḥ // (14.2)
