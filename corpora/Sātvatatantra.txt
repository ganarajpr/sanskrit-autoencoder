śrīsūta uvāca / (1.1)
	ya eko bahudhā kṛṣṇaḥ sṛṣṭyādau bahudheyate / (1.2)
	tam ahaṃ śaraṇaṃ yāmi paramānandavigraham // (1.3)
	kailāsaśikharāsīnaṃ śivaṃ śivakaraṃ param / (2.1)
	nāradaḥ paripapraccha sarvabhūtahite rataḥ // (2.2)
	bhagavan śrotum icchāmi harer adbhutakarmaṇaḥ / (3.1)
	śrīkṛṣṇasyāprameyasya nānā līlātanūr vibhoḥ // (3.2)
	yadarthaṃ yatsvarūpaṃ ca yadyat kāle yathā rataḥ / (4.1)
	gṛhṇāti bhagavān svasthas tan mamākhyātum arhasi // (4.2)
	avatāranimittaṃ yac cirād vigrahasambhavam / (5.1)
	prathamaṃ tattvato brūhi tvaṃ parāparavittamaḥ // (5.2)
	śrīśiva uvāca / (6.1)
	pṛṣṭaṃ mahābhāga tvayā bhāgavatottama / (6.2)
	yad ahaṃ noditaḥ samyagbhagavadvīryavarṇane // (6.3)
	śrīviṣṇor avatārāṇāṃ virājaś ca mahāmate / (7.1)
	kathane naiva paśyāmi pāraṃ varṣaśatair api / (7.2)
	tathāpi sāram uddhṛtya tantrarūpeṇa nārada // (7.3)
	varṇayāmi yathaivoktam īśvareṇa dayālunā / (8.1)
	jayapūrvaṃ namaskṛtya goparūpiṇam īśvaram // (8.2)
	vakṣye sātvatatantrākhyaṃ bhagavadbhaktivardhanam / (9.1)
	yadāsīd ekam avyaktaṃ nityaṃ cidrūpam avyayam // (9.2)
	brahmeti yad vidur vijñā bhagavān iti sātvatāḥ / (10.1)
	tadāsantam ivātmānaṃ matvā dṛśya vinā vibhuḥ // (10.2)
	dviprakāram abhūt satyaṃ sattābhūtvā svayaṃ svayam / (11.1)
	tac cicchaktisvarūpeṇa prakṛtiḥ puruṣo mahān // (11.2)
	sarvalokaikanilayo bhagavān iti śabdyate / (12.1)
	tasyaiva śaktiḥ prakṛtiḥ kāryakāraṇarūpiṇī // (12.2)
	guṇatrayasvarūpeṇa yā svayaṃ bhidyate punaḥ / (13.1)
	yaḥ kālas taṃ vadanty eke hareś ceṣṭāṃ duranvayām // (13.2)
	yasmād guṇatrayakṣobhāt pṛthagbhūto 'bhavat purā / (14.1)
	jīvasya yasmād bhavati śubhāśubhaphalagrahaḥ // (14.2)
	tat karma mahato janmahetur avyaktamūrtimat / (15.1)
	bhāvānāṃ pariṇāmo hi yato bhavati sarvadā // (15.2)
	tam evāhur vedavidaḥ sūkṣmarūpaṃ svabhāvakam / (16.1)
	ukto 'yaṃ puruṣaḥ sākṣād īśvaro bhagavattanuḥ // (16.2)
	kālakarmasvabhāvasthaḥ prakṛtiṃ prati noditaḥ / (17.1)
	puruṣādhiṣṭhitā devī prakṛtir guṇasaṃgrahā // (17.2)
	mahattattvam abhūt tattatparijñānakriyātmakam / (18.1)
	tasmāj jāto hy ahaṃkāras trividho daivanoditāt // (18.2)
	vaikārikas taijasaś ca tāmasaś ceti yaṃ viduḥ / (19.1)
	vaikārikān manodevā jātā jñānakriyādhipāḥ // (19.2)
	marutkeśau diśaḥ sūryo nāsatyo jñānanodakāḥ / (20.1)
	vahnīndramitrakopetā ete karmāpanodakāḥ // (20.2)
	rājasād viṣayagrāhajñānakarmasvarūpiṇaḥ / (21.1)
	tvagrasajñā śrutiś cakṣur ghrāṇaṃ buddhiś ca tanmayāḥ // (21.2)
	vākpāṇipāyūpasthāś ca gatiś ceti kriyātmakāḥ / (22.1)
	tāmasāt pañcabhūtāni tanmātrāṇi ca bhāgaśaḥ // (22.2)
	prathamaṃ tāmasāj jajñe śabdas tasmād abhūn nabhaḥ / (23.1)
	śabdarūpāt tu nabhasaḥ sparśas tasmād abhūn marut // (23.2)
	maruto 'bhūt tatas tejas tejaso rūpam uttamam / (24.1)
	rūpamātrād raso jajñe tasmād āpo 'bhavan śucīḥ // (24.2)
	adbhyo jāto gandhaguṇo gandhād bhūmir ajāyata / (25.1)
	mahattattvam ahaṃkāraḥ saśabdasparśatejasaḥ // (25.2)
	rasagandhāv ime sarve smṛtāḥ prakṛtivikriyāḥ / (26.1)
	śabdasya prakṛter eva saṃdṛśyante yato budhaiḥ // (26.2)
	ato 'bhavan prakṛtayo vikārān vikṛtīr viduḥ / (27.1)
	ākāśe śabdamātraṃ syād vāyau sparśaḥ saśabdakaḥ // (27.2)
	rūpaṃ tejasi śabdaś ca sparśaś caiva jale tathā / (28.1)
	rasaśabdasparśarūpaṃ pṛthivyāṃ sarvam eva hi // (28.2)
	kāraṇānāṃ yataḥ kārye samanvayavidhis tataḥ / (29.1)
	dṛśyate tv adhikas tatra guṇo yāvati kaśca ha // (29.2)
	mahadādīni tattvāni puruṣasya mahātmanaḥ / (30.1)
	kāryāvatārarūpāṇi jānīhi dvijasattama // (30.2)
	sarvāṇy etāni saṃgṛhya puruṣasyecchayā yā / (31.1)
	aṃśair utpādayāmāsur virājaṃ bhuvanātmakam // (31.2)
	tasya cāntargataṃ chidraṃ pañcāśatkoṭivistṛtam / (32.1)
	daśottarādhikair etaiḥ saptabhir bahir āvṛtam // (32.2)
	tam āhuḥ puruṣasyaiva gehaṃ yatrāviśat svayam / (33.1)
	yato 'cetanam evāsīt kevalaṃ sarvavistaram // (33.2)
	narād utpannatattvānāṃ saṃgrahe nārasaṃjñake / (34.1)
	ayanaṃ tasya yad abhūt tasmān nārāyaṇaḥ smṛtaḥ // (34.2)
	virāḍdehe yad avasad bhagavān purasaṃjñake / (35.1)
	ataḥ puruṣanāmānam avāpa puruṣaḥ paraḥ // (35.2)
	yadā sa bhagavān devo mūlaprakṛtivistaraḥ / (36.1)
	nārāyaṇena rūpeṇa virajam aviśat svayam // (36.2)
	tadā cetanam āpādya cirād vigraha utthitaḥ / (37.1)
	samaṣṭiśabdatāvācyo dvisaptabhuvanāśrayaḥ // (37.2)
	yasmin carācaraṃ bhūtaṃ sraṣṭā brahmā hares tanūḥ / (38.1)
	tatra lokamayaṃ yāvat taṃ virājaṃ vido viduḥ // (38.2)
	tasyābhimāninaṃ jīvaṃ vairājaṃ puruṣābhidham / (39.1)
	tadantaryāmiṇaṃ devaṃ nārāyaṇam anāmayam // (39.2)
	sarvajīvaikanilayaṃ bhagavantaṃ pracakṣate / (40.1)
	avatārasahasrāṇāṃ nidhānaṃ bījam avyayam // (40.2)
	yasyāṃśena rajoyuktaḥ sṛṣṭau brahmā vyajāyata / (41.1)
	viṣṇuḥ sattvaguṇādhīśaḥ sthitau sthāpayituṃ jagat // (41.2)
	tamasā rudrarūpo 'bhūt pratisaṃcaraṇo vibhuḥ / (42.1)
	ete viṣṇor guṇamayā avatārāḥ kriyākṛtāḥ // (42.2)
	eṣām aṃśāvatārān me nibodha gadato mama / (43.1)
	brahmaṇo 'ṃśena samabhūn marīcir atrir aṅgirāḥ // (43.2)
	pulastyaḥ pulahaś caiva kratur dakṣo dvijottama / (44.1)
	bhṛgur vasiṣṭho 'tharvā ca kardamādyāḥ prajeśvarāḥ // (44.2)
	eṣāṃ putrāś ca pautrāś ca prapautrāś ca mahaujasaḥ / (45.1)
	sṛṣṭyartham udgatāḥ sarve bhagavadvīryasaṃyutāḥ // (45.2)
	viṣṇor aṃśena samabhūd dharmo yajño bṛhattrivṛt / (46.1)
	svāyambhuvādyā manavo dvisaptā lokaviśrutāḥ // (46.2)
	eṣāṃ putrāś ca pautrāś ca tathendrādyāś ca devatāḥ / (47.1)
	viṣṇvaṃśayuktā lokānāṃ pālakāḥ kathitā mayā // (47.2)
	rudrasyāṃśena śataśo jātā rudragaṇāḥ pṛthak / (48.1)
	sarpāś ca śataśo jātā ye ca hiṃsrāḥ svabhāvataḥ // (48.2)
	ete saṃkṣepataḥ proktā rudrasyāṃśās tamojuṣaḥ / (49.1)
	saṃhāriṇo 'sya jagato ghorarūpā vilakṣaṇāḥ // (49.2)
	atha te sampravakṣyāmi līlādehān hareḥ pṛthak / (50.1)
	śuddhasattvamayān śāntān lokapremāspadān śṛṇu // (50.2)
	ya īśa eko bhagavān ananto brahmasvarūpī puruṣo 'dhiyajñaḥ / (51.1)
	pātuṃ punar viśvam asau svakāryaṃ bheje tanūs taṃ praṇamāmi kṛṣṇam // (51.2)
	iti śrīsātvatatantre śivanāradasaṃvāde prathamaḥ paṭalaḥ // (52.1)
