śriyaḥ kurūṇām adhipasya pālanīṃ prajāsu vṛttiṃ yam ayuṅkta veditum / (1.1)
	sa varṇiliṅgī viditaḥ samāyayau yudhiṣṭhiraṃ dvaitavane vanecaraḥ // (1.2)
	kṛtapraṇāmasya mahīṃ mahībhuje jitāṃ sapatnena nivedayiṣyataḥ / (2.1)
	na vivyathe tasya mano na hi priyaṃ pravaktum icchanti mṛṣā hitaiṣiṇaḥ // (2.2)
	dviṣāṃ vighātāya vidhātum icchato rahasy anujñām adhigamya bhūbhṛtaḥ / (3.1)
	sa sauṣṭhavaudāryaviśeṣaśālinīṃ viniścitārthām iti vācam ādadhe // (3.2)
	kriyāsu yuktair nṛpa cāracakṣuṣo na vañcanīyāḥ prabhavo 'nujīvibhiḥ / (4.1)
	ato 'rhasi kṣantum asādhu sādhu vā hitaṃ manohāri ca durlabhaṃ vacaḥ // (4.2)
	sa kiṃsakhā sādhu na śāsti yo 'dhipaṃ hitān na yaḥ saṃśṛṇute sa kiṃprabhuḥ / (5.1)
	sadānukūleṣu hi kurvate ratiṃ nṛpeṣv amātyeṣu ca sarvasampadaḥ // (5.2)
	nisargadurbodham abodhaviklavāḥ kva bhūpatīnāṃ caritaṃ kva jantavaḥ / (6.1)
	tavānubhāvo 'yam abodhi yan mayā nigūḍhatattvaṃ nayavartma vidviṣām // (6.2)
	viśaṅkamāno bhavataḥ parābhavaṃ nṛpāsanastho 'pi vanādhivāsinaḥ / (7.1)
	durodaracchadmajitāṃ samīhate nayena jetuṃ jagatīṃ suyodhanaḥ // (7.2)
	tathāpi jihmaḥ sa bhavajjigīṣayā tanoti śubhraṃ guṇasampadā yaśaḥ / (8.1)
	samunnayan bhūtim anāryasaṃgamād varaṃ virodho 'pi samaṃ mahātmabhiḥ // (8.2)
	kṛtāriṣaḍvargajayena mānavīm agamyarūpāṃ padavīṃ prapitsunā / (9.1)
	vibhajya naktaṃdinam astatandriṇā vitanyate tena nayena pauruṣam // (9.2)
	sakhīn iva prītiyujo 'nujīvinaḥ samānamānān suhṛdaś ca bandhubhiḥ / (10.1)
	sa saṃtataṃ darśayate gatasmayaḥ kṛtādhipatyām iva sādhu bandhutām // (10.2)
	asaktam ārādhayato yathāyathaṃ vibhajya bhaktyā samapakṣapātayā / (11.1)
	guṇānurāgād iva sakhyam īyivān na bādhate 'sya trigaṇaḥ parasparam // (11.2)
	niratyayaṃ sāma na dānavarjitaṃ na bhūri dānaṃ virahayya satkriyām / (12.1)
	pravartate tasya viśeṣaśālinī guṇānurodhena vinā na satkriyā // (12.2)
	vasūni vāñchan na vaśī na manyunā svadharma ity eva nivṛttakāraṇaḥ / (13.1)
	gurūpadiṣṭena ripau sute 'pi vā nihanti daṇḍena sa dharmaviplavam // (13.2)
	vidhāya rakṣān paritaḥ paretarān aśaṅkitākāram upaiti śaṅkitaḥ / (14.1)
	kriyāpavargeṣv anujīvisātkṛtāḥ kṛtajñatām asya vadanti sampadaḥ // (14.2)
	anārataṃ tena padeṣu lambhitā vibhajya samyag viniyogasatkriyām / (15.1)
	phalanty upāyāḥ paribṛṃhitāyatīr upetya saṃgharṣam ivārthasampadaḥ // (15.2)
	bhūyaḥ // (46.2)
