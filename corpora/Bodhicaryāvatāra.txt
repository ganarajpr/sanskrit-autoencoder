	na hi kiṃcidapūrvamatra vācyaṃ na ca saṃgrathanakauśalaṃ mamāsti / (2.1)
	ataeva na me parārthacintā svamano vāsayituṃ kṛtaṃ mamedam // (2.2)
	mama tāvadanena yāti vṛddhiṃ kuśalaṃ bhāvayituṃ prasādavegaḥ / (3.1)
	atha matsamadhātureva paśyed aparo'pyenamato'pi sārthako'yam // (3.2)
	kṣaṇasampadiyaṃ sudurlabhā pratilabdhā puruṣārthasādhanī / (4.1)
	yadi nātra vicintyate hitaṃ punarapyeṣa samāgamaḥ kutaḥ // (4.2)
	rātrau yathā meghaghanāndhakāre vidyutkṣaṇaṃ darśayati prakāśam / (5.1)
	buddhānubhāvena tathā kadācil lokasya puṇyeṣu matiḥ kṣaṇaṃ syāt // (5.2)
	tasmācchubhaṃ durbalameva nityaṃ balaṃ tu pāpasya mahat sughoram / (6.1)
	taj jīyate'nyena śubhena kena sambodhicittaṃ yadi nāma na syāt // (6.2)
	kalpānanalpān pravicintayadbhir dṛṣṭaṃ munīndrairhitametadeva / (7.1)
	yataḥ sukhenaiva sukhaṃ pravṛddham utplāvayaty apramitāñjanaughān // (7.2)
	bhavaduḥkhaśatāni tartukāmairapi sattvavyasanāni hartukāmaiḥ / (8.1)
	bahusaukhyaśatāni bhoktukāmairna vimocyaṃ hi sadaiva bodhicittam // (8.2)
	bhavacārakabandhano varākaḥ sugatānāṃ suta ucyate kṣaṇena / (9.1)
	na narāmaralokavandanīyo bhavati smodita eva bodhicitte // (9.2)
	aśucipratimāmimāṃ gṛhītvā jinaratnapratimāṃ karotyanarghām / (10.1)
	rasajātam atīva vedhanīyaṃ sudṛḍhaṃ gṛhṇata bodhicittasaṃjñam // (10.2)
	suparīkṣitam aprameyadhībhir bahumūlyaṃ jagadekasārthavāhaiḥ / (11.1)
	gatipattanavipravāsaśīlāḥ sudṛḍhaṃ gṛhṇata bodhicittaratnam // (11.2)
	kadalīva phalaṃ vihāya yāti kṣayamanyatkuśalaṃ hi sarvameva / (12.1)
	satataṃ phalati kṣayaṃ na yāti prasavaty eva tu bodhicittavṛkṣaḥ // (12.2)
	kṛtvāpi pāpāni sudāruṇāni yadāśrayāduttarati kṣaṇena / (13.1)
	śūrāśrayeṇeva mahābhayāni nāśrīyate tatkathamajñasattvaiḥ // (13.2)
	yugāntakālānalavan mahānti pāpāni yan nirdahati kṣaṇena / (14.1)
	yasyānuśaṃsān amitān uvāca maitreyanāthaḥ sudhanāya dhīmān // (14.2)
	tadbodhicittaṃ dvividhaṃ vijñātavyaṃ samāsataḥ / (15.1)
	bodhipraṇidhicittaṃ ca bodhiprasthānameva ca // (15.2)
	gantukāmasya gantuśca yathā bhedaḥ pratīyate / (16.1)
	tathā bhedo'nayorjñeyo yathāsaṃkhyena paṇḍitaiḥ // (16.2)
	bodhipraṇidhicittasya saṃsāre'pi phalaṃ mahat / (17.1)
	na tv avicchinnapuṇyatvaṃ yathā prasthānacetasaḥ // (17.2)
	yataḥ prabhṛtyaparyantasattvadhātupramokṣaṇe / (18.1)
	samādadāti taccittam anivartyena cetasā // (18.2)
	tataḥ prabhṛti suptasya pramattasyāpyanekaśaḥ / (19.1)
	avicchinnāḥ puṇyadhārāḥ pravartante nabhaḥsamāḥ // (19.2)
	idaṃ subāhupṛcchāyāṃ sopapattikamuktavān / (20.1)
	hīnādhimuktisattvārthaṃ svayameva tathāgataḥ // (20.2)
	śiraḥśūlāni sattvānāṃ nāśayāmīti cintayan / (21.1)
	aprameyeṇa puṇyena gṛhyate sma hitāśayaḥ // (21.2)
	kimutāpramitaṃ śūlamekaikasya jihīrṣataḥ / (22.1)
	aprameyaguṇaṃ sattvamekaikaṃ ca cikīrṣataḥ // (22.2)
	kasya mātuḥ piturvāpi hitāśaṃseyamīdṛśī / (23.1)
	devatānāmṛṣīṇāṃ vā brahmaṇāṃ vā bhaviṣyati // (23.2)
	teṣāmeva ca sattvānāṃ svārthe'pyeṣa manorathaḥ / (24.1)
	notpannapūrvaḥ svapne'pi parārthe sambhavaḥ kutaḥ // (24.2)
	sattvaratnaviśeṣo'yamapūrvo jāyate katham / (25.1)
	yatparārthāśayo'nyeṣāṃ na svārthe'pyupajāyate // (25.2)
	jagadānandabījasya jagadduḥkhauṣadhasya ca / (26.1)
	cittaratnasya yatpuṇyaṃ tatkathaṃ hi pramīyatām // (26.2)
	hitāśaṃsanamātreṇa buddhapūjā viśiṣyate / (27.1)
	kiṃ punaḥ sarvasattvānāṃ sarvasaukhyārthamudyamāt // (27.2)
	duḥkhamevābhidhāvanti duḥkhaniḥsaraṇāśayā / (28.1)
	sukhecchayaiva sammohāt svasukhaṃ ghnanti śatruvat // (28.2)
	yasteṣāṃ sukharaṅkāṇāṃ pīḍitānāmanekaśaḥ / (29.1)

	nāśayatyapi sammohaṃ sādhustena samaḥ kutaḥ / (30.1)
	kuto vā tādṛśaṃ mitraṃ puṇyaṃ vā tādṛśaṃ kutaḥ // (30.2)
	kṛte yaḥ pratikurvīta so 'pi tāvatpraśasyate / (31.1)
	avyāpāritasādhustu bodhisattvaḥ kimucyatām // (31.2)
	katipayajanasattradāyakaḥ kuśalakṛdityabhipūjyate janaiḥ / (32.1)
	saparibhavaṃ divasārdhayāpanāt // (32.2)
	kimu niravadhisattvasaṃkhyayā niravadhikālam anuprayacchataḥ / (33.1)
