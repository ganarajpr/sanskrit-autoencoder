	jyeṣṭho'bhūd bhuvi pārijātakataruḥ khaṇḍelavālānvaye tatputraḥ kila nāthaballavasudaḥ prāṇair yaśo'rthānvitaḥ / (4.1)
	tatputreṇa ca sāvareṇa patinā bandhasya dharmārthinā gīrvāṇāśu rugoñcajena satataṃ tenātra yatnaḥ kṛtaḥ // (4.2)
	iha śāstrārambhe ācāryaśrīmadgovindapādāḥ śiṣṭasamayaparipālanārthe śāstrasya deśayato gurupādasya bhagavato vastunirdeśarūpaṃ maṅgalam ācaranti jayatītyādi // (5.0)
	maṅgalalakṣaṇaṃ yathā āśīr namaskriyā vastunirdeśo maṅgala iti // (6.0)
	vastunirdeśaḥ pañcadhā yathāsti bhāti priyaṃ rūpaṃ nāma cetyaṃśapañcakam // (7.0)
	ādyaṃ trayaṃ brahmarūpaṃ śaktirūpaṃ tato dvayam iti // (8.0)
	sa haro jayati sarvotkarṣeṇa vartate // (9.0)
	akhilaṃ jagat paśyata iti vastunirdeśābjayaddraṣṭā // (10.0)
	tathā ca śrutiḥ apāṇipādo javano grahītā // (11.0)
	paśyatyacakṣuḥ sa śṛṇotyakarṇaḥ / (12.1)
	sa vetti vedyaṃ na ca tasya vettā tamāhur agryaṃ puruṣaṃ purāṇam iti // (12.2)
	yasya hṛdayasyaiva rasarūpiṇī karuṇā ghṛṇā jātā prādurbhūtā // (13.0)

	eṣā hṛdayasthaiva // (15.0)
	evāvyayam anyasthānaniṣedhavāci // (16.0)
	rasarūpiṇīti rasaḥ pāradastatsvarūpaṃ yasyāḥ sā // (17.0)
	tathā ca vākyam / (18.1)
	raso jalaṃ raso harṣo rasaḥ śṛṅgārapūrvakaḥ / (18.2)
	svādvādiṣu ca niryāse pārade'pi raso viṣe / (18.3)
	ityanekārthaḥ // (18.4)
	sāmānyatastadrūpaṃ dravatvaṃ viśeṣato rasarūpaṃ sarvopakāritvam // (19.0)
	tathā ca vākyaṃ rasaratnākare / (20.1)
	ājanmapāpakṛtanirdahanaikavahnir dāridryaduḥkhagajavāraṇasiṃharūpaḥ iti // (20.2)
	kiṃbhūtasya harasya dainyagadākulaṃ jagat saṃsāraṃ paśyataḥ dainyayaṃ ca gadāśca tair ākulaṃ vyāptaṃ dainyaṃ dīnabhāvo dāridryaṃ gadā vyādhaya iti // (21.0)
	akhilamiti sarvavyāpipadam // (22.0)
	yathā mañjaryā / (23.1)
