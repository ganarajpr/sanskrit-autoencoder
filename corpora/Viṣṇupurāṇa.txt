parāśaraṃ munivaraṃ kṛtapūrvāhṇikakriyam / (1.1)
	maitreyaḥ paripapraccha praṇipatyābhivādya ca // (1.2)
	tvatto hi vedādhyayanam adhītam akhilaṃ guro / (2.1)
	dharmaśāstrāṇi sarvāṇi vedāṅgāni yathākramam // (2.2)
	tvatprasādān muniśreṣṭha mām anye nākṛtaśramam / (3.1)
	vakṣyanti sarvaśāstreṣu prāyaśo ye 'pi vidviṣaḥ // (3.2)
	so 'ham icchāmi dharmajña śrotuṃ tvatto yathā jagat / (4.1)
	babhūva bhūyaś ca yathā mahābhāga bhaviṣyati // (4.2)
	yanmayaṃ ca jagad brahman yataś caitac carācaram / (5.1)
	līnam āsīd yathā yatra layam eṣyati yatra ca // (5.2)
	yatpramāṇāni bhūtāni devādīnāṃ ca saṃbhavam / (6.1)
	samudraparvatānāṃ ca saṃsthānaṃ ca yathā bhuvaḥ // (6.2)
	sūryādīnāṃ ca saṃsthānaṃ pramāṇaṃ munisattama / (7.1)
	devādīnāṃ tathā vaṃśān manūn manvantarāṇi ca // (7.2)
	kalpān kalpavikalpāṃś ca caturyugavikalpitān / (8.1)
	kalpāntasya svarūpaṃ ca yugadharmāṃś ca kṛtsnaśaḥ // (8.2)
	devarṣipārthivānāṃ ca caritaṃ yan mahāmune / (9.1)
	vedaśākhāpraṇayanaṃ yathāvad vyāsakartṛkam // (9.2)
	dharmāṃś ca brāhmaṇādīnāṃ tathā cāśramavāsinām / (10.1)
	śrotum icchāmy ahaṃ sarvaṃ tvatto vāsiṣṭhanandana // (10.2)
	brahman prasādapravaṇaṃ kuruṣva mayi mānasam / (11.1)
	yenāham etaj jānīyāṃ tvatprasādān mahāmune // (11.2)
	parāśara uvāca / (12.1)
	sādhu maitreya dharmajña smārito 'smi purātanam / (12.2)
	pituḥ pitā me bhagavān vasiṣṭho yad uvāca ha // (12.3)
	viśvāmitraprayuktena rakṣasā bhakṣito mayā / (13.1)
	śrutas tātas tataḥ krodho maitreyāsīn mamātulaḥ // (13.2)
	tato 'haṃ rakṣasāṃ satraṃ vināśāya samārabham / (14.1)
	bhasmīkṛtāś ca śataśas tasmin satre niśācarāḥ // (14.2)
	tataḥ saṃkṣīyamāṇeṣu teṣu rakṣaḥsv aśeṣataḥ / (15.1)
	mām uvāca mahābhāgo vasiṣṭho 'smatpitāmahaḥ // (15.2)
	alam atyantakopena tāta manyum imaṃ jahi / (16.1)
	rākṣasā nāparādhyante pitus te vihitaṃ tathā // (16.2)
	mūḍhānām eṣa bhavati krodho jñānavatāṃ kutaḥ / (17.1)
	hanyate tāta kaḥ kena yataḥ svakṛtabhuk pumān // (17.2)
	saṃcitasyāpi mahatā vatsa kleśena mānavaiḥ / (18.1)
	yaśasas tapasaś caiva krodho nāśakaraḥ paraḥ // (18.2)
	svargāpavargavyāsedhakāraṇaṃ paramarṣayaḥ / (19.1)
	varjayanti sadā krodhaṃ tāta mā tadvaśo bhava // (19.2)
	alaṃ niśācarair dagdhair dīnair anaparādhibhiḥ / (20.1)
	satraṃ te viramatv etat kṣamāsārā hi sādhavaḥ // (20.2)
	evaṃ tātena tenāham anunīto mahātmanā / (21.1)
	upasaṃhṛtavān satraṃ sadyas tadvākyagauravāt // (21.2)
	tataḥ prītaḥ sa bhagavān vasiṣṭho munisattamaḥ / (22.1)
	samprāptaś ca tadā tatra pulastyo brahmaṇaḥ sutaḥ // (22.2)
	pitāmahena dattārghyaḥ kṛtāsanaparigrahaḥ / (23.1)
	mām uvāca mahābhāgo maitreya pulahāgrajaḥ // (23.2)
	pulastya uvāca / (24.1)
	vaire mahati yad vākyād guror asyāśritā kṣamā / (24.2)
	tvayā tasmāt samastāni bhavāñchāstrāṇi vetsyati // (24.3)
	saṃtater na mamocchedaḥ kruddhenāpi yataḥ kṛtaḥ / (25.1)
	tvayā tasmān mahābhāga dadāmy anyaṃ mahāvaram // (25.2)
	purāṇasaṃhitākartā bhavān vatsa bhaviṣyati / (26.1)
	devatāpāramārthyaṃ ca yathāvad vetsyate bhavān // (26.2)
	pravṛtte ca nivṛtte ca karmaṇy astamalā matiḥ / (27.1)
	matprasādād asaṃdigdhā tava vatsa bhaviṣyati // (27.2)
	tataś ca bhagavān prāha vasiṣṭho me pitāmahaḥ / (28.1)
	pulastyena yad uktaṃ te sarvam etad bhaviṣyati // (28.2)
	iti pūrvaṃ vasiṣṭhena pulastyena ca dhīmatā / (29.1)
	yad uktaṃ tat smṛtiṃ yātaṃ tvatpraśnād akhilaṃ mama // (29.2)
	so 'haṃ vadāmy aśeṣaṃ te maitreya paripṛcchate / (30.1)
	purāṇasaṃhitāṃ samyak tāṃ nibodha yathātatham // (30.2)
	viṣṇoḥ sakāśād udbhūtaṃ jagat tatraiva ca sthitam / (31.1)
	sthitisaṃyamakartāsau jagato 'sya jagac ca saḥ // (31.2)
