	īśvara śrotum icchāmi lokanātha jagatprabho / (2.1)
	prasādaṃ kuru deveśa brūhi dharmārthasādhakam // (2.2)
	vaśīkaraṇam uccāṭaṃ mohanaṃ stambhanaṃ tathā / (3.1)
	śāntikaṃ pauṣṭikaṃ vātha karaṇāni bahūni ca // (3.2)
	cakṣurhāniṃ maheśāna śrutihāniṃ tathaiva ca / (4.1)
	jñānahāniṃ kriyāhāniṃ kīlakaṃ ca tathāparam // (4.2)
	kāryastambhaṃ sureśāna śoṣaṇaṃ pūraṇaṃ tathā / (5.1)
	mantradhyānaṃ viśeṣeṇa tat sarvaṃ vada me prabho // (5.2)
	anyac ca vividhaṃ kāryaṃ prasādād brūhi bhairava / (6.1)
	yasya vijñānamātreṇa manuṣyo bhuvi durlabhaḥ // (6.2)
	śrīr īśvara uvāca / (7.1)
	śṛṇu tvaṃ hi varārohe siddhyarthaṃ yadi pṛcchasi / (7.2)
	tadvad iṣyāmi te devi tat sarvaṃ samudāhṛtam // (7.3)
	auṣadhair mantrajāpaiś ca ripuṃ hanyān na saṃśayaḥ / (8.1)
	uḍḍīśāt sāram ākṛṣya mayoktaṃ tava bhaktitaḥ // (8.2)
	uḍḍīśaṃ ca namaskṛtya rudraṃ caiva sudurlabham / (9.1)
	kapardinaṃ virūpākṣaṃ sarvabhūtabhayāpaham // (9.2)
	vakṣye rudrodbhavān yogān sarvaśatruvināśakān / (10.1)
	tais tu prayojitaiḥ saḥ prāṇān hanti na saṃśayaḥ // (10.2)
	prathamaṃ bhūtakaraṇaṃ dvitīyonmādanaṃ tathā / (11.1)
	tṛtīyaṃ dveṣaṇaṃ cātha turyam uccāṭanaṃ tathā // (11.2)
	grāmoccāṭaṃ pañcamaṃ ca jalastambhaṃ ca ṣaṣṭhakam / (12.1)
	vahneḥ stambhakaraṃ cātha vaśīkaraṇam uttamam // (12.2)
	anyān api prayogāṃś ca bahūn śṛṇu varānane / (13.1)
	śivena kathitā yogā uḍḍīśe śāstraniścaye // (13.2)
	andhīkaraṇaṃ mūkīkaraṇaṃ badhirīkaraṇaṃ tathā / (14.1)
	bhūtajvarasya karaṇam astraśastrasya dūṣaṇam // (14.2)
	jaladoṣapraśamanaṃ dadhno madhuvināśanam / (15.1)
	vināśaṃ mattakaraṇaṃ gajavājiprakopanam // (15.2)
	ākarṣaṇaṃ bhujaṃgānāṃ mānavānāṃ tathā dhruvam / (16.1)
	vahner vināśanaṃ kuryāt parṇānāṃ hi vināśanam // (16.2)
	gardabhasyātmakaraṇaṃ parakāyapraveśanam / (17.1)
	vetālapādukāsiddhim ulvakājjvalanaṃ tathā // (17.2)
	anyān bahuprayogāṃś ca raudrān romapraharṣaṇān / (18.1)
	vidyāmantraprayogādīn auṣadhāṃś cābhicārikān // (18.2)
	guptāguptatarāḥ kāryā rakṣitavyāḥ prayatnataḥ / (19.1)
	uḍḍīśaṃ yo na jānāti sa ruṣṭaḥ kiṃ kariṣyati // (19.2)
	sumeruṃ cālayet sthānāt sāgaraiḥ plāvayen mahīm / (20.1)
	sūryaṃ ca pātayed bhūmau nedaṃ mithyā bhaviṣyati // (20.2)
	yathaivendrasya vajraṃ ca pāśaṃ hi varuṇasya ca / (21.1)
	yamasya ca yathā daṇḍaṃ kuberasya gadā yathā // (21.2)
	vahneḥ śaktir yathā proktā khaḍgas tu nirṛter yathā / (22.1)
	yathā vāyoś cāṅkuśaṃ hi triśūlaṃ śūlapāṇinaḥ // (22.2)
	skandasya ca yathā śaktir viṣṇoś cakraṃ sudarśanam / (23.1)
	tathaite ca mahāyogāḥ prayuktāḥ śatrukāraṇe // (23.2)
	anivṛtte nivartante amoghā nātra saṃśayaḥ / (24.1)
	saṃtuṣṭena prayuktena sidhyanti suvicārataḥ // (24.2)
	ratānāṃ karaṇaṃ vakṣye śatrusādhanam uttamam / (25.1)
	yenaiva kṛtamātreṇa bhūto gṛhṇāti mānavam // (25.2)
	nimbakāṣṭhaṃ samādāya caturaṅgulamānataḥ / (26.1)
	śatruviṣṭhāsamāliptaṃ tathā nāma samālikhet // (26.2)
	citāṅgāreṇa tannāmnā dhūpaṃ dadyān maheśvari / (27.1)
	citāntaḥ saṃsthito bhūtvā yasya gātramṛdāharet // (27.2)
	kṛṣṇāṣṭamyāṃ caturdaśyām aṣṭottaraśataṃ japet / (28.1)
	bhūto gṛhṇāti taṃ śīghraṃ mantreṇānena mantritaḥ // (28.2)
	uoṃ namo bhagavate sarvabhūtādhipataye virūpākṣāya nityaṃ krūrāya daṃṣṭriṇe vikarāline grahayakṣabhūtavetālena saha śaṃkara manuṣyaṃ daha daha paca paca gṛhṇa gṛhṇa gṛhṇāpaya gṛhṇāpaya huṃ phaṭ svāhā / (29.1)
	pratyānayaṃ yadīccheta tasyaiṣā kriyate kriyā / (29.2)
	dāpayed devam īśānaṃ ghṛtena vacayā saha // (29.3)
	guggulaṃ pradaded dhūpaṃ ghṛtamiśraṃ samantataḥ / (30.1)
	mantreṇa mantrayitvā tu tataḥ svāsthyaṃ bhavet kila // (30.2)
	mantraḥ uoṃ namaḥ śivāya śāntāya prabhāya muktāya devādhidevāya śubhrabāhave vyādhiṃ śamaya śamaya amukaḥ svastho bhavatu namo 'stu te / (31.1)
	iti svāsthyamantraḥ / (31.2)
	athānyat sampravakṣyāmi prayogaṃ bhuvi durlabham / (31.3)
	yenaiva kṛtamātreṇa jvaro gṛhṇāti mānavam // (31.4)
	nimbakāṣṭhākṛtiṃ kṛtvā caturaṅgulamānataḥ / (32.1)
	tasyā hṛdi viṣonmattarājikālavaṇais tathā // (32.2)
	nāma saṃlikhya prakṛtau pācyamānāyāṃ tataḥ param / (33.1)
	tatas tu śatrunāmnā ca aṣṭottaraśataṃ japet // (33.2)
	tataḥ sa praharārdhena jvarabhūtena gṛhyate / (34.1)
	atha japamantraḥ uoṃ bakāmukhā cāmuṇḍā amukasya kṣīramāṃsaśoṇitabhojinī amukaṃ khaḥ khaḥ jvareṇa gṛhṇa gṛhṇa gṛhṇāpaya gṛhṇāpaya huṃ phaṭ svāhā / (34.2)
	pratyānayaṃ yadīccheta tasyaiṣā kriyate kriyā // (34.3)
	dugdhasnātanāmākṣarāṇi tadā svastho bhaved iti / (35.1)
	athānyat sampravakṣyāmi prayogaṃ bhuvi durlabham // (35.2)
	yena vijñānamātreṇa śatror uccāṭanaṃ bhavet / (36.1)
	brahmadaṇḍī citābhasma gomayasya tathaiva ca // (36.2)
	kṣāraṃ cāpi samādāya kākajaṅgāsamanvitam / (37.1)
	kṣipec chirasi śatrūṇāṃ tūrṇam uccāṭayed ripum // (37.2)
	tataḥ saṃsnāpayed enaṃ gokṣīreṇa samanvitam / (38.1)
	muṇḍanaṃ cātha keśānāṃ tataḥ svastho bhaved dhruvam // (38.2)
	athānyat sampravakṣyāmi śatror uccāṭanaṃ varam / (39.1)
	dhūpaṃ nirbhedakaṃ nāma svayaṃ rudreṇa bhāṣitam // (39.2)
	brahmadaṇḍī samūlā ca kākajaṅgā samanvitā / (40.1)
	nirmokaṃ sarparājasya dhūpam uccāṭayed ripūn // (40.2)
	sarpakañcukam ādāya kṛṣṇoragaśiras tathā / (41.1)
	nikhaned yasya ca dvāre tam uccāṭayate haṭhāt // (41.2)
	brahmadaṇḍī surāmāṃsī kacchapasya śiras tathā / (42.1)
	śmaśānabhasmasaṃyuktaṃ kapāle mānuṣe nyaset // (42.2)
	śatror dvāre nikhātena tūrṇam uccāṭayed ripūn / (43.1)
	saptarātrau deveśi samūlaṃ naśyate gṛham // (43.2)
	atha vidveṣaṇaṃ vakṣye śatrūṇāṃ śṛṇu śaṃkari / (44.1)
	yenaiva kṛtamātreṇa vidveṣo jāyate nṛṇām // (44.2)
	brahmadaṇḍī samūlā ca kākamāṃsena saṃyutā / (45.1)
	jātīpuṣparasair bhāvyā saptarātraṃ punaḥ punaḥ // (45.2)
	tato mārjāramūtreṇa saptāhaṃ bhāvayet tataḥ / (46.1)
	eṣa dhūpaḥ pradātavyaḥ śatrugotrasya madhyataḥ // (46.2)
	yathā gandhaṃ samāghrāti pitā putraiḥ samaṃ kaliḥ / (47.1)
	vidveṣaṇaṃ paraṃ teṣāṃ suhṛdbhir bāndhavaiḥ saha // (47.2)
	svasthīkaraṇakaṃ proktaṃ ghṛtagugguladhūpataḥ / (48.1)
	atha māraṇam / (48.2)
	surāṃ mānuṣamāṃsaṃ ca samādāya vicakṣaṇaḥ // (48.3)
	gṛdhrāsthi śatruviṣṭhāṃ ca dvāramadhye nikhātayet / (49.1)
	saptāhena bhaven māro yathā rudreṇa bhāṣitam // (49.2)
	brahmadaṇḍī citābhasma surāmāṃsīsamāyutam / (50.1)
	ariṣṭasya ca pattrāṇi viṣaṃ rudhiram eva ca // (50.2)
	mohāya ca prayoktavyaḥ śatrūṇāṃ nāmayogataḥ / (51.1)
	saptarātraprayogena sarvaśatrupraṇāśanam // (51.2)
	eteṣāṃ duṣṭayogānāṃ śatrave tad udāhṛtam / (52.1)

	uoṃ namo bhagavate śrīuḍḍāmareśvarāya amukam uccāṭaya uccāṭaya vidveṣaya vidveṣaya svāhā / (53.1)

	devam abhyarcya ceśānaṃ śatam aṣṭādhikaṃ japet // (53.3)
	mantram etat prayoktavyaṃ siddhaye siddhikāmyayā / (54.1)
	śatrusainye prayoktavyaṃ rājñaś ca vijayārthinaḥ // (54.2)
	lāṅgūlaṃ gṛhagodhāyāḥ kṛkalāsasya mastakam / (55.1)
	indragopakasaṃyuktaṃ jambūkasya śiras tathā // (55.2)

	amukte hi prayoktavyaṃ sarvaṃ janaśataṃ haret // (56.2)

	asidhārāṅgadā nāma svayaṃ rudreṇa bhāṣitā // (65.2)
