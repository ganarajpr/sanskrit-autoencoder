janaka uvāca / (1.1)
	kathaṃ jñānam avāpto 'ti kathaṃ muktir bhaviṣyati / (1.2)
	vairāgyaṃ ca kathaṃ prāptam etad brūhi mama prabho // (1.3)
	aṣṭāvakra uvāca / (2.1)
	muktim icchasi cet tāta viṣayān viṣavat tyaja / (2.2)
	kṣamārjavadayātoṣasatyaṃ pīyūṣavad bhaja // (2.3)
	na pṛthvī na jalaṃ nāgnir na vāyur dyaur na vā bhavān / (3.1)
	eṣāṃ sākṣiṇam ātmānaṃ cidrūpaṃ viddhi muktaye // (3.2)
	yadi dehaṃ pṛthak kṛtya citi viśrāmya tiṣṭhasi / (4.1)
	adhunaiva sukhī śānto bandhamukto bhaviṣyasi // (4.2)
	na tvaṃ viprādiko varṇo nāśramī nākṣagocaraḥ / (5.1)
	asaṅgo 'si nirākāro viśvasākṣī sukhī bhava // (5.2)
	dharmādharmau sukhaṃ duḥkhaṃ mānasāni na te vibho / (6.1)
	na kartāsi na bhoktāsi mukta evāsi sarvadā // (6.2)
	eko draṣṭāsi sarvasya muktaprāyo 'si sarvadā / (7.1)
	ayam eva hi te bandho draṣṭāraṃ paśyasītaram // (7.2)
	ahaṃ kartety ahaṃmānamahākṛṣṇāhidaṃśitaḥ / (8.1)
	nāhaṃ karteti viśvāsāmṛtaṃ pītvā sukhī bhava // (8.2)
	eko viśuddhabodho 'ham iti niścayavahninā / (9.1)
	prajvālyājñānagahanaṃ vītaśokaḥ sukhī bhava // (9.2)
	yatra viśvam idaṃ bhāti kalpitaṃ rajjusarpavat / (10.1)
	ānandaparamānandaḥ sa bodhas tvaṃ sukhaṃ cara // (10.2)
	muktābhimānī mukto hi baddho baddhābhimāny api / (11.1)
	kiṃ vadantīha satye 'yaṃ yā matiḥ sā gatir bhavet // (11.2)
	ātmā sākṣī vibhuḥ pūrṇa eko muktaś cid akriyaḥ / (12.1)
	asaṅgo nispṛhaḥ śānto bhramāt saṃsāravān iva // (12.2)
	kūṭasthaṃ bodham advaitam ātmānaṃ paribhāvaya / (13.1)
	ābhāso 'haṃ bhramaṃ muktvā bhāvaṃ bāhyam athāntaraṃ // (13.2)
	dehābhimānapāśena ciraṃ baddho 'si putraka / (14.1)
	bodho 'haṃ jñānakhaḍgena tan niṣkṛtya sukhī bhava // (14.2)
	niḥsaṅgo niṣkriyo 'si tvaṃ svaprakāśo nirañjanaḥ / (15.1)
	ayam eva hi te bandhaḥ samādhim anutiṣṭhasi // (15.2)
	tvayā vyāptam idaṃ viśvaṃ tvayi protaṃ yathārthataḥ / (16.1)
	śuddhabuddhasvarūpas tvaṃ mā gamaḥ kṣudracittatām // (16.2)
	nirapekṣo nirvikāro nirbharaḥ śītalāśayaḥ / (17.1)
	agādhabuddhir akṣubdho bhava cinmātravāsanaḥ // (17.2)
	sākāram anṛtaṃ viddhi nirākāraṃ tu niścalam / (18.1)
	etattattvopadeśena na punarbhavasambhavaḥ // (18.2)
	yathaivādarśam adhyasthe rūpe 'ntaḥ paritas tu saḥ / (19.1)
	tathaivāsmin śarīre 'ntaḥ paritaḥ parameśvaraḥ // (19.2)
	ekaṃ sarvagataṃ vyoma bahir antar yathā ghaṭe / (20.1)
	nityaṃ nirantaraṃ brahma sarvabhūtagaṇe tathā // (20.2)
