evaṃ mayā śrutam / (1.1)
	ekasmin samaye bhagavāṃllaṅkāpure samudramalayaśikhare viharati sma nānāratnagotrapuṣpapratimaṇḍite mahatā bhikṣusaṅghena sārdhaṃ mahatā ca bodhisattvagaṇena nānābuddhakṣetrasaṃnipatitair bodhisattvairmahāsattvaiḥ anekasamādhivaśitābalābhijñāvikrīḍitair mahāmatibodhisattvapūrvaṃgamaiḥ sarvabuddhapāṇyabhiṣekābhiṣiktaiḥ svacittadṛśyagocaraparijñānārthakuśalair nānāsattvacittacaritrarūpanayavinayadhāribhiḥ pañcadharmasvabhāvavijñānanairātmyādvayagatiṃgataiḥ / (1.2)
	tena khalu punaḥ samayena bhagavān sāgaranāgarājabhavanāt saptāhenottīrṇo'bhūt / (1.3)
	anekaśakrabrahmanāgakanyākoṭibhiḥ pratyudgamyamāno laṅkāmalayamavalokya smitamakarot pūrvakairapi tathāgatairarhadbhiḥ samyaksaṃbuddhairasmiṃllaṅkāpurīmalayaśikhare svapratyātmāryajñānatarkadṛṣṭitīrthyaśrāvakapratyekabuddhāryaviṣaye tadbhāvito dharmo deśitaḥ / (1.4)
	yannvahamapi atraiva rāvaṇaṃ yakṣādhipatimadhikṛtya etadevodbhāvayan dharmaṃ deśayeyam / (1.5)
	aśrauṣīdrāvaṇo rākṣasādhipatistathāgatādhiṣṭhānāt bhagavān kila sāgaranāgarājabhavanāduttīrya anekaśakrabrahmanāgakanyākoṭibhiḥ parivṛtaḥ puraskṛtaḥ samudrataraṃgānavalokya ālayavijñānodadhipravṛttivijñānapavanaviṣaye preritāstebhyaḥ saṃnipatitebhyaścittānyavalokya tasminneva sthitaḥ udānamudānayati sma yannvahaṃ gatvā bhagavantamadhyeṣya laṅkāṃ praveśayeyam / (1.6)
	tanme syāddīrgharātramarthāya hitāya sukhāya devānāṃ ca manuṣyāṇāṃ ca / (1.7)
	atha rāvaṇo rākṣasādhipatiḥ saparivāraḥ pauṣpakaṃ vimānamadhiruhya yena bhagavāṃstenopajagāma / (1.8)
	upetya vimānādavatīrya saparivāro bhagavantaṃ triṣkṛtvaḥ pradakṣiṇīkṛtya tūryatālāvacaraiḥ pravādyadbhir indranīlamayena daṇḍena vaiḍūryamusārapratyuptāṃ vīṇāṃ priyaṅgupāṇḍunā anarghyeṇa vastreṇa pārśvāvalambitāṃ kṛtvā ṣaḍjarṣabhagāndhāradhaivataniṣādamadhyamakaiśikagītasvaragrāmamūrchanādiyuktenānusārya salīlaṃ vīṇāmanupraviśya gāthābhigītairanugāyati sma / (1.9)
	cittasvabhāvanayadharmavidhiṃ nairātmyaṃ dṛṣṭivigataṃ hyamalam / (1.10)
	pratyātmavedyagatisūcanakaṃ deśehi nāyaka iha // (1.11)
	śubhadharmasaṃcitatanuṃ sugataṃ nirmāṇanirmitapradarśanakam / (2.1)
	pratyātmavedyagatidharmarataṃ laṅkāṃ hi gantuṃ samayo'dya mune // (2.2)
	laṅkāmimāṃ pūrvajinādhyuṣitāṃ putraiśca teṣāṃ bahurūpadharaiḥ / (3.1)
	deśehi nātha iha dharmavaraṃ śroṣyanti yakṣa bahurūpadharāḥ // (3.2)
	atha rāvaṇo laṅkādhipatiḥ toṭakavṛttenānugāyya punarapi gāthābhigītenānugāyati sma saptarātreṇa bhagavān sāgarānmakarālayāt / (4.1)
	sāgarendrasya bhavanāt samuttīrya taṭe sthitaḥ // (4.2)
	sthitamātrasya buddhasya rāvaṇo hyapsaraiḥ saha / (5.1)
	yakṣaiśca nānāvividhaiḥ śukasāraṇapaṇḍitaiḥ // (5.2)
	ṛddhyā gatvā tamadhvānaṃ yatra tiṣṭhati nāyakaḥ / (6.1)
	avatīrya pauṣpakādyānādvandya pūjya tathāgatam / (6.2)
	nāma saṃśrāvayaṃstasmai jinendreṇa adhiṣṭhitaḥ // (6.3)
	rāvaṇo'haṃ daśagrīvo rākṣasendra ihāgataḥ / (7.1)
	anugṛhṇāhi me laṅkāṃ ye cāsmin puravāsinaḥ // (7.2)
	pūrvairapi hi saṃbuddhaiḥ pratyātmagatigocaram / (8.1)
	śikhare ratnakhacite puramadhye prakāśitam // (8.2)
	bhagavānapi tatraiva śikhare ratnamaṇḍite / (9.1)
	deśetu dharmaṃ virajaṃ jinaputraiḥ parīvṛtaḥ / (9.2)
	śrotukāmā vayaṃ cādya ye ca laṅkānivāsinaḥ // (9.3)
	deśanānayanirmuktaṃ pratyātmagatigocaram / (10.1)
	laṅkāvatārasūtraṃ vai pūrvabuddhānuvarṇitam // (10.2)
	smarāmi pūrvakairbuddhairjinaputrapuraskṛtaiḥ / (11.1)
	sūtrametannigadyate bhagavānapi bhāṣatām // (11.2)
	bhaviṣyantyanāgate kāle buddhā buddhasutāśca ye / (12.1)
	etameva nayaṃ divyaṃ śikhare ratnabhūṣite / (12.2)
	deśayiṣyanti yakṣāṇāmanukampāya nāyakāḥ // (12.3)
	divyalaṅkāpurīramyāṃ nānāratnairvibhūṣitām / (13.1)
	prāgbhāraiḥ śītalaiḥ ramyai ratnajālavitānakaiḥ // (13.2)
	rāgadoṣavinirmuktāḥ pratyātmagaticintakāḥ / (14.1)
	santyatra bhagavan yakṣāḥ pūrvabuddhaiḥ kṛtārthinaḥ / (14.2)
	mahāyānanaye śraddhā niviṣṭānyonyayojakāḥ // (14.3)
	yakṣiṇyo yakṣaputrāśca mahāyānabubhutsavaḥ / (15.1)
	āyātu bhagavān śāstā laṅkāmalayaparvatam // (15.2)
	kumbhakarṇapurogāśca rākṣasāḥ puravāsinaḥ / (16.1)
	śroṣyanti pratyātmagatiṃ mahāyānaparāyaṇāḥ // (16.2)
	kṛtādhikārā buddheṣu kariṣyantyadhunā ca vai / (17.1)
	anukampārthaṃ mahyaṃ vai yāhi laṅkāṃ sutaiḥ saha // (17.2)
	gṛhamapsaravargāśca hārāṇi vividhāni ca / (18.1)
	ramyāṃ cāśokavanikāṃ pratigṛhṇa mahāmune // (18.2)
	ājñākaro'haṃ buddhānāṃ ye ca teṣāṃ jinātmajāḥ / (19.1)
	nāsti tadyanna deyaṃ me anukampa mahāmune // (19.2)
	tasya tadvacanaṃ śrutvā uvāca tribhaveśvaraḥ / (20.1)
	atītairapi yakṣendra nāyakai ratnaparvate // (20.2)
	pratyātmadharmo nirdiṣṭaḥ tvaṃ caivāpyanukampitaḥ / (21.1)
	anāgatāśca vakṣyanti girau ratnavibhūṣite // (21.2)
	yogināṃ nilayo hyeṣa dṛṣṭadharmavihāriṇām / (22.1)
	anukampyo'si yakṣendra sugatānāṃ mamāpi ca // (22.2)
	adhivāsya bhagavāṃstūṣṇīṃ śamabuddhyā vyavasthitaḥ / (23.1)
	ārūḍhaḥ puṣpake yāne rāvaṇenopanāmite // (23.2)
	tatraiva rāvaṇo'nye ca jinaputrā viśāradāḥ / (24.1)
	apsarairhāsyalāsādyaiḥ pūjyamānāḥ purīṃ gatāḥ // (24.2)
	tatra gatvā purīṃ ramyāṃ punaḥ pūjāṃ pralabdhavān / (25.1)
	rāvaṇādyairyakṣavargairyakṣiṇībhiśca pūjitaḥ / (25.2)
	yakṣaputrairyakṣakanyābhī ratnajālaiśca pūjitaḥ // (25.3)
	rāvaṇenāpi buddhasya hārā ratnavibhūṣitāḥ / (26.1)
	jinasya jinaputrāṇāmuttamāṅgeṣu sthāpitāḥ // (26.2)
	pragṛhya pūjāṃ bhagavān jinaputraiśca paṇḍitaiḥ / (27.1)
	dharmaṃ vibhāvayāmāsa pratyātmagatigocaram // (27.2)
	rāvaṇo yakṣavargāśca sampūjya vadatāṃ varam / (28.1)
	mahāmatiṃ pūjayanti adhyeṣanti punaḥ punaḥ / (28.2)
	tvaṃ praṣṭā sarvabuddhānāṃ pratyātmagatigocaram // (28.3)
	ahaṃ hi śrotā yakṣāśca jinaputrāśca sanniha / (29.1)
	adhyeṣayāmi tvā yakṣā jinaputrāśca paṇḍitāḥ // (29.2)
	vādināṃ tvaṃ mahāvādī yogināṃ yogavāhakaḥ / (30.1)
	adhyeṣayāmi tvāṃ bhaktyā nayaṃ pṛccha viśārada // (30.2)
	tīrthyadoṣairvinirmuktaṃ pratyekajinaśrāvakaiḥ / (31.1)
	pratyātmadharmatāśuddhaṃ buddhabhūmiprabhāvakam // (31.2)
	nirmāya bhagavāṃstatra śikharānratnabhūṣitān / (32.1)
	anyāni caiva divyāni ratnakoṭīralaṃkṛtāḥ // (32.2)
	ekaikasmin girivare ātmabhāvaṃ vidarśayan / (33.1)
	tatraiva rāvaṇo yakṣa ekaikasmin vyavasthitaḥ // (33.2)
	atra tāḥ parṣadaḥ sarvā ekaikasmin hi dṛśyate / (34.1)
	sarvakṣetrāṇi tatraiva ye ca teṣu vināyakāḥ // (34.2)
	rākṣasendraśca tatraiva ye ca laṅkānivāsinaḥ / (35.1)
	tatpratispardhinī laṅkā jinena abhinirmitā / (35.2)
	anyāścāśokavanikā vanaśobhāśca tatra yāḥ // (35.3)
	ekaikasmin girau nātho mahāmatipracoditaḥ / (36.1)
	dharmaṃ dideśa yakṣāya pratyātmagatisūcakam / (36.2)
	dideśa nikhilaṃ sūtraṃ śatasāhasrikaṃ girau // (36.3)
	śāstā ca jinaputrāśca tatraivāntarhitāstataḥ / (37.1)
	adrākṣīdrāvaṇo yakṣa ātmabhāvaṃ gṛhe sthitam // (37.2)
	cinteti kimidaṃ ko'yaṃ deśitaṃ kena vā śrutam / (38.1)
	kiṃ dṛṣṭaṃ kena vā dṛṣṭaṃ nagaro vā kva saugataḥ // (38.2)
	tāni kṣetrāṇi te buddhā ratnaśobhāḥ kva saugatāḥ / (39.1)
	svapno'yamatha vā māyā nagaraṃ gandharvaśabditam // (39.2)
	timiro mṛgatṛṣṇā vā svapno vandhyāprasūyatam / (40.1)
	alātacakradhūmo vā yadahaṃ dṛṣṭavāniha // (40.2)
	atha vā dharmatā hyeṣā dharmāṇāṃ cittagocare / (41.1)
	na ca bālāvabudhyante mohitā viśvakalpanaiḥ // (41.2)
	na draṣṭā na ca draṣṭavyaṃ na vācyo nāpi vācakaḥ / (42.1)
	anyatra hi vikalpo'yaṃ buddhadharmākṛtisthitiḥ / (42.2)
	ye paśyanti yathādṛṣṭaṃ na te paśyanti nāyakam // (42.3)
	apravṛttivikalpaśca yadā buddhaṃ na paśyati / (43.1)
	apravṛttibhave buddhaḥ saṃbuddho yadi paśyati // (43.2)
	samanantaraprativibuddhe parāvṛttāśraye svacittadṛśyamātrādhigame'vikalpapracārasthitasya laṅkādhipateḥ pūrvakuśalamūlasaṃcoditasya sarvaśāstravidagdhabuddher yathātathyadarśanasya aparapraṇeyasya svabuddhivicālanakuśalasya tarkadṛṣṭivyapetadarśanasya aparapraṇeyasya mahāyogayogino mahāviśvarūpadhāriṇaḥ upāyakauśalyagatiṃgatasya sarvabhūmyuttarottarasvalakṣaṇādhigamanakuśalasya cittamagomanovijñānasvabhāvavivekaratasya trisaṃtativyavacchinnadarśanasya sarvakāraṇatīrthyavyapetabuddheḥ tathā gatagarbhabuddhabhūmyadhyātmasamāpannasya sthitabuddhabuddher gaganādadhyātmavedyaśabdam aśrauṣīt sādhu sādhu laṅkādhipate sādhu khalu punastvaṃ laṅkādhipate / (44.1)
	evaṃ śikṣitavyaṃ yoginā yathā tvaṃ śikṣase / (44.2)
	evaṃ ca tathāgatā draṣṭavyāḥ dharmāśca yathā tvayā dṛṣṭāḥ / (44.3)
	anyathā dṛśyamāne ucchedamāśrayaḥ / (44.4)
	cittamanomanovijñānavigatena tvayā sarvadharmā vibhāvayitavyāḥ / (44.5)
	antaścāriṇā na bāhyārthadṛṣṭyabhiniviṣṭena / (44.6)
	na ca tvayā śrāvakapratyekabuddhatīrthādhigamapadārthagocarapatitadṛṣṭisamādhinā bhavitavyam / (44.7)
	nākhyāyiketihāsaratena bhavitavyam / (44.8)
	na svabhāvadṛṣṭinā na rājādhipatyamadapatitena na ṣaḍdhyānādidhyāyinā / (44.9)
	eṣa laṅkādhipate abhisamayo mahāyoginā parapravādamathanānām akuśaladṛṣṭidālanānām ātmadṛṣṭivyāvartanakuśalānāṃ sūkṣmam abhivijñānaparāvṛttikuśalānāṃ jinaputrāṇāṃ mahāyānacaritānām / (44.10)
	tathāgatasvapratyātmabhūmipraveśādhigamāya tvayā yogaḥ karaṇīyaḥ / (44.11)
	evaṃ kriyamāṇe bhūyo 'pyuttarottaraviśodhako 'yaṃ laṅkādhipate mārgo yastvayā parigṛhītaḥ samādhikauśalasamāpattyā / (44.12)
	na ca śrāvakapratyekabuddhatīrthyānupraveśasukhagocaro yathā bālatīrthayogayogibhiḥ kalpyate ātmagrāhadṛśyalakṣaṇābhiniviṣṭair bhūtaguṇadravyānucāribhir avidyāpratyayadṛṣṭyabhiniveśābhiniviṣṭaiḥ śūnyatotpādavikṣiptair vikalpābhiniviṣṭair lakṣyalakṣaṇapatitāśayaiḥ / (44.13)
	viśvarūpagatiprāpako 'yaṃ laṅkādhipate svapratyātmagatibodhako 'yaṃ mahāyānādhigamaḥ / (44.14)
	viśeṣabhavopapattipratilambhāya ca pravartate / (44.15)
	paṭalakośavividhavijñānataraṃgavyāvartako'yaṃ laṅkādhipate mahāyānayogapraveśo na tīrthyayogāśrayapatanam / (44.16)
	tīrthyayogo hi laṅkādhipate tīrthyānāmātmābhiniveśātpravartate / (44.17)
	vijñānasvabhāvadvayārthānām abhiniveśadarśanād asaumyayogastīrthakarāṇām / (44.18)
	tatsādhu laṅkādhipate etam evārtham anuvicintayeḥ / (44.19)
	yathā vicintitavāṃstathāgatadarśanāt / (44.20)
	etadeva tathāgatadarśanam / (44.21)
	atha tasminnantare rāvaṇasyaitadabhavat yannvahaṃ punarapi bhagavantaṃ sarvayogavaśavartinaṃ tīrthyayogavyāvartakaṃ pratyātmagatigocarodbhāvakaṃ nairmitanairmāṇikavyapetam adhigamabuddhir yadyogināṃ yogābhisamayakāle samādhimukhe samāptānāmadhigamo bhavati / (44.22)
	tasya ca adhigamādyoginā yogaśabdo nipātyate adhigamaneneti / (44.23)
	tadahaṃ kāruṇikaṃ kleśendhanavikalpakṣayakaraṃ taṃ jinaputraiḥ parivṛtaṃ sarvasattvacittāśayānupraviṣṭaṃ sarvagataṃ sarvajñaṃ kriyālakṣaṇavinivṛttaṃ tayaivamṛddhyā paśyeyam taddarśanānnādhigatamadhigaccheyam adhigataṃ ca me nirvikalpācāraḥ sukhasamādhisamāpattivihārastathāgatagatibhūmiprāpako vivṛddhiṃ yāyāt / (44.24)
	atha bhagavāṃstasyāṃ velāyāṃ laṅkādhipater anutpattikadharmakṣāntyadhigataṃ viditvā tayaiva śobhayā daśagrīvasyānukampayā punarapyātmānaṃ śikhare subahuratnakhacite ratnajālavitate darśayati sma / (44.25)
	adrākṣīddaśagrīvo laṅkādhipatiḥ punarapi dṛṣṭānubhūtāṃ śobhāṃ śikhare tathāgatamarhantaṃ samyaksaṃbuddhaṃ dvātriṃśadvaralakṣaṇavibhūṣitatanum / (44.26)
	svātmabhāvaṃ caikaikasmingirau tathāgatānāṃ purataḥ samyaksaṃbuddhānāṃ mahāmatinā sārdhaṃ tathāgatapratyātmagatigocarakathāṃ prakurvantaṃ yakṣaiḥ parivṛtaṃ tāṃ deśanāpāṭhakathāṃ kathayantam / (44.27)
	te ca kṣetrāḥ sanāyakāḥ / (44.28)
	atha bhagavān punarapi tasyāṃ velāyāṃ parṣadamavalokya buddhyā na māṃsacakṣuṣā siṃharājavadvijṛmbhya mahāhāsamahasat / (44.29)
	ūrṇākośācca raśmiṃ niścāryamāṇaḥ pārśvorukaṭikāyācca śrīvatsātsarvaromakūpebhyo yugāntāgniriva dīpyamānaḥ tejasendradhanurudayabhāskaropamena prabhāmaṇḍalena dedīpyamānaḥ śakrabrahmalokapālairgaganatale nirīkṣyamāṇaḥ sumeruśṛṅgapratispardhini śikhare niṣaṇṇo mahāhāsamahasat / (44.30)
	atha tasyā bodhisattvaparṣadaḥ teṣāṃ ca śakrabrahmādīnāmetad abhavat ko nu khalvatra hetuḥ kaḥ pratyayo yadbhagavān sarvadharmavaśavartī mahāhāsaṃ smitapūrvakaṃ hasati raśmīṃśca svavigrahebhyo niścārayati niścārya tūṣṇīmabhavat svapratyātmāryajñānagocarasamādhimukhe patitāśayo'vismitaḥ siṃhāvalokanatayā diśo'valokya rāvaṇasyaiva yogagatipracāram anuvicintayamānaḥ / (44.31)
	atha khalu mahāmatirbodhisattvo mahāsattvaḥ pūrvamevādhyeṣito rāvaṇasyānukampāmupādāya tasyā bodhisattvaparṣadaścittāśayavicāramājñāya anāgatāṃ janatāṃ cāvalokya deśanāpāṭhābhiratānāṃ sattvānāṃ cittavibhramo bhaviṣyatīti yathārutārthābhiniviṣṭānāṃ sarvaśrāvakapratyekabuddhatīrthyayogabalābhiniviṣṭānāṃ tathāgatā api bhagavanto vinivṛttavijñānaviṣayā mahāhāsaṃ hasanti / (44.32)
	teṣāṃ kautūhalavinivṛttyarthaṃ bhagavantaṃ paripṛcchati sma kaḥ khalvatra hetuḥ kaḥ pratyayaḥ smitasya pravṛttaye bhagavānāha sādhu sādhu mahāmate sādhu khalu punastvaṃ mahāmate lokasvabhāvamavalokya kudṛṣṭipatitānāṃ ca lokānāṃ traikālyacittāvabodhāya mā praṣṭumārabdhaḥ / (44.33)
	evaṃ paṇḍitaiḥ paripṛcchanajātīyair bhavitavyaṃ svaparobhayārtham / (44.34)
	eṣa mahāmate rāvaṇo laṅkādhipatiḥ pūrvakānapi tathāgatānarhataḥ samyaksaṃbuddhān praśnadvayaṃ pṛṣṭavān / (44.35)
	māmapyetarhi praṣṭukāmo yadanālīḍhaṃ sarvaśrāvakapratyekabuddhatīrthyayogayogināṃ praśnadvayaprabhedagatilakṣaṇaṃ vibhāvayitum / (44.36)
	ya eṣa praṣṭukāmo daśagrīvo'nāgatānapi jinān prakṣyati / (44.37)
	jānanneva bhagavāťl laṅkādhipatim etad avocat pṛccha tvaṃ laṅkādhipate / (44.38)
	kṛtaste tathāgatenāvakāśaḥ / (44.39)
	mā vilamba pracalitamaulin / (44.40)
	yadyadevākāṅkṣasi ahaṃ te tasya tasyaiva praśnasya vyākaraṇena cittamārādhayiṣyāmi / (44.41)
	yathā tvaṃ parāvṛttavikalpāśraye bhūmivipakṣakauśalena pravicayabuddhyā vicārayamāṇaḥ pratyātmanayalakṣaṇasamādhisukhavihāraṃ samādhibuddhaiḥ parigṛhītaḥ śamathasukhavyavasthitaḥ śrāvakapratyekabuddhasamādhipakṣān atikramya acalāsādhumatīdharmameghābhūmivyavasthito dharmanairātmyayathātathākuśalo mahāratnapadmavimāne samādhijinābhiṣekatāṃ pratilapsyase / (44.42)
	tadanurūpaiḥ padmaiḥ svakāyavicitrādhiṣṭhānādhiṣṭhitais taiḥ padmaiḥ svakāyaṃ niṣaṇṇaṃ drakṣyasi anyonyavaktramukhanirīkṣaṇaṃ ca kariṣyasi / (44.43)
	evamacintyo'sau viṣayaḥ yadekenābhinirhārakauśalenābhinirhṛtaś caryābhūmau sthitaḥ / (44.44)
	upāyakauśalaparigrahābhinirhārābhinirhṛte tam acintyaviṣayam anuprāpsyasi bahurūpavikāratāṃ ca tathāgatabhūmim / (44.45)
	yadadṛṣṭapūrvaṃ śrāvakapratyekabuddhatīrthyabrahmendropendrādibhis taṃ prāpsyasi / (44.46)
	atha khalu laṅkādhipatirbhagavatā kṛtāvakāśa utthāya tasmād raśmivimalaprabhād ratnapadmasadṛśād ratnaśikharāt sāpsarogaṇaparivṛto vividhairanekavidhairnānāprakāraiḥ puṣpamālyagandhadhūpavilepanachattradhvajapatākāhārārdhahārakirīṭamukuṭair anyaiśca adṛṣṭaśrutapūrvairābharaṇaviśeṣair viśiṣṭais tūryatālāvacarair devanāgayakṣarākṣasagandharvakiṃnaramahoragamanuṣyātikrāntaiḥ sarvakāmadhātuparyāpannān vādyabhāṇḍānabhinirmāya ye cānyeṣu buddhakṣetreṣu tūryaviśeṣā dṛṣṭāḥ tānabhinirmāya bhagavantaṃ bodhisattvāṃśca ratnajālenāvaṣṭabhya nānāvastrocchritapatākaṃ kṛtvā sapta tālān gagane'bhyudgamya mahāpūjāmeghānabhipravṛṣya tūryatālāvacarāṇi nirnādya tasmādgaganādavatīrya sūryavidyutprabhe dvitīye mahāratnapadmālaṃkṛtau ratnaśikhare niṣasāda / (44.47)
	niṣadya upacārātsmitapūrvaṃ bhagavatā kṛtāvakāśo bhagavantaṃ praśnadvayaṃ pṛcchati sma pṛṣṭā mayā pūrvakāstathāgatā arhantaḥ samyaksaṃbuddhāḥ / (44.48)
	taiścāpi visarjitam / (44.49)
	bhagavantamapyetarhi pṛcchāmi / (44.50)
	deśanāpāṭhe cāyaṃ buddhaistvayā cāvaśyamanuvarṇitaṃ bhaviṣyati / (44.51)
	nirmitanirmāṇabhāṣitamidaṃ bhagavandharmadvayam / (44.52)
	na maunaistathāgatairbhāṣitam / (44.53)
	maunā hi bhagavaṃstathāgatāḥ samādhisukhagocaramevodbhāvayanti / (44.54)
	na ca gocaraṃ vikalpayanti / (44.55)
	taṃ deśayanti / (44.56)
	tatsādhu me bhagavān svayameva dharmavaśavartī dharmadvayaṃ tathāgato 'rhan samyaksaṃbuddho deśayatu / (44.57)
	śroṣyantīme jinaputrā ahaṃ ca / (44.58)
	bhagavānāha brūhi laṅkādhipate dharmadvayam / (44.59)
	rākṣasendra āha kirīṭāṅgadahāravajrasūtrāvabaddhābharaṇatanuśobhāśobhita dharmā eva prahātavyāḥ prāgevādharmāḥ / (44.60)
	tatkathaṃ bhagavan dharmadvayaṃ prahāṇaṃ bhavati ke cādharmā dharmāḥ kathaṃ sati dvitvaṃ prahāṇadharmāṇāṃ vikalpalakṣaṇapatitānāṃ vikalpasvabhāvābhāvānām abhautikabhautikānām ālayavijñānāparijñānād aviśeṣalakṣaṇānāṃ keśoṇḍukasvabhāvāvasthitānām aśuddhakṣayajñānaviṣayiṇām / (44.61)
	tatkathaṃ teṣāṃ prahāṇam evaṃbhāvinām bhagavānāha nanu laṅkādhipate dṛṣṭo ghaṭādīnāṃ bhedanātmakānāṃ vināśadharmiṇāṃ bālavikalpagocaraiḥ prativibhāgaḥ / (44.62)
	evamihāpi kiṃ na gṛhyate asti dharmādharmayoḥ prativibhāgo bālaprativikalpam upādāya na tvāryajñānādhigamaṃ prati darśanena / (44.63)
	tiṣṭhantu tāvallaṅkādhipate ghaṭādayo bhāvā vicitralakṣaṇapatitā bālānāṃ na tvāryāṇām / (44.64)
	ekasvābhāvikānām ekajvālodbhavaprajvālitānāṃ gṛhabhavanodyānaprāsādapratiṣṭhāpitānāṃ dṛṣṭaḥ prativibhāgaḥ indhanavaśāddīrghahrasvaprabhālpamahāviśeṣāśca / (44.65)
	evamihāpi kiṃ na gṛhyate asti dharmādharmayoḥ prativibhāgaḥ / (44.66)
	na kevalam agnijvālāyā ekasaṃtānapatitāyā dṛṣṭo'rciṣaśca prativibhāgaḥ / (44.67)
	ekabījaprasūtānāṃ yatsaṃtānānām api laṅkādhipate nālāṅkuragaṇḍaparvapatrapalāśapuṣpaphalaśākhāviśeṣāḥ / (44.68)
	evaṃ sarvadharmaprarohadharmiṇāṃ bāhyānām ādhyātmikānām apyavidyāniryātānāṃ skandhadhātvāyatanopagānāṃ sarvadharmāṇāṃ traidhātukopapannānāṃ dṛṣṭasukhasaṃsthānām abhilāpyagativiśeṣāḥ / (44.69)
	vijñānānām ekalakṣaṇānāṃ viṣayābhigrahaṇapravṛttānāṃ dṛṣṭo hīnotkṛṣṭamadhyamaviśeṣo vyavadānāvyavadānataśca kuśalākuśalataśca / (44.70)
	na kevalam eṣā laṅkādhipate dharmāṇāṃ prativibhāgaviśeṣaḥ yogināmapi yogamabhyasyatāṃ yogamārge pratyātmagatilakṣaṇaviśeṣo dṛṣṭaḥ / (44.71)
	kimaṅga punar dharmādharmayoḥ prativikalpapravṛttayor viśeṣo na bhavati bhavatyeva / (44.72)
	asti laṅkādhipate dharmādharmayoḥ prativibhāgo vikalpalakṣaṇatvāt / (44.73)
	tatra laṅkādhipate dharmāḥ katame yaduta ete tīrthyaśrāvakapratyekabuddhabālavikalpakalpitāḥ / (44.74)
	kāraṇato guṇadravyapūrvakā dharmā ityupadiśyante te prahātavyāḥ / (44.75)
	na lakṣaṇataḥ prativikalpayitavyāḥ / (44.76)
	svacittadṛśyadharmatābhiniveśānna santi ghaṭādayo dharmā bālaparikalpitā alabdhaśarīrāḥ / (44.77)
	evaṃ vidarśanayā prativipaśyataḥ prahīṇā bhavanti / (44.78)
	tatra adharmāḥ katame ye'labdhātmakā lakṣaṇavikalpāpracārā dharmā ahetukāḥ teṣāmapravṛttirdṛṣṭā bhūtābhūtataḥ / (44.79)
	atha dharmasya prahāṇaṃ bhavati / (44.80)
	punarapyalabdhātmakā dharmāḥ katame yaduta śaśakharoṣṭravājiviṣāṇavandhyāputraprabhṛtayo dharmāḥ / (44.81)
	alabdhātmakatvānna lakṣaṇataḥ kalpyāḥ / (44.82)
	te'nyatra saṃvyavahārārthā abhidhīyante nābhiniveśato yathā ghaṭādayaḥ / (44.83)
	yathā te praheyā agrahaṇato vijñānena tathā vikalpabhāvā api praheyāḥ / (44.84)
	ato dharmādharmayoḥ prahāṇaṃ bhavati / (44.85)
	yaduktavānasi laṅkādhipate dharmādharmāḥ kathaṃ praheyā iti tadetaduktam / (44.86)
	yadapyuktavānasi laṅkādhipate pūrvakā api tathāgatā arhantaḥ samyaksaṃbuddhā mayā pṛṣṭāḥ taiśca visarjitaṃ pūrvam / (44.87)
	iti laṅkādhipate vikalpasyaitadadhivacanam / (44.88)
	atīto'pyeva vikalpyate atītaḥ / (44.89)
	evamanāgato'dhunāpi dharmatayā / (44.90)
	nirvikalpāstathāgatāḥ sarvavikalpaprapañcātītāḥ / (44.91)
	na yathā rūpasvabhāvo vikalpyate / (44.92)
	anyatrājñānādhigamataḥ sukhārthaṃ vibhāvyate / (44.93)
	prajñayā animittacāriṇaḥ / (44.94)
	ato jñānātmakāstathāgatā jñānaśarīrāḥ / (44.95)
	na kalpante na kalpyante / (44.96)
	kena na kalpante manasā ātmato jīvataḥ pudgalataḥ / (44.97)
	kathaṃ na vikalpante manovijñānena viṣayārthahetukena yathā rūpalakṣaṇasaṃsthānākārataśca / (44.98)
	ato vikalpāvikalpāgatena bhavitavyam / (44.99)
	api ca laṅkādhipate bhittikhacitavigrahasamaḥ sattvapracāraḥ / (44.100)
	niśceṣṭo laṅkādhipate lokasaṃniveśaḥ karmakriyārahito 'sattvātsarvadharmāṇām / (44.101)
	na cātra kaścicchṛṇoti śrūyate vā / (44.102)
	nirmitapratimo hi laṅkādhipate lokasaṃniveśaḥ / (44.103)
	na ca tīrthyabālayogino vibhāvayanti / (44.104)
	ya evaṃ paśyati laṅkādhipate sa samyakpaśyati / (44.105)
	anyathā paśyanto vikalpe carantīti / (44.106)
	svavikalpā dvidhā gṛhṇanti / (44.107)
	tadyathā darpaṇāntargataṃ svabimbapratibimbaṃ jale vā svāṅgacchāyā vā jyotsnādīpapradīpite vā gṛhe vā aṅgacchāyā pratiśrutkāni / (44.108)
	atha svavikalpagrahaṇaṃ pratigṛhya dharmādharmaṃ prativikalpayanti / (44.109)
	na ca dharmādharmayoḥ prahāṇena caranti / (44.110)
	vikalpayanti puṣṇanti na praśamaṃ pratilabhante / (44.111)
