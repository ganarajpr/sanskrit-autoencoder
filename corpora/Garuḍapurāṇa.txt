śrīgaṇādhipataye namaḥ / (1.1)
	sarasvatyai namaḥ / (1.2)
	atha śrīgaruḍamahāpurāṇaṃ prārabhyate / (1.3)
	tatrādau karmakāṇḍākhyaḥ pūrvakhaṇḍaḥ prārabhyate / (1.4)
	oṃ nārāyaṇaṃ namaskṛtya naraṃ caiva narottamam / (1.5)
	devīṃ sarasvatīṃ caiva tato jayamudīrayet / (1.6)
	oṃ ajamajaramanantaṃ jñānarūpaṃ mahāntaṃ śivamamalamanādiṃ bhūtadehādihīnam / (1.7)
	sakalakaraṇahīnaṃ sarvabhūtasthitaṃ taṃ harim amalam amāyaṃ sarvagaṃ vanda ekam // (1.8)
	namasyāmi hariṃ rudraṃ brahmāṇaṃ ca gaṇādhipam / (2.1)
	devīṃ sarasvatīṃ caiva manovākkarmabhiḥ sadā // (2.2)
	sūtaṃ paurāṇikaṃ śāntaṃ sarvaśāstraviśāradam / (3.1)
	viṣṇubhaktaṃ mahātmānaṃ naimiṣāraṇyam āgatam // (3.2)
	tīrthayātrāprasaṅgena upaviṣṭaṃ śubhāsane / (4.1)
	dhyāyantaṃ viṣṇumanaghaṃ tamabhyarcyāstuvankavim // (4.2)
	śaunakādyā mahābhāgā naimiṣīyāstapodhanāḥ / (5.1)
	munayo ravisaṅkāśāḥ śāntā yajñaparāyaṇāḥ // (5.2)
	ṛṣaya ūcuḥ / (6.1)
	sūta jānāsi sarvaṃ tvaṃ pṛcchāmastvāmato vayam / (6.2)
	devatānāṃ hi ko deva īśvaraḥ pūjya eva kaḥ // (6.3)
	ko dhyeyaḥ ko jagatsraṣṭā jagat pāti ca hanti kaḥ / (7.1)
	kasmātpravartate dharmo duṣṭahantā ca kaḥ smṛtaḥ // (7.2)
	tasya devasya kiṃ rūpaṃ jagatsargaḥ kathaṃ mataḥ / (8.1)
	kairvrataiḥ sa tu tuṣṭaḥ syātkena yogena vāpyate // (8.2)
	avatārāśca ke tasya kathaṃ vaṃśādisambhavaḥ / (9.1)
	varṇāśramādidharmāṇāṃ kaḥ pātā kaḥ pravartakaḥ // (9.2)
	etatsarvaṃ tathānyacca brūhi sūta mahāmate / (10.1)
	nārāyaṇakathāḥ sarvāḥ kathayāsmākamuttamāḥ // (10.2)
	sūta uvāca / (11.1)
	purāṇaṃ gāruḍaṃ vakṣye sāraṃ viṣṇukathāśrayam / (11.2)
	garuḍoktaṃ kaśyapāya purā vyāsācchrutaṃ mayā // (11.3)
	eko nārāyaṇo devo devānāmīśvareśvaraḥ / (12.1)
	paramātmā paraṃ brahma janmādyasya yato bhavet // (12.2)
	jagato rakṣaṇārthāya vāsudevo 'jaro 'maraḥ / (13.1)
	sa kumārādirūpeṇa avatārānkarotyajaḥ // (13.2)
	hariḥ sa prathamaṃ devaḥ kaumāraṃ sargamāsthitaḥ / (14.1)
	cacāra duścaraṃ brahman brahmacaryamakhaṇḍitam // (14.2)
	dvitīyaṃ tu bhavāyāsya rasātalagatāṃ mahīm / (15.1)
	uddhariṣyannupādatte yajñeśaḥ saukaraṃ vapuḥ // (15.2)
	tṛtīyamṛṣisargaṃ tu devarṣitvamupetya saḥ / (16.1)
	tantraṃ sātvatamācaṣṭe naiṣkarmyaṃ karmaṇāṃ yataḥ // (16.2)
	naranārāyaṇo bhūtvā turye tepe tapo hariḥ / (17.1)
	dharmasaṃrakṣaṇārthāya pūjitaḥ sa surāsuraiḥ // (17.2)
	pañcamaḥ kapilo nāma siddheśaḥ kālaviplutam / (18.1)
	provāca sūraye sāṃkhyaṃ tattvagrāmavinirṇayam // (18.2)
	ṣaṣṭhamatrerapatyatvaṃ dattaḥ prāpto 'nasūyayā / (19.1)
	ānvīkṣikīmalarkāya prahlādādibhya ūcivān // (19.2)
	tataḥ sapta ākūtyāṃ ruceryajño 'bhyajāyata / (20.1)
	sutrāmādyaiḥ suragaṇair yaṣṭvā svāyambhuvāntare // (20.2)
	aṣṭame merudevyāṃ tu nābherjāta urukramaḥ / (21.1)
	darśayanvartma nārīṇāṃ sarvāśramanamaskṛtam // (21.2)
	ṛṣibhiryācito bheje navamaṃ pārthivaṃ vapuḥ / (22.1)
	dugdhairmahauṣadhairviprāstena saṃjīvitāḥ prajāḥ // (22.2)
	rūpaṃ sa jagṛhe mātsyaṃ cākṣuṣāntarasaṃplave / (23.1)
	nāvyāropya mahīmayyāmapād vaivasvataṃ manum // (23.2)
	surāsurāṇāmudadhiṃ mathnatāṃ mandarācalam / (24.1)
	dadhre kamaṭharūpeṇa pṛṣṭha ekādaśe vibhuḥ // (24.2)
	dhānvantaraṃ dvādaśamaṃ trayodaśamameva ca / (25.1)
	āpyāyat surān anyān mohinyā mohayan striyā // (25.2)
	caturdaśaṃ nārasiṃhaṃ caitya daityendramūrjitam / (26.1)
	dadāra karajairugrairerakāṃ kaṭakudyathā // (26.2)
	pañcadaśaṃ vāmanako bhūtvāgādadhvaraṃ baleḥ / (27.1)
	pādatrayaṃ yācamānaḥ pratyāditsustriviṣṭapam // (27.2)
	avatāre ṣoḍaśame paśyanbrahmadruho nṛpān / (28.1)
	triḥ saptakṛtvaḥ kupito niḥkṣattrām akaronmahīm // (28.2)
	tataḥ saptadaśe jātaḥ satyavatyāṃ parāśarāt / (29.1)
	cakre vedataroḥ śākhāṃ dṛṣṭvā puṃso 'lpamedhasaḥ // (29.2)
	naradevatvamāpannaḥ surakāryacikīrṣayā / (30.1)
	samudranigrahādīni cakre kāryāṇyataḥ param // (30.2)
	ekonaviṃśe viṃśatime vṛṣṇiṣu prāpya janmanī / (31.1)
	rāmakṛṣṇāviti bhuvo bhagavānaharadbharam // (31.2)
	tataḥ kalestu sandhyānte saṃmohāya suradviṣām / (32.1)
	buddho nāmnā jinasutaḥ kīkaṭeṣu bhaviṣyati // (32.2)
	atha so 'ṣṭamasandhyāyāṃ naṣṭaprāyeṣu rāñjasu / (33.1)
	bhavitā viṣṇuyaśaso nāmnā kalkī jagatpatiḥ // (33.2)
	avatārā hyasaṃkhyeyā hareḥ sattvanidherdvijāḥ / (34.1)
	manuvedavido hyādyāḥ sarve viṣṇukalāḥ smṛtāḥ // (34.2)
	tasmātsargādayo jātāḥ saṃpūjyāśca vratādinā / (35.1)
	purāṇaṃ gāruḍaṃ vyāsaḥ purāsau me 'bravīdidam // (35.2)
	iti śrīgāruḍe mahāpurāṇe pūrvakhaṇḍe prathamāṃśākhye karmakāṇḍe etatpurāṇapravṛttihetunirūpaṇaṃ nāma prathamo 'dhyāyaḥ // (36.1)
