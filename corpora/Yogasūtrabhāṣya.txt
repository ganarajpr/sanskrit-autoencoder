atha iti ayam adhikārārthaḥ // (1.1)
	yogānuśāsanaṃ śāstram adhikṛtaṃ veditavyam // (2.1)
	yogaḥ samādhiḥ sa ca sārvabhaumaścittasya dharmaḥ // (3.1)
	kṣiptaṃ mūḍhaṃ vikṣiptaṃ ekāgraṃ niruddham iti cittabhūmayaḥ // (4.1)
	tatra vikṣipte cetasi vikṣepopasarjanībhūtaḥ samādhir na yogapakṣe vartate // (5.1)
	yas tv ekāgre cetasi sadbhūtam arthaṃ pradyotayati kṣiṇoti ca kleśān karmabandhanāni ślathayati nirodham abhimukhaṃ karoti sa samprajñāto yoga ity ākhyāyate // (6.1)
	sa ca vitarkānugato vicārānugata ānandānugato 'smitānugata iti upariṣṭāt pravedayiṣyāmaḥ // (7.1)
	sarvavṛttinirodhe tu asaṃprajñātaḥ samādhiḥ // (8.1)
	tasya lakṣaṇābhidhitsayedaṃ sūtraṃ pravavṛte // (9.1)
