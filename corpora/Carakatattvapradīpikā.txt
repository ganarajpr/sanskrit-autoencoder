samprati madhuratiktakaṣāyāṇāṃ śītatvaṃ tathā kaṭvamlalavaṇānāṃ coṣṇatvaṃ tathā kaṭutiktakaṣāyāṇāṃ cāvṛṣyatvamityādayo rasadvāreṇa dravyāṇāṃ ye guṇā uktāstadapavādam āha teṣāmityādi // (1.0)
	rasopadeśena rasaguṇakathanadvāreṇa dravyāṇāṃ yaḥ śītoṣṇādiguṇasaṃgrahaḥ kṛtaḥ sa vīryataḥ pākataścāviparītānāṃ teṣāṃ vakṣyamāṇakṣīrādidravyāṇāmeva nirdeṣṭuṃ śakyaḥ na tu rasaviparītavīryavipākānām ityarthaḥ // (2.0)
	teṣvaviparītavīryavipākān upadiśatyupadekṣyata ityādi // (3.0)
	upadekṣyata iti yathā paya ityādibhiḥ sambadhyate // (4.0)
	etāni hi dravyaguṇakathanaprasaṅge rasāviparītavīryavipākatayaivopadeṣṭavyānīti rasānurūpaguṇatvam eṣāṃ jñātavyam ityarthaḥ // (5.0)
	idaṃ tūdāharaṇaikadeśamātraṃ tenāparāṇy apyevaṃjātīyāny udāhartavyānītyāha evamādīnītyādi // (6.0)
	evamādīni evaṃprakārāṇi godhūmādīnītyarthaḥ // (7.0)
