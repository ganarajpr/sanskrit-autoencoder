śrīgaṇeśāya namaḥ // (1.1)
	oṃ namaḥ śrīpuruṣottamāya / (2.1)
	oṃ namaḥ śrīnarmadāyai / (2.2)
	oṃ namo hariharahiraṇyagarbhebhyo namo vyāsavālmīkiśukaparāśarebhyo namo gurugobrāhmaṇebhyaḥ / (2.3)
	oṃ majjanmātaṅgagaṇḍacyutamadamadirāmodamattālimālaṃ snānaiḥ siddhāṅganānāṃ kucayugavigalat kuṅkumāsaṅgapiṅgam / (2.4)
	sāyaṃ prātarmunīnāṃ kusumacayasamācchannatīrasthavṛkṣaṃ pāyādvo narmadāmbhaḥ karimakarakarākrāntarahaṃstaraṃgam // (2.5)
	ubhayataṭapuṇyatīrthā prakṣālitasakalalalokaduritaughā / (3.1)
	devamunimanujavandyā haratu sadā narmadā duritam // (3.2)
	nāśayatu duritamakhilaṃ bhūtaṃ bhavyaṃ bhavacca bhuvi bhavinām / (4.1)
	sakalapavitri tava sudhā puṇyajalā narmadā bhavati // (4.2)
	taṭapulinaṃ śivadevā yasyā yatayo 'pi kāmayante vā / (5.1)
	muninivahavihitasevā śivāya mama jāyatāṃ revā // (5.2)
	nārāyaṇaṃ namaskṛtvā naraṃ caiva narottamam / (6.1)
	devīṃ sarasvatīṃ vyāsaṃ tato jayamudīrayet // (6.2)
	naimiṣe puṇyanilaye nānāṛṣiniṣevite / (7.1)
	śaunakaḥ satramāsīnaḥ sūta papraccha vistarāt // (7.2)
	manye 'haṃ dharmanaipuṇyaṃ tvayi sūta sadārcitam / (8.1)
	puṇyāmṛtakathāvaktā vyāsasaśiṣyastvameva hi // (8.2)
	atastvāṃ paripṛcchāmi dharmatīrthāśrayaṃ kave / (9.1)
	bahūni santi tīrthāni bahuśo me śrutāni ca // (9.2)
	śrutā divyanadī brāhmī tathā viṣṇunadī mayā / (10.1)
	tṛtīyā na mayā kvāpi śrutā raudrī saridvarā // (10.2)
	tāṃ vedagarbhāṃ vikhyātāṃ vibudhaughābhivanditām / (11.1)
	vada me tvaṃ mahāprājña tīrthapūgapariṣkṛtām // (11.2)
	kaṃ deśam āśritā revā kathaṃ śrīrudrasaṃbhavā / (12.1)
	tatsaṃśritāni tīrthāni yāni tāni vadasva me // (12.2)
	sūta uvāca / (13.1)
	sādhu pṛṣṭaṃ kulapate caritraṃ narmadāśritam / (13.2)
	citraṃ pavitraṃ doṣaghnaṃ śrutamuktaṃ ca sattama // (13.3)
	vedopavedavedāṅgādīnyabhivyasya pūritaḥ / (14.1)
	aṣṭādaśapurāṇānāṃ vaktā satyavatīsutaḥ // (14.2)
	taṃ namaskṛtya vakṣyāmi purāṇāni yathākramam / (15.1)
	yeṣām abhivyāharaṇād abhivṛddhir vṛṣāyuṣoḥ // (15.2)
	śrutiḥ smṛtiśca viprāṇāṃ cakṣuṣī parikīrtite / (16.1)
	kāṇastatraikayā hīno dvābhyāmandhaḥ prakīrtitaḥ // (16.2)
	śrutismṛtipurāṇāni viduṣāṃ locanatrayam / (17.1)
	yastribhirnayanaiḥ paśyetso 'śo māheśvaro mataḥ // (17.2)
	ātmano vedavidyā ca īśvareṇa vinirmitā / (18.1)
	śaunakīyā ca paurāṇī dharmaśāstrātmikā ca yā // (18.2)
	tisro vidyā imā mukhyāḥ sarvaśāstravinirṇaye / (19.1)
	purāṇaṃ pañcamo veda iti brahmānuśāsanam // (19.2)
	yo na veda purāṇaṃ hi na sa vedātra kiṃcana / (20.1)
	katamaḥ sa hi dharmo 'sti kiṃ vā jñānaṃ tathāvidham // (20.2)
	anyadvā tatkimatrāha purāṇe yanna dṛśyate / (21.1)
	vedāḥ pratiṣṭhitāḥ pūrvaṃ purāṇe nātra saṃśayaḥ // (21.2)
	bibhetyalpaśrutādvedo māmayaṃ pratariṣyati / (22.1)
	itihāsapurāṇaiśca kṛto'yaṃ niścayaḥ purā // (22.2)
	ātmā purāṇaṃ vedānāṃ pṛthagaṃgāni tāni ṣaṭ / (23.1)
	yacca dṛṣṭaṃ hi vedeṣu taddṛṣṭaṃ smṛtibhiḥ kila // (23.2)
	ubhābhyāṃ yattu dṛṣṭaṃ hi tatpurāṇeṣu gīyate / (24.1)
	purāṇaṃ sarvaśāstrāṇāṃ prathamaṃ brahmaṇaḥ smṛtam // (24.2)
	anantaraṃ ca vaktrebhyo vedāstasya vinirgatāḥ / (25.1)
	purāṇamekamevāsīd asmin kalpāntare mune // (25.2)
	trivargasādhanaṃ puṇyaṃ śatakoṭipravistaram / (26.1)
	smṛtvā jagāda ca munīnprati devaścaturmukhaḥ // (26.2)
	pravṛttiḥ sarvaśāstrāṇāṃ purāṇasyābhavattataḥ / (27.1)
	kālenāgrahaṇaṃ dṛṣṭvā purāṇasya tato muniḥ // (27.2)
	vyāsarūpaṃ vibhuḥ kṛtvā saṃharetsa yuge yuge / (28.1)
	aṣṭalakṣapramāṇe tu dvāpare dvāpare sadā // (28.2)
	tadaṣṭādaśadhā kṛtvā bhūloke 'smin prabhāṣyate / (29.1)
	adyāpa devaloke tacchatakoṭipravistaram // (29.2)
	tathātra caturlakṣaṃ saṃkṣepeṇa niveśitam / (30.1)
	purāṇāni daśāṣṭau ca sāmprataṃ tadihocyate / (30.2)
	nāmatastāni vakṣyāmi śṛṇu tvamṛṣisattama // (30.3)
	sargaśca pratisargaśca vaṃśo manvantarāṇi ca / (31.1)
	vaṃśānucaritaṃ caiva purāṇaṃ pañcalakṣaṇam // (31.2)
	brāhmaṃ purāṇaṃ tatrādyaṃ saṃhitāyāṃ vibhūṣitam / (32.1)
	ślokānāṃ daśasāhasraṃ nānāpuṇyakathāyutam // (32.2)
	pādmaṃ ca pañcapañcāśatsahasrāṇi nigadyate / (33.1)
	tṛtīyaṃ vaiṣṇavaṃnāma trayoviṃśatisaṃkhyayā // (33.2)
	caturthaṃ vāyunā proktaṃ vāyavīyamiti smṛtam / (34.1)
	śivabhaktisamāyogācchaivaṃ taccāparākhyayā // (34.2)
	caturviṃśatisaṃkhyātaṃ sahasrāṇi tu śaunaka / (35.1)
	caturbhiḥ parvabhiḥ proktaṃ bhaviṣyaṃ pañcamaṃ tathā // (35.2)
	caturdaśasahasrāṇi tathā pañca śatāni tat / (36.1)
	mārkaṇḍaṃ navasāhasraṃ ṣaṣṭhaṃ tatparikīrtitam // (36.2)
	āgneyaṃ saptamaṃ proktaṃ sahasrāṇi tu ṣoḍaśa / (37.1)
	aṣṭamaṃ nāradīyaṃ tu proktaṃ vai pañcaviṃśatiḥ // (37.2)
	navamaṃ bhagavannāma bhāgadvayavibhūṣitam / (38.1)
	tadaṣṭādaśasāhasraṃ procyate granthasaṃkhyayā // (38.2)
	daśamaṃ brahmavaivartaṃ tāvatsaṃkhyam ihocyate / (39.1)
	laiṅgamekādaśaṃ jñeyaṃ tathaikādaśasaṃkhyayā // (39.2)
	bhāgadvayaṃ viracitaṃ talliṅgamṛṣipuṃgava / (40.1)
	caturviṃśatisāhasraṃ vārāhaṃ dvādaśaṃ viduḥ // (40.2)
	vibhaktaṃ saptabhiḥ khaṇḍaiḥ skāndaṃ bhāgyavatāṃ vara / (41.1)
	tadekāśītisāhasraṃ saṃkhyayā vai nirūpitam // (41.2)
	tatastu vāmanaṃ nāma caturdaśatamaṃ smṛtam / (42.1)
	saṃkhyayā daśasāhasraṃ proktaṃ kulapate purā // (42.2)
	kaurmaṃ pañcadaśaṃ prāhur bhāgadvayavibhūṣitam / (43.1)
	daśasaptasahasrāṇi purā sāṃkhyapate kalau // (43.2)
	mātsyaṃ matsyena yatproktaṃ manave ṣoḍaśaṃ kramāt / (44.1)
	taccaturdaśasāhasraṃ saṃkhyayā vadatāṃ vara // (44.2)
	gāruḍaṃ saptadaśamaṃ smṛtaṃ caikonaviṃśatiḥ / (45.1)
	aṣṭādaśaṃ tu brahmāṇḍaṃ bhāgadvayavibhūṣitam // (45.2)
	tacca dvādaśasāhasraṃ śatamaṣṭasamanvitam / (46.1)
	tathaivopapurāṇāni yāni coktāni vedhasā // (46.2)
	idaṃ brahmapurāṇasya sulabhaṃ sauramuttamam / (47.1)
	saṃhitādvayasaṃyuktaṃ puṇyaṃ śivakathāśrayam // (47.2)
	ādyā sanatkumāroktā dvitīyā sūryabhāṣitā / (48.1)
	sanatkumāranāmnā hi tadvikhyātaṃ mahāmune // (48.2)
	dvitīyaṃ nārasiṃhaṃ ca purāṇe pādmasaṃjñite / (49.1)
	śaukeyaṃ hi tṛtīyaṃ tu purāṇe vaiṣṇave matam // (49.2)
	bārhaspatyaṃ caturthaṃ ca vāyavyaṃ saṃmataṃ sadā / (50.1)
	daurvāsasaṃ pañcamaṃ ca smṛtaṃ bhāgavate sadā // (50.2)
	bhaviṣye nāradoktaṃ ca sūribhiḥ kathitaṃ purā / (51.1)
	kāpilaṃ mānavaṃ caiva tathaivośanaseritam // (51.2)
	brahmāṇḍaṃ vāruṇaṃ cātha kālikādvayameva ca / (52.1)
	māheśvaraṃ tathā sāmbaṃ sauraṃ sarvārthasaṃcayam // (52.2)
	pārāśaraṃ bhāgavataṃ kaurmaṃ cāṣṭādaśaṃ kramāt / (53.1)
	etānyupapurāṇāni mayoktāni yathākramam // (53.2)
	purāṇasaṃhitāmetāṃ yaḥ paṭhedvā śṛṇoti ca / (54.1)
	so 'nantapuṇyabhāgī syānmṛto brahmapuraṃ vrajet // (54.2)
	iti śrīskānde mahāpurāṇa ekāśītisāhasryāṃ saṃhitāyāṃ pañcama āvantyakhaṇḍe revākhaṇḍe purāṇasaṃhitāvarṇanām prathamo 'dhyāyaḥ // (55.1)
