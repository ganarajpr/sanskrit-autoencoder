parameśaṃ namaskṛtya bharadvājam ṛṣiṃ tataḥ / (1.1)
	harād indrakramāyātaṃ jñānaṃ śṛṇuta suvratāḥ // (1.2)
	nārāyaṇāśrame puṇye bharadvājādayo dvijāḥ / (2.1)
	tepuḥ śivaṃ pratiṣṭhāpya tadekāhitamānasāḥ // (2.2)
	atha tān bhāvitān matvā kadācit tridaśādhipaḥ / (3.1)
	tad āśramapadaṃ bheje svayaṃ tāpasaveṣabhṛt // (3.2)
	sa taiḥ sampūjitaḥ pṛṣṭvā tāṃś ca sarvān anāmayam / (4.1)
	provāca codanādharmaḥ kim arthaṃ nānuvartyate // (4.2)
	ta ūcur nanv ayaṃ dharmaś codanāvihito mune / (5.1)
	devatārādhanopāyas tapasābhīṣṭasiddhaye // (5.2)
	vede 'sti saṃhitā raudrī vācyā rudraś ca devatā / (6.1)
	sāṃnidhyakaraṇe 'py asmin vihitaḥ kālpiko vidhiḥ // (6.2)
	ity ukte 'pi paraṃ bhāvaṃ jijñāsuḥ prahasan prabhuḥ / (7.1)
	tān āha mithyā jñānaṃ vaḥ śabdamātraṃ hi devatā // (7.2)
	śabdetaratve yugapadbhinnadeśeṣu yaṣṭṛṣu / (8.1)
	na sā prayāti sāṃnidhyaṃ mūrtatvād asmadādivat // (8.2)
	na ca tatsādhakaṃ kiṃcit pramāṇaṃ bhāty abādhitam / (9.1)
	vākyaṃ tad anyathāsiddhaṃ lokavādāḥ kva sādhavaḥ // (9.2)
	ity anīśavacovārivelānunno 'bdhineva saḥ / (10.1)
	śakreṇa na cacālaiṣāṃ dhīśailaḥ sāragauravāt // (10.2)
	na jātu devatāmūrtir asmadādiśarīravat / (11.1)
	viśiṣṭaiśvaryasampannā sāto naitan nidarśanam // (11.2)
	athāstv evaṃ ghaṭe nyāyaḥ śabdatvād indraśabdavat / (12.1)
	nādatte ghaṭaśabdo 'mbhaś candraśabdo na rājate // (12.2)
	athānyaviṣayaṃ vākyam astu śakrādivācakam / (13.1)
	karmarūpādiśabdānāṃ sārthakatvaṃ kathaṃ bhavet // (13.2)
	pravādo 'py akhilo mithyā samūlatvān na yuktimat / (14.1)
	sa ced amūlo bhūtānāṃ hatāḥ sarvāḥ pravṛttayaḥ // (14.2)
	upamanyur haraṃ dṛṣṭvā vimanyur abhavan muniḥ / (15.1)
	kathaṃ tasya vaco mithyā yasya vaśyaḥ payonidhiḥ // (15.2)
	kroḍīkṛto hi pāśena viṣajvālāvalīmucā huṃkṛtya / (16.1)
	mocitaḥ patyā dṛṣṭaḥ śveto ghanair janaiḥ // (16.2)
	iti vādānuṣaṅgeṇa haraśaṃsāpraharṣitān / (17.1)
	sāśrugadgadavācas tān vīkṣya prīto 'bhavad dhīraḥ // (17.2)
	svaṃ rūpaṃ darśayāmāsa vajrī devaḥ śatakratuḥ / (18.1)
	taruṇādityasaṃkāśaṃ stūyamānaṃ marudgaṇaiḥ // (18.2)
	te tam ṛgbhir yajurbhiś ca sāmabhiś cāstuvan natāḥ / (19.1)
	so 'bravīd ucyatāṃ kāmo jagatsu pravaro 'pi yaḥ // (19.2)
	te vavrire śivajñānaṃ śrūyatām iti so 'bravīt / (20.1)
	kiṃ tv eko 'stu mama praṣṭā nikhilaśrotṛsaṃmataḥ // (20.2)
	atha teṣāṃ bharadvājo bhagavān agraṇīr abhūt / (21.1)
	vāgmī pragalbhaḥ papraccha nyāyataḥ surapūjitam // (21.2)
	kathaṃ maheśvarād etad āgataṃ jñānam uttamam / (22.1)
	kiṃ ca cetasi saṃsthāpya nirmame bhagavān idam // (22.2)
	sṛṣṭikāle maheśānaḥ puruṣārthaprasiddhaye / (23.1)
	vidhatte vimalaṃ jñānaṃ pañcasroto 'bhilakṣitam // (23.2)
	tadvartivācakavrātavācyān aṣṭau maheśvarān / (24.1)
	saptakoṭiprasaṃkhyātān mantrāṃś ca parame 'dhvani // (24.2)
	aṣṭādaśādhikaṃ cānyac chrutaṃ māyādhikāriṇām / (25.1)
	mantreśvarāṇām ūrdhvādhvasthiteśopamatejasām // (25.2)
	teṣu vyaktaḥ sa bhagavān idaṃ yogyeṣu siddhaye / (26.1)
	prakāśayaty ato 'nyeṣu yo 'rthaḥ samupapadyate // (26.2)
	śivodgīrṇam idaṃ jñānaṃ mantramantreśvareśvaraiḥ / (27.1)
	kāmadatvāt kāmiketi pragītaṃ bahuvistaram // (27.2)
	tebhyo 'vagatya dṛgjyotirjvālālīḍhasmaradrumaḥ / (28.1)
	dadāv umāpatir mahyaṃ sahasrair bhavasaṃmitaiḥ // (28.2)
	tatrāpi vistaraṃ hitvā sūtraiḥ sārārthavācakaiḥ / (29.1)
	vakṣye nirākulaṃ jñānaṃ taduktair eva bhūyasā // (29.2)
