vratamupaiṣyan / (1.1)
	antareṇāhavanīyaṃ ca gārhapatyaṃ ca prāṅ tiṣṭhann apa upaspṛśati tadyadapa upaspṛśatyamedhyo vai puruṣo yadanṛtaṃ vadati tena pūtirantarato medhyā vā āpo medhyo bhūtvā vratamupāyānīti pavitraṃ vā āpaḥ pavitrapūto vratamupāyānīti tasmādvā apa upaspṛśati // (1.2)
	so 'gnim evābhīkṣamāṇo vratamupaiti / (2.1)
	agne vratapate vrataṃ cariṣyāmi taccakeyaṃ tanme rādhyatām ityagnir vai devānāṃ vratapatistasmā evaitatprāha vrataṃ cariṣyāmi taccakeyaṃ tanme rādhyatāmiti nātra tirohitamivāsti // (2.2)
	atha saṃsthite visṛjate / (3.1)
	agne vratapate vratamacāriṣaṃ tādaśakam tanme rādhītyaśakadyetadyo yajñasya saṃsthām agann arādhi hyasmai yo yajñasya saṃsthām agannetena nveva bhūyiṣṭhā iva vratamupayantyanena tvevopeyāt // (3.2)
	dvayaṃ vā idaṃ na tṛtīyamasti / (4.1)
	satyaṃ caivānṛtaṃ ca satyameva devā anṛtam manuṣyā idamahamanṛtātsatyamupaimīti tanmanuṣyebhyo devānupaiti // (4.2)
	sa vai satyameva vadet / (5.1)
	etaddhavai devā vrataṃ caranti yatsatyaṃ tasmātte yaśo yaśo ha bhavati ya evaṃ vidvāṃt satyaṃ vadati // (5.2)
	atha saṃsthite visṛjate / (6.1)
	idamahaṃ ya evāsmi so 'smītyamānuṣa iva vā etadbhavati yadvratamupaiti na hi tadavakalpate yadbrūyādidamahaṃ satyādanṛtamupaimīti tad u khalu punarmānuṣo bhavati tasmādidamahaṃ ya evāsmi so 'smītyevaṃ vrataṃ visṛjeta // (6.2)
	athāto 'śanānaśanasyaiva / (7.1)
	tad u hāṣāḍhaḥ sāvayaso 'naśanameva vratam mene mano ha vai devā manuṣyasyājānanti ta enametadvratamupayantaṃ viduḥ prātarno yakṣyata iti te 'sya viśve devā gṛhān āgacchanti te 'sya gṛheṣūpavasanti sa upavasathaḥ // (7.2)
	tannvevānavakąptam / (8.1)
	yo manuṣyeṣvanaśnatsu pūrvo 'śnīyādatha kimu yo deveṣvanaśnatsu pūrvo 'śnīyāt tasmād u naivāśnīyāt // (8.2)
	tad u hovāca yājñavalkyaḥ / (9.1)
	yadi nāśnāti pitṛdevatyo bhavati yady u aśnāti devānatyaśnātīti sa yadevāśitam anaśitaṃ tadaśnīyāditi yasya vai havirna gṛhṇanti tadaśitam anaśitaṃ sa yadaśnāti tenāpitṛdevatyo bhavati yady u tadaśnāti yasya havirna gṛhṇanti teno devānnātyaśnāti // (9.2)
	sa vā āraṇyamevāśnīyāt / (10.1)
	yā vāraṇyā oṣadhayo yadvā vṛkṣyaṃ tad u ha smāhāpi barkurvārṣṇo māsānme pacata na vā etesāṃ havirgṛhṇantīti tad u tathā na kuryādvrīhiyavayorvā etadupajaṃ yacamīdhānyaṃ tadvrīhiyavāvevaitena bhūyāṃsau karoti tasmādāraṇyamevāśnīyāt // (10.2)
	sa āhavanīyāgāre vaitāṃ rātrīṃ śayīta / (11.1)
	gārhapatyāgāre vā devānvā eṣa upāvartate yo vratamupaiti sa yānevopāvartate teṣāmevaitanmadhye śete 'dhaḥ śayītādhastādiva hi śreyasa upacāraḥ // (11.2)
	sa vai prātarapa eva / (12.1)
	prathamena karmaṇābhipadyate 'paḥ praṇayati yajño vā āpo yajñamevaitatprathamena karmaṇābhipadyate tāḥ praṇayati yajñamevaitadvitanoti // (12.2)
	sa praṇayati / (13.1)
	kastvā yunakti sa tvā yunakti kasmai tvā yunakti tasmai tvā yunaktītyetābhiraniruktābhirvyāhṛtibhiranirukto vai prajāpatiḥ prajāpatiryajñastatprajāpatimevaitadyajñaṃ yunakti // (13.2)
	yad v evāpaḥ praṇayati / (14.1)
	adbhirvā idaṃ sarvamāptaṃ tatprathamenaivaitatkarmaṇā sarvamāpnoti // (14.2)
	yad v evāsyātra / (15.1)
	hotā vādhvaryurvā brahmā vāgnīdhro vā svayaṃ vā yajamāno nābhyāpayati tadevāsyaitena sarvam āptam bhavati // (15.2)
	yad v evāpaḥ praṇayati / (16.1)
	devānha vai yajñena yajamānāṃstān asurarakṣasāni rarakṣurna yakṣyadhva iti tadyadarakṣaṃstasmādrakṣāṃsi // (16.2)
	tato devā etaṃ vajraṃ dadṛśuḥ / (17.1)
	yadapo vajro vā āpo vajro hi vā āpastasmādyenaitā yanti nimnaṃ kurvanti yatropatiṣṭhante nirdahanti tata etaṃ vajram udayacchaṃs tasyābhaye 'nāṣṭre nivāte yajñamatanvata tatho evaiṣa etaṃ vajram udyacchati tasyābhaye 'nāṣṭre nivāte yajñaṃ tanute tasmādapaḥ praṇayati // (17.2)
	tā utsicyottareṇa gārhapatyaṃ sādayati / (18.1)
	yoṣā vā āpo vṛṣāgnirgṛhā vai gārhapatyastadgṛheṣvevaitanmithunam prajananaṃ kriyate vajraṃ vā eṣa udyacchati yo 'paḥ praṇayati yo vā apratiṣṭhito vajram udyacchati nainaṃ śaknotyudyantuṃ saṃ hainaṃ śṛṇāti // (18.2)
	sa yadgārhapatye sādayati / (19.1)
	gṛhā vai gārhapatyo gṛhā vai pratiṣṭhāyām pratitiṣṭhati tatho hainameṣa vajro na hinasti tasmādgārhapatye sādayati // (19.2)
	tā uttareṇāhavanīyam praṇayati / (20.1)
	yoṣā vā āpo vṛṣāgnir mithunam evaitat prajananaṃ kriyata evamiva hi mithunaṃ kąptam uttarato hi strī pumāṃsamupaśete // (20.2)
	tā nāntareṇa saṃcareyuḥ / (21.1)
	nen mithunaṃ caryamāṇamantareṇa saṃcarāṇīti tā nātihṛtya sādayenno anāptāḥ sādayetsa yadatihṛtya sādayedasti vā agneścāpāṃ ca vibhrātṛvyamiva sa yatheva ha tadagnerbhavati yatrāsyāpa upaspṛśantyagnau hādhi bhrātṛvyaṃ vardhayedyadatihṛtya sādayed yadya anāptāḥ sādayenno hābhistaṃ kāmam abhyāpayed yasmai kāmāya praṇīyante tasmād u sampratyevottareṇāhavanīyam praṇayati // (21.2)
	atha tṛṇaiḥ paristṛṇāti / (22.1)
	dvandvam pātrāṇyudāharati śūrpaṃ cāgnihotrahavaṇīṃ ca sphyaṃ ca kapālāni ca śamyāṃ cakṛṣṇājinaṃ colūkhalamusale dṛṣadupale taddaśa daśākṣarā vai virāḍvirāḍvai yajñastadvirājamevaitad yajñam abhisaṃpādayatyatha yaddvandvaṃ dvandvaṃ vai vīryaṃ yadā vai dvau saṃrabhete atha tadvīryam bhavati dvandvaṃ vai mithunam prajananam mithunamevaitatprajananaṃ kriyate // (22.2)
