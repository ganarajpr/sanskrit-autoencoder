pracaṇḍatāṇḍavāṭope prakṣiptā yena diggajāḥ / (1.1)
	bhavantu vighnabhaṅgāya bhavasya caraṇāmbujāḥ // (1.2)
	pātālādutpatiṣṇor makaravasatayo yasya pucchābhighātād ūrdhvaṃ brahmāṇḍakhaṇḍavyatikaravihitavyatyayenāpatanti / (2.1)
	viṣṇormatsyāvatāre sakalavasumatīmaṇḍalaṃ vyaśnuvānās tasyāsyodīritānāṃ dhvanir apaharatād aśriyaṃ vaḥ śrutīnām / (2.2)
	nārāyaṇaṃ namaskṛtya naraṃ caiva narottamam / (2.3)
	devīṃ sarasvatīṃ caiva tato jayam udīrayet // (2.4)
	ajo'pi yaḥ kriyāyogānnārāyaṇa iti smṛtaḥ / (3.1)
	triguṇāya trivedāya namas tasmai svayambhave // (3.2)
	sūtamekāgramāsīnaṃ naimiṣāraṇyavāsinaḥ / (4.1)
	munayo dīrghasattrānte papracchur dīrghasaṃhitām // (4.2)
	pravṛttāsu purāṇīṣu dharmyāsu lalitāsu ca / (5.1)
	kathāsu śaunakādyās tu abhinandya muhurmuhuḥ // (5.2)
	kathitāni purāṇāni yānyasmākaṃ tvayānagha / (6.1)
	tānyevāmṛtakalpāni śrotum icchāmahe punaḥ // (6.2)
	kathaṃ sasarja bhagavaṃl lokanāthaścarācaram / (7.1)
	kasmācca bhagavān viṣṇur matsyarūpatvam āśritaḥ // (7.2)
	bhairavatvaṃ bhavasyāpi purāritvaṃ ca kena hi / (8.1)
	kasya hetoḥ kapālitvaṃ jagāma vṛṣabhadhvajaḥ // (8.2)
	sarvam etat samācakṣva sūta vistaraśaḥ kramāt / (9.1)
	tvadvākyenāmṛtasyeva na tṛptiriha jāyate // (9.2)
	sūta uvāca / (10.1)
	puṇyaṃ pavitram āyuṣyam idānīṃ śṛṇuta dvijāḥ / (10.2)
	mātsyaṃ purāṇamakhilaṃ yajjagāda gadādharaḥ // (10.3)
	purā rājā manur nāma cīrṇavān vipulaṃ tapaḥ / (11.1)
	putre rājyaṃ samāropya kṣamāvān ravinandanaḥ // (11.2)
	malayasyaikadeśe tu sarvātmaguṇasaṃyutaḥ / (12.1)
	samaduḥkhasukho vīraḥ prāptavān yogam uttamam // (12.2)
	babhūva varadaś cāsya varṣāyutaśate gate / (13.1)
	varaṃ vṛṇīṣva provāca prītaḥ sa kamalāsanaḥ // (13.2)
	evamukto 'bravīd rājā praṇamya sa pitāmaham / (14.1)
	ekam evāham icchāmi tvatto varamanuttamam // (14.2)
	bhūtagrāmasya sarvasya sthāvarasya carasya ca / (15.1)
	bhaveyaṃ rakṣaṇāyālaṃ pralaye samupasthite // (15.2)
	evamastviti viśvātmā tatraivāntaradhīyata / (16.1)
	puṣpavṛṣṭiḥ sumahatī khātpapāta surārpitā // (16.2)
	kadācidāśrame tasya kurvataḥ pitṛtarpaṇam / (17.1)
	papāta pāṇyor upari śapharī jalasaṃyutā // (17.2)
	dṛṣṭvā tacchapharīrūpaṃ sa dayālurmahīpatiḥ / (18.1)
	rakṣaṇāyākarodyatnaṃ sa tasminkarakodare // (18.2)
	ahorātreṇa caikena ṣoḍaśāṅgulavistṛtaḥ / (19.1)
	so 'bhavanmatsyarūpeṇa pāhi pāhīti cābravīt // (19.2)
	sa tamādāya maṇike prākṣipajjalacāriṇam / (20.1)
	tatrāpi caikarātreṇa hastatrayam avardhata // (20.2)
	punaḥ prāhārtanādena sahasrakiraṇātmajam / (21.1)
	sa matsyaḥ pāhi pāhīti tvāmahaṃ śaraṇaṃ gataḥ // (21.2)
	tataḥ sa kūpe taṃ matsyaṃ prāhiṇod ravinandanaḥ / (22.1)
	yadā na māti tatrāpi kūpe matsyaḥ sarovare // (22.2)
	kṣipto 'sau pṛthutāmāgātpunar yojanasaṃmitām / (23.1)
	tatrāpyāha punar dīnaḥ pāhi pāhi nṛpottama // (23.2)
	tataḥ sa manunā kṣipto gaṅgāyāmapyavardhata / (24.1)
	yadā tadā samudre taṃ prākṣipanmedinīpatiḥ // (24.2)
	yadā samudramakhilaṃ vyāpyāsau samupasthitaḥ / (25.1)
	tadā prāha manurbhītaḥ ko 'pi tvamasureśvaraḥ // (25.2)
	athavā vāsudevastvamanya īdṛkkathaṃ bhavet / (26.1)
	yojanāyutaviṃśatyā kasya tulyaṃ bhavedvapuḥ // (26.2)
	jñātastvaṃ matsyarūpeṇa māṃ khedayasi keśava / (27.1)
	hṛṣīkeśa jagannātha jagaddhāma namo'stu te // (27.2)
	evamuktaḥ sa bhagavānmatsyarūpī janārdanaḥ / (28.1)
	sādhu sādhviti covāca samyagjñātas tvayānagha // (28.2)
	acireṇaiva kālena medinī medinīpate / (29.1)
	bhaviṣyati jale magnā saśailavanakānanā // (29.2)
	naur iyaṃ sarvadevānāṃ nikāyena vinirmitā / (30.1)
	mahājīvanikāyasya rakṣaṇārthaṃ mahīpate // (30.2)
	svedāṇḍajodbhido ye vai ye ca jīvā jarāyujāḥ / (31.1)
	asyāṃ nidhāya sarvāṃs tān anāthān pāhi suvrata // (31.2)
	yugāntavātābhihatā yadā bhavati naur nṛpa / (32.1)
	śṛṅge 'sminmama rājendra tademāṃ saṃyamiṣyasi // (32.2)
	tato layānte sarvasya sthāvarasya carasya ca / (33.1)
	prajāpatistvaṃ bhavitā jagataḥ pṛthivīpate // (33.2)
	evaṃ kṛtayugasyādau sarvajño dhṛtimānnṛpaḥ / (34.1)
	manvantarādhipaścāpi devapūjyo bhaviṣyasi // (34.2)
