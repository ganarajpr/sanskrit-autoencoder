dharmaikatānāḥ puruṣā yadāsan satyavādinaḥ / (1.1)
	tadā na vyavahāro 'bhūn na dveṣo nāpi matsaraḥ // (1.2)
	naṣṭe dharme manuṣyeṣu vyavahāraḥ pravartate / (2.1)
	draṣṭā ca vyavahārāṇāṃ rājā daṇḍadharaḥ kṛtaḥ // (2.2)
	likhitaṃ sākṣiṇaś cātra dvau vidhī saṃprakīrtitau / (3.1)
	saṃdigdhārthaviśuddhyarthaṃ dvayor vivadamānayoḥ // (3.2)
	sottaro 'nuttaraś caiva sa vijñeyo dvilakṣaṇaḥ / (4.1)
	sottaro 'bhyadhiko yatra vilekhāpūrvakaḥ paṇaḥ // (4.2)
	vivāde sottarapaṇe dvayor yas tatra hīyate / (5.1)
	sa paṇaṃ svakṛtaṃ dāpyo vinayaṃ ca parājaye // (5.2)
	sāras tu vyavahārāṇāṃ pratijñā samudāhṛtā / (6.1)
	taddhānau hīyate vādī taraṃs tām uttaro bhavet // (6.2)
	kulāni śreṇayaś caiva gaṇāś cādhikṛto nṛpaḥ / (7.1)
	pratiṣṭhā vyavahārāṇāṃ gurvebhyas tūttarottaram // (7.2)
	sa catuṣpāc catuḥsthānaś catuḥsādhana eva ca / (8.1)
	caturhitaś caturvyāpī catuṣkārī ca kīrtyate // (8.2)
	aṣṭāṅgo 'ṣṭādaśapadaḥ śataśākhas tathaiva ca / (9.1)
	triyonir dvyabhiyogaś ca dvidvāro dvigatis tathā // (9.2)
	dharmaś ca vyavahāraś ca caritraṃ rājaśāsanam / (10.1)
	catuṣpād vyavahāro 'yam uttaraḥ pūrvabādhakaḥ // (10.2)
	tatra satye sthito dharmo vyavahāras tu sākṣiṣu / (11.1)
	caritraṃ pustakaraṇe rājājñāyāṃ tu śāsanam // (11.2)
	sāmādyupāyasādhyatvāc catuḥsādhana ucyate / (12.1)
	caturṇām āśramāṇāṃ ca rakṣaṇāt sa caturhitaḥ // (12.2)
	kartṝn atho sākṣiṇaś ca sabhyān rājānam eva ca / (13.1)
	vyāpnoti pādaśo yasmāc caturvyāpī tataḥ smṛtaḥ // (13.2)
	dharmasyārthasya yaśaso lokapaktes tathaiva ca / (14.1)
	caturṇāṃ karaṇād eṣāṃ catuṣkārī prakīrtitaḥ // (14.2)
	rājā sapuruṣaḥ sabhyāḥ śāstraṃ gaṇakalekhakau / (15.1)
	hiraṇyam agnir udakam aṣṭāṅgaḥ sa udāhṛtaḥ // (15.2)
	ṛṇādānaṃ hy upanidhiḥ sambhūyotthānam eva ca / (16.1)
	dattasya punar ādānam aśuśrūṣābhyupetya ca // (16.2)
	vetanasyānapākarma tathaivāsvāmivikrayaḥ / (17.1)
	vikrīyāsaṃpradānaṃ ca krītvānuśaya eva ca // (17.2)
	samayasyānapākarma vivādaḥ kṣetrajas tathā / (18.1)
	strīpuṃsayoś ca saṃbandho dāyabhāgo 'tha sāhasam // (18.2)
	vākpāruṣyaṃ tathaivoktaṃ daṇḍapāruṣyam eva ca / (19.1)
	dyūtaṃ prakīrṇakaṃ caivety aṣṭādaśapadaḥ smṛtaḥ // (19.2)
	eṣām eva prabhedo 'nyaḥ śatam aṣṭottaraṃ smṛtam / (20.1)
	kriyābhedān manuṣyāṇāṃ śataśākho nigadyate // (20.2)
	kāmāt krodhāc ca lobhāc ca tribhyo yasmāt pravartate / (21.1)
	triyoniḥ kīrtyate tena trayam etad vivādakṛt // (21.2)
	dvyabhiyogas tu vijñeyaḥ śaṅkātattvābhiyogataḥ / (22.1)
	śaṅkāsatāṃ tu saṃsargāt tattvaṃ hoḍhādidarśanāt // (22.2)
	pakṣadvayābhisaṃbandhād dvidvāraḥ samudāhṛtaḥ / (23.1)
	pūrvavādas tayoḥ pakṣaḥ pratipakṣas taduttaram // (23.2)
	bhūtacchalānusāritvād dvigatiḥ sa udāhṛtaḥ / (24.1)
	bhūtaṃ tattvārthasaṃyuktaṃ pramādābhihitaṃ chalam // (24.2)
	tatra śiṣṭaṃ chalaṃ rājā marṣayed dharmasādhanaḥ / (25.1)
	bhūtam eva prapadyeta dharmamūlā yataḥ śriyaḥ // (25.2)
	dharmeṇoddharato rājño vyavahārān kṛtātmanaḥ / (26.1)
	sambhavanti guṇāḥ sapta sapta vahner ivārciṣaḥ // (26.2)
	dharmaś cārthaś ca kīrtiś ca lokapaktir upagrahaḥ / (27.1)
	prajābhyo bahumānaś ca svarge sthānaṃ ca śāśvatam // (27.2)
	tasmād dharmāsanaṃ prāpya rājā vigatamatsaraḥ / (28.1)
	samaḥ syāt sarvabhūteṣu bibhrad vaivasvataṃ vratam // (28.2)
	dharmaśāstraṃ puraskṛtya prāḍvivākamate sthitaḥ / (29.1)
	samāhitamatiḥ paśyed vyavahārān anukramāt // (29.2)
	āgamaḥ prathamaṃ kāryo vyavahārapadaṃ tataḥ / (30.1)
	vivitsā nirṇayaś caiva darśanaṃ syāc caturvidham // (30.2)
	dharmaśāstrārthaśāstrābhyām avirodhena mārgataḥ / (31.1)
	samīkṣamāṇo nipuṇaṃ vyavahāragatiṃ nayet // (31.2)
	yathā mṛgasya viddhasya vyādho mṛgapadaṃ nayet / (32.1)
	kakṣe śoṇitaleśena tathā dharmapadaṃ nayet // (32.2)
	yatra vipratipattiḥ syād dharmaśāstrārthaśāstrayoḥ / (33.1)
	arthaśāstroktam utsṛjya dharmaśāstroktam ācaret // (33.2)
	dharmaśāstravirodhe tu yuktiyukto 'pi dharmataḥ / (34.1)
	vyavahāro hi balavān dharmas tenāvahīyate // (34.2)
	sūkṣmo hi bhagavān dharmaḥ parokṣo durvicāraṇaḥ / (35.1)
	ataḥ pratyakṣamārgeṇa vyavahāragatiṃ nayet // (35.2)
	yāty acauro 'pi cauratvaṃ cauraś cāyāty acauratām / (36.1)
	acauraś cauratāṃ prāpto māṇḍavyo vyavahārataḥ // (36.2)
	strīṣu rātrau bahir grāmād antarveśmany arātiṣu / (37.1)
	vyavahāraḥ kṛto 'py eṣu punaḥ kartavyatām iyāt // (37.2)
	gahanatvād vivādānām asāmarthyāt smṛter api / (38.1)
	ṛṇādiṣu haret kālaṃ kāmaṃ tattvabubhutsayā // (38.2)
	gobhūhiraṇyastrīsteyapāruṣyātyayikeṣu ca / (39.1)
	sāhaseṣv abhiśāpe ca sadya eva vivādayet // (39.2)
	anāvedya tu yo rājñe saṃdigdhe 'rthe pravartate / (40.1)
	prasahya sa vineyaḥ syāt sa cāsyārtho na sidhyati // (40.2)
	vaktavye 'rthe na tiṣṭhantam utkrāmantaṃ ca tadvacaḥ / (41.1)
	āsedhayed vivādārthī yāvad āhvānadarśanam // (41.2)
	sthānāsedhaḥ kālakṛtaḥ pravāsāt karmaṇas tathā / (42.1)
	caturvidhaḥ syād āsedho nāsiddhas taṃ vilaṅghayet // (42.2)
	nadīsaṃtārakāntāradurdeśopaplavādiṣu / (43.1)
	āsiddhas taṃ parāsedham utkrāman nāparādhnuyāt // (43.2)
	āsedhakāla āsiddha āsedham yo vyatikramet / (44.1)
	sa vineyo 'nyathā kurvann āseddhā daṇḍabhāg bhavet // (44.2)
	nirveṣṭukāmo rogārto yiyakṣur vyasane sthitaḥ / (45.1)
	abhiyuktas tathānyena rājakāryodyatas tathā // (45.2)
	gavāṃ pracāre gopālāḥ sasyabandhe kṛṣīvalāḥ / (46.1)
	śilpinaḥ cāpi tatkālam āyudhīyāś ca vigrahe // (46.2)
	aprāptavyavahāraś ca dūto dānonmukho vratī / (47.1)
	viṣamasthaś ca nāsedhyo na cainān āhvayen nṛpaḥ // (47.2)
	nābhiyukto 'bhiyuñjīta tam atīrtvārtham anyataḥ / (48.1)
	na cābhiyuktam anyena na viddhaṃ veddhum arhati // (48.2)
	yam artham abhiyuñjīta na taṃ viprakṛtiṃ nayet / (49.1)
	nānyat pakṣāntaraṃ gacched gacchan pūrvāt sa hīyate // (49.2)
	na ca mithyābhiyuñjīta doṣo mithyābhiyoginaḥ / (50.1)
	yas tatra vinayaḥ proktaḥ so 'bhiyoktāram āvrajet // (50.2)
	sāpadeśaṃ haran kālam abruvaṃś cāpi saṃsadi / (51.1)
	uktvā vaco vibruvaṃś ca hīyamānasya lakṣaṇam // (51.2)
	palāyate ya āhūtaḥ prāptaś ca vivaden na yaḥ / (52.1)
	vineyaḥ sa bhaved rājñā hīna eva sa vādataḥ // (52.2)
	nirṇiktavyavahāreṣu pramāṇam aphalaṃ bhavet / (53.1)
	likhitaṃ sākṣiṇo vāpi pūrvam āveditaṃ na cet // (53.2)
	yathā pakveṣu dhānyeṣu niṣphalāḥ prāvṛṣo guṇāḥ / (54.1)
	nirṇiktavyavahārāṇāṃ pramāṇam aphalaṃ tathā // (54.2)
	abhūtam apy abhihitaṃ prāptakālaṃ parīkṣyate / (55.1)
	yat tu pramādān nocyeta tad bhūtam api hīyate // (55.2)
	tīritaṃ cānuśiṣṭaṃ ca yo manyeta vidharmataḥ / (56.1)
	dviguṇaṃ daṇḍam āsthāya tat kāryaṃ punar uddharet // (56.2)
	durdṛṣṭe vyavahāre tu sabhyās taṃ daṇḍam āpnuyuḥ / (57.1)
	na hi jātu vinā daṇḍaṃ kaścin mārge 'vatiṣṭhate // (57.2)
	rāgād ajñānato vāpi lobhād vā yo 'nyathā vadet / (58.1)
	sabhyo 'sabhyaḥ sa vijñeyas taṃ rājā vinayed bhṛśam // (58.2)
	kiṃtu rājñā viśeṣeṇa svadharmam anurakṣatā / (59.1)
	manuṣyacittavaicitryāt parīkṣyā sādhvasādhutā // (59.2)
	puruṣāḥ santi ye lobhāt prabrūyuḥ sākṣyam anyathā / (60.1)
	santi cānye durātmānaḥ kūṭalekhyakṛto janāḥ // (60.2)
	ataḥ parīkṣyam ubhayam etad rājñā viśeṣataḥ / (61.1)
	lekhyācāreṇa likhitaṃ sākṣyācāreṇa sākṣiṇaḥ // (61.2)
	asatyāḥ satyasaṃkāśāḥ satyāś cāsatyadarśanāḥ / (62.1)
	dṛśyante vividhā bhāvās tasmād yuktaṃ parīkṣaṇam // (62.2)
	talavad dṛśyate vyoma khadyoto havyavāḍ iva / (63.1)
	na talaṃ vidyate vyomni na khadyote hutāśanaḥ // (63.2)
	tasmāt pratyakṣadṛṣṭo 'pi yuktam arthaḥ parīkṣitum / (64.1)
	parīkṣya jñāpayan arthān na dharmāt parihīyate // (64.2)
	evaṃ paśyan sadā rājā vyavahārān samāhitaḥ / (65.1)
	vitatyeha yaśo dīptaṃ bradhnasyāpnoti viṣṭapam // (65.2)
