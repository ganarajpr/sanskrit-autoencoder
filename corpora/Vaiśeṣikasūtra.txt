athāto dharmaṃ vyākhyāsyāmaḥ // (1.1)
	yato 'bhyudayaniḥśreyasasiddhiḥ sa dharmaḥ // (2.1)
	tadvacanād āmnāyaprāmāṇyam // (3.1)
	pṛthivy āpas tejo vāyur ākāśaṃ kālo dig ātmā mana iti dravyāṇi // (4.1)
	rūparasagandhasparśāḥ saṃkhyāḥ parimāṇāni pṛthaktvaṃ saṃyogavibhāgau paratvāparatve buddhayaḥ sukhaduḥkhe icchādveṣau prayatnaś ca guṇāḥ // (5.1)
	utkṣepaṇam avakṣepaṇam ākuñcanaṃ prasāraṇaṃ gamanam iti karmāṇi // (6.1)
	sad anityaṃ dravyavat kāryaṃ kāraṇaṃ sāmānyaviśeṣavad iti dravyaguṇakarmaṇām aviśeṣaḥ // (7.1)
	dravyāṇi dravyāntaram ārabhante // (8.1)
	guṇāś ca guṇāntaram // (9.1)
	karma karmasādhyaṃ na vidyate // (10.1)
	kāryāvirodhi dravyaṃ kāraṇāvirodhi ca // (11.1)
	ubhayathā guṇaḥ // (12.1)
	kāryavirodhi karma // (13.1)
	kriyāvad guṇavat samavāyikāraṇam iti dravyalakṣaṇam // (14.1)
	dravyāśrayy aguṇavān saṃyogavibhāgeṣv akāraṇam anapekṣa iti guṇalakṣaṇam // (15.1)
	ekadravyam aguṇaṃ saṃyogavibhāgeṣv anapekṣaṃ kāraṇam iti karmalakṣaṇam // (16.1)
	dravyaguṇakarmaṇāṃ dravyaṃ kāraṇaṃ sāmānyam // (17.1)
	tathā guṇaḥ // (18.1)
	saṃyogavibhāgānāṃ karma // (19.1)
	na dravyāṇāṃ vyatirekāt // (20.1)
	guṇavaidharmyān na karmaṇāṃ // (21.1)
	dravyāṇāṃ dravyaṃ kāryaṃ sāmānyam // (22.1)
	dvitvaprabhṛtayaś ca saṃkhyāḥ pṛthaktvaṃ saṃyogavibhāgāś ca // (23.1)
	asamavāyāt sāmānyaṃ karma kāryaṃ na vidyate // (24.1)
	saṃyogānāṃ dravyam // (25.1)
	rūpānāṃ rūpam // (26.1)
	gurutvaprayatnasaṃyogānām utkṣepaṇam // (27.1)
	saṃyogavibhāgāḥ karmaṇām // (28.1)
	kāraṇasāmānye dravyakarmaṇāṃ karmākāraṇam uktam // (29.1)
