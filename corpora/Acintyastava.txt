pratītyajānāṃ bhāvānāṃ naiḥsvābhāvyaṃ jagāda yaḥ / (1.1)
	taṃ namāmy asamajñānam acintyam anidarśanam // (1.2)
	yathā tvayā mahāyāne dharmanairātmyam ātmanā / (2.1)
	viditaṃ deśitaṃ tadvad dhīmadbhyaḥ karuṇāvaśāt // (2.2)
	pratyayebhyaḥ samutpannam anutpannaṃ tvayoditam / (3.1)
	svabhāvena na taj jātam iti śūnyaṃ prakāśitam // (3.2)
	yadvacchabdaṃ pratītyeha pratiśabdasamudbhavaḥ / (4.1)
	māyāmarīcivac cāpi tathā bhavasamudbhavaḥ // (4.2)
	māyāmarīcigandharvanagarapratibimbakāḥ / (5.1)
	yady ajātāḥ saha svapnair na syāt taddarśanādikam // (5.2)
	hetupratyayasambhūtā yathaite kṛtakāḥ smṛtāḥ / (6.1)
	tadvat pratyayajaṃ viśvaṃ tvayoktaṃ nātha sāṃvṛtam // (6.2)
	asty etat kṛtakaṃ sarvaṃ yat kiṃcid bālalāpanam / (7.1)
	riktamuṣṭipratīkāśam ayathārthaprakāśitam // (7.2)
	kṛtakaṃ vastu no jātaṃ tadā kiṃ vārtamānikam / (8.1)
	kasya nāśād atītaṃ syād utpitsuḥ kim apekṣate // (8.2)
	svasmān na jāyate bhāvaḥ parasmān nobhayād api / (9.1)
	na san nāsan na sadasan kutaḥ kasyodayas tadā // (9.2)
	ajāte na svabhāvo 'sti kutaḥ svasmāt samudbhavaḥ / (10.1)
	svabhāvābhāvasiddhyaiva parasmād apy asaṃbhavaḥ // (10.2)
	svatve sati paratvaṃ syāt paratve svatvam iṣyate / (11.1)
	āpekṣikī tayoḥ siddhiḥ pārāvāram ivoditā // (11.2)
	yadā nāpekṣate kiṃcit kutaḥ kiṃcit tadā bhavet / (12.1)
	yadā nāpekṣate dīrghaṃ kuto hrasvādikaṃ tadā // (12.2)
	astitve sati nāstitvaṃ dīrghe hrasvaṃ tathā sati / (13.1)
	nāstitve sati cāstitvaṃ yat tasmād ubhayaṃ na sat // (13.2)
	ekatvaṃ ca tathānekam atītānāgatādi ca / (14.1)
	saṃkleśo vyavadānaṃ ca samyaṅmithyā svataḥ kutaḥ // (14.2)
	svata eva hi yo nāsti bhāvaḥ sarvo 'sti kas tadā / (15.1)
	para ity ucyate yo 'yaṃ na vinā svasvabhāvataḥ // (15.2)
	na svabhāvo 'sti bhāvānāṃ parabhāvo 'sti no yadā / (16.1)
	bhāvagrahagrahāveśaḥ paratantro 'sti kas tadā // (16.2)
	ādāv eva samaṃ jātāḥ svabhāvena ca nirvṛtāḥ / (17.1)
	anutpannāś ca tattvena tasmād dharmās tvayoditāḥ // (17.2)
	niḥsvabhāvās tvayā dhīman rūpādyāḥ saṃprakāśitāḥ / (18.1)
	phenabudbudamāyābhramarīcikadalīsamāḥ // (18.2)
	indriyair upalabdhaṃ yat tat tattvena bhaved yadi / (19.1)
	jātās tattvavido bālās tattvajñānena kiṃ tadā // (19.2)
	jaḍatvam apramāṇatvam athāvyākṛtatām api / (20.1)
	viparītaparijñānam indriyānāṃ tvam ūcivān // (20.2)
	ajñānenāvṛto yena yathāvan na prapadyate / (21.1)
	lokas tena yathābhūtam iti matvā tvayoditam // (21.2)
	astīti śāśvatī dṛṣṭir nāstīty ucchedadarśanam / (22.1)
	tenāntadvayanirmukto dharmo 'yaṃ deśitas tvayā // (22.2)
	catuṣkoṭivinirmuktās tena dharmās tvayoditāḥ / (23.1)
	vijñānasyāpy avijñeyā vācāṃ kimuta gocarāḥ // (23.2)
	svapnendrajālikodbhūtaṃ dvicandrodvīkṣaṇaṃ yathā / (24.1)
	bhūtaṃ tadvastu no bhūtaṃ tathā dṛṣṭaṃ jagat tvayā // (24.2)
	utpannaś ca sthito naṣṭaḥ svapne yadvat sutas tathā / (25.1)
	na cotpannaḥ sthito naṣṭa ukto loko 'rthatas tvayā // (25.2)
	kāraṇāt saṃbhavo dṛṣṭo yathā svapne tathetaraḥ / (26.1)
	saṃbhavaḥ sarvabhāvānāṃ vibhavo 'pi matas tathā // (26.2)
	rāgādijaṃ yathā duḥkhaṃ saṃkleśasaṃsṛtī tathā / (27.1)
	saṃbhārapūraṇān muktiḥ svapnavad bhāṣitā tvayā // (27.2)
	jātaṃ tathaiva no jātam āgataṃ gatam ity api / (28.1)
	baddho muktas tathā jñānī dvayam icchen na tattvavit // (28.2)
	utpattir yasya naivāsti tasya kā nirvṛtir bhavet / (29.1)
	māyāgajaprakāśatvād ādiśāntatvam arthataḥ // (29.2)
	utpanno 'pi na cotpanno yadvan māyāgajo mataḥ / (30.1)
	utpannaṃ ca tathā viśvam anutpannaṃ ca tattvataḥ // (30.2)
	ameyair aprameyānāṃ pratyekaṃ nirvṛtiḥ kṛtā / (31.1)
	lokanāthair hi sattvānāṃ na kaścin mocitaś ca taiḥ // (31.2)
	te ca sattvāś ca no jātā ye nirvānti na te sphuṭam / (32.1)
	na kaścin mocitaḥ kaiścid iti proktaṃ mahāmune // (32.2)
	māyākārakṛtaṃ yadvad vastu śūnyaṃ tathetarat / (33.1)
	vastuśūnyaṃ jagat sarvaṃ tvayoktaṃ kārakas tathā // (33.2)
	kārako 'pi kṛto 'nyena kṛtatvaṃ nātivartate / (34.1)
	athavā tatkriyākartṛkārakasya prasajyate // (34.2)
	nāmamātraṃ jagat sarvam ity uccair bhāṣitaṃ tvayā / (35.1)
	abhidhānāt pṛthagbhūtam abhidheyaṃ na vidyate // (35.2)
	kalpanāmātram ity asmāt sarvadharmāḥ prakāśitāḥ / (36.1)
	kalpanāpy asatī proktā yayā śūnyaṃ vikalpyate // (36.2)
	bhāvābhāvadvayātītam anatītaṃ ca kutracit / (37.1)
	na ca jñānaṃ na ca jñeyaṃ na cāsti na ca nāsti yat // (37.2)
	yan na caikaṃ na cānekaṃ nobhayaṃ na ca nobhayam / (38.1)
	anālayam athāvyaktam acintyam anidarśanam // (38.2)
	yan nodeti na ca vyeti nocchedi na ca śāśvatam / (39.1)
	tad ākāśapratīkāśaṃ nākṣarajñānagocaram // (39.2)
	yaḥ pratītyasamutpādaḥ śūnyatā saiva te matā / (40.1)
	tathāvidhaś ca saddharmas tatsamaś ca tathāgataḥ // (40.2)
	tat tattvaṃ paramārtho 'pi tathatā dravyam iṣyate / (41.1)
	bhūtaṃ tad avisaṃvādi tadbodhād buddha ucyate // (41.2)
	buddhānāṃ sattvadhātoś ca tenābhinnatvam arthataḥ / (42.1)
	ātmanaś ca pareṣāṃ ca samatā tena te matā // (42.2)
	bhāvebhyaḥ śūnyatā nānyā na ca bhāvo 'sti tāṃ vinā / (43.1)
	tasmāt pratītyajā bhāvās tvayā śūnyāḥ prakāśitāḥ // (43.2)
	hetupratyayasambhūtā paratantrā ca saṃvṛtiḥ / (44.1)
	paratantra iti proktaḥ paramārthas tv akṛtrimaḥ // (44.2)
	svabhāvaḥ prakṛtis tattvaṃ dravyaṃ vastu sad ity api / (45.1)
	nāsti vai bhāvo paratantras tu vidyate // (45.2)
	astīti kalpite bhāve samāropas tvayoditaḥ / (46.1)
	nāstīti kṛtakocchedād ucchedaś ca prakāśitaḥ // (46.2)
	tattvajñānena nocchedo na ca śāśvatatā matā / (47.1)
	vastuśūnyaṃ jagat sarvaṃ marīcipratimaṃ matam // (47.2)
	mṛgatṛṣṇājalaṃ yadvan nocchedi na ca śāśvatam / (48.1)
	tadvat sarvaṃ jagat proktaṃ nocchedi na ca śāśvatam // (48.2)
	dravyam utpadyate yasya tasyocchedādikaṃ bhavet / (49.1)
	antavān nāntavāṃś cāpi lokas tasya prasajyate // (49.2)
	jñāne sati yathā jñeyaṃ jñeye jñānaṃ tathā sati / (50.1)
	yatrobhayam anutpannam iti buddhaṃ tadāsti kim // (50.2)
	iti māyādidṛṣṭāntaiḥ sphuṭam uktvā bhiṣagvaraḥ / (51.1)
	deśayāmāsa saddharmaṃ sarvadṛṣṭicikitsakam // (51.2)
	etat tat paramaṃ tattvaṃ niḥsvabhāvārthadeśanā / (52.1)
	bhāvagrahagṛhītānāṃ cikitseyam anuttarā // (52.2)
	dharmayājñika tenaiva dharmayajño niruttaraḥ / (53.1)
	abhīkṣṇam iṣṭas trailokye niṣkapāṭo nirargalaḥ // (53.2)
	vastugrāhabhayocchedī kutīrthyamṛgabhīkaraḥ / (54.1)
	nairātmyasiṃhanādo 'yam adbhuto naditas tvayā // (54.2)
	śūnyatādharmagambhīrā dharmabherī parāhatā / (55.1)
	naiḥsvābhāvyamahānādo dharmaśaṅkhaḥ prapūritaḥ // (55.2)
	dharmayautukam ākhyātaṃ buddhānāṃ śāsanāmṛtam / (56.1)
	nītārtham iti nirdiṣṭaṃ dharmāṇāṃ śūnyataiva hi // (56.2)
	yā tūtpādanirodhādisattvajīvādideśanā / (57.1)
	neyārthā ca tvayā nātha bhāṣitā saṃvṛtiś ca sā // (57.2)
	prajñāpāramitāmbhodher yo 'tyantam pāram āgataḥ / (58.1)
	sa puṇyaguṇaratnāḍhyas tvadguṇārṇavapāragaḥ // (58.2)
	iti stutvā jagannātham acintyam anidarśanam / (59.1)
	yad avāptaṃ mayā puṇyaṃ tenāstu tvatsamaṃ jagat // (59.2)
	ity acintyastavaḥ samāptaḥ // (60.1)
