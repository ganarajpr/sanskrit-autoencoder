tathā hi dūrādrūpaṃ paśyatīti // (1.0)
	yatra rūpaṃ dṛśyate na tatra tadgrāhakaṃ cakṣurindriyam asti tatra pramāṇānupalabhyamānatvāt tatrāvidyamānadevadattādivat // (2.0)
	yathā cakṣuḥ evaṃ śrotramapi vaktavyam // (3.0)
	itara āha svaviṣayadeśaprāpi cakṣuḥśrotram indriyatvāt ghrāṇendriyādivat // (4.0)
	anenānumānena tatra pramāṇānupalabhyamānatvaṃ hetum asiddhaṃ darśayati // (5.0)
	ācārya āha sati ca prāptaviṣayatva iti // (6.0)
	vistaraḥ yadi prāptaviṣayaṃ cakṣuḥśrotraṃ kalpyeta divyaṃ cakṣuḥśrotramiha manuṣyeṣu dhyāyināṃ nopajāyeta // (7.0)
	yadi hi cakṣuḥśrotram ativiprakṛṣṭadeśasthaṃ vyavahitaṃ ca kuḍyādibhiryathāyogaṃ rūpaṃ śabdaṃ gṛhṇīyād evamasya divyatvaṃ sambhavet tacca prāptaviṣayatve na syāt // (8.0)
	prāṇādivat // (9.0)
	yathā ghrāṇajihvākāyāḥ prāptaviṣayatvād divyā dhyāyināṃ nopajāyeran tadvat // (10.0)
	anena svaviṣayadeśaprāpitvapakṣasya dharmiviśeṣaviparyayāpakṣālatvaṃ darśayati // (11.0)
	sambhavaddivyatve hi cakṣuḥśrotre dharmī // (12.0)
	asaṃbhavaddivyatvabhāvo viśeṣaviparyayaḥ // (13.0)
	sa prāpnotīti doṣaḥ // (14.0)
	anena doṣeṇānanumānatvāt tatpramāṇānupalabhyamānatvaṃ siddhaṃ vyavasthāpayati // (15.0)
	tasmādaprāptaviṣayaṃ cakṣuḥśrotram // (16.0)
	yadyaprāptaviṣayaṃ cakṣuriti vistaraḥ // (17.0)
	āsannenātidūrasthena tiraskṛtena vā tulyā tadaprāptiriti tatra darśanaṃ prasañjayati // (18.0)
	kimidaṃ parasya sādhanam uta dūṣaṇamiti yadi tāvadevaṃ sādhanam atidūraṃ tiraskṛtaṃ cakṣuḥśrotreṇa gṛhyate aprāptatvāt āsannaviṣayavat iti tadasādhanam hetoḥ svayamaniścitatvāt pūrvābhyupagamavirodhād vā // (19.0)
	atha dūṣaṇam sarvāprāptagrāhakatvaṃ cakṣuḥśrotralakṣaṇasya dharmiṇaḥ prasajyate tad adūṣaṇam anumānabādhanāt // (20.0)
	katham ityāha kathaṃ tāvad ayaskānto na sarvam aprāptam ayaḥ karṣatīti // (21.0)
	praśnamukhenāyaskāntanidarśanamupanyasya sarvāprāptagrāhakatvaṃ cakṣuḥśrotrasya sādhayati na sarvāprāptagrāhakaṃ cakṣuḥśrotram sarvāprāptagrahaṇaśaktihīnatvāt ayaskāntavat // (22.0)
	ayaskānto hyaprāptam ayo gṛhṇāti karṣatītyarthaḥ na ca sarvamaprāptaṃ gṛhṇāti tadvaccakṣuḥśrotram // (23.0)
	prāptaviṣayatve'pi caitat samānamiti // (24.0)
	nātidūratiraskṛto viṣayaścakṣuḥśrotreṇa gṛhyate grahaṇāyogyatvāt samprāptāñjanaśalākāvat // (25.0)
	athavā na sarvasvagrāhyagrāhi cakṣuḥśrotram indriyasvābhāvyāt ghrāṇendriyādivat // (26.0)
	ghrāṇādīnāṃ hi prāpto viṣayo na tu sarvaḥ ghrāṇādisahabhūni hi gandharasaspraṣṭavyāni ghrāṇādibhirna gṛhyante śaktir hīndriyāṇām īdṛśīti // (28.0)
	manastvarūpitvād iti // (29.0)
	prāptatvaṃ mūrtānāmeva vyavasthāpyeta nāmūrtānāmiti // (30.0)
	mano'prāptaviṣayamiti na vicāraḥ kriyate // (31.0)
