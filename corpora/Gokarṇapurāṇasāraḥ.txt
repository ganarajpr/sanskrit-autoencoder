śrīkṣetragokarṇapurāṇam / (1.1)
	gokarṇakṣetranilayaṃ dvibhujaṃ varadaṃ satām / (1.2)
	sarvābhīṣṭaprasiddhyarthaṃ taṃ namāmi gajānanam / (1.3)
	kurūṇām anvaye jāto rājā paramadhārmikaḥ / (1.4)
	śaunakādyair munivaraiḥ kṛtasatpraśnasatkathaḥ // (1.5)
	ṛṣibhiḥ sahitaś cāpi śatānīko mahāmatiḥ / (2.1)
	āste kṛtābhiṣeko 'tha saṃnyāsāya kṛtakṣaṇaḥ // (2.2)
	athājagāma dharmātmā vyāsaśiṣyo mahāmatiḥ / (3.1)
	tīrthayātrāṃ paricaran sūtaḥ paurāṇiko 'rthavit // (3.2)
	ugraśravāḥ śatānīkaṃ jayāśīrbhir avardhayat / (4.1)
	abhivādya munīn sarvān śaunakādīṃs tapodhanān // (4.2)
	papraccha kuśalaṃ sarvaṃ tapaḥsvādhyāyakarmasu / (5.1)
	tataḥ sa rājā dharmātmā yudhiṣṭhirakulodbhavaḥ // (5.2)
	sūtam āha mahāprājñam āsyatām iti sādaram / (6.1)
	āsanaṃ kalpayāmāsa śaunakasya tadā nṛpaḥ // (6.2)
	anujñāṃ prāpya sūtas tu munīnāṃ bhāvitātmanām / (7.1)
	upaviṣṭaḥ sabhāmadhye kṛtātithyo mahāmatiḥ // (7.2)
	viśrānto 'bhūc ciraṃ tasyāṃ sabhāyāṃ sūtanandanaḥ / (8.1)
	tato 'bravīd idaṃ vākyaṃ śaunakaḥ sūtam ādarāt // (8.2)
	śaunaka uvāca / (9.1)
	kuta āgamyate vidvann idānīṃ sūtanandana / (9.2)
	kva gato 'si bhavān pūrvaṃ kim uddiśya prayojanam // (9.3)
	sūta uvāca / (10.1)
	tīrthayātrāṃ gato brahman triḥ parītya dharām imām / (10.2)
	bhavantaṃ draṣṭumāyātaḥ śatānīkaṃ ca pārthivam // (10.3)
	pārikṣitaṃ ca rājarṣiṃ saṃnyāsāya kṛtakṣaṇam / (11.1)
	saha sthitvā dvijavarair gokarṇe suciraṃ dvija // (11.2)
	ihāgato 'smi viprarṣe kiṃ karomi praśādhi mām / (12.1)
	śaunaka uvāca / (12.2)
	tīrthānāṃ darśane puṇyaṃ snāne dāne ca yat phalam // (12.3)
	upavāse jape home tīrthe devārcane ca yat / (13.1)
	tīrthānām api cotpattiṃ puṇyaṃ puṇyaphalaṃ tathā // (13.2)
	tat sarvaṃ kathayāsmākaṃ paraṃ kautūhalaṃ hi naḥ / (14.1)
	ity ukte muninā tatra śatānīko 'pi pārthivaḥ // (14.2)
	papracchedaṃ sabhāmadhye sūtaṃ vai romaharṣaṇim / (15.1)
	śatānīka uvāca / (15.2)
	gokarṇād āgatam iti śrutaṃ te vadatāṃ vara // (15.3)
	kasmin deśe sa gokarṇaṃ tīrtharājaḥ praśasyate / (16.1)
	kim artham uṣitaṃ tatra kair vā viprarṣibhiḥ saha // (16.2)
	brūhy aśeṣeṇa naḥ sūta yasmād adyāgato hy asi / (17.1)
	sūta uvāca / (17.2)
	śrūyatāṃ kuruśārdūla sahaibhir munipuṅgavaiḥ // (17.3)
	vyāsaprasādād vakṣyāmi sarvaṃ pratyakṣadarśivat / (18.1)
	purā provāca kailāse ṣaṇmukhāya maheśvaraḥ // (18.2)
	gokarṇakṣetramāhātmyaṃ nārado 'gnisutāya ca / (19.1)
	ahaṃ tad varṇayiṣyāmi śṛṇu rājan mahāmate // (19.2)
	siddhikṣetrāṇi trīṇy āhūrahasyāni sureṣv api / (20.1)
	śālagrāmaṃ puṣkaraṃ ca gokarṇaṃ ca narādhipa // (20.2)
	śālagrāmāt puṣkarāc ca gokarṇaṃ śīghrasiddhidam / (21.1)
	yad yad icchati gokarṇe tat tad āpnoti niścayāt // (21.2)
	ekārṇave jagaty asmin sṛṣṭikāla upasthite / (22.1)
	brahmā caturmukhaḥ pūrvaṃ sṛṣṭyarthaṃ tapa āsthitaḥ // (22.2)
	tadā tasya bhruvor madhyād rudraḥ prādurabhūt prabhuḥ / (23.1)
	brahmā provāca taṃ dṛṣṭvā sṛṣṭiṃ kuru mahāmate // (23.2)
	tathety uktvā tadā rudro jarāmaraṇavarjitāḥ / (24.1)
	sattvaikaguṇasampannāḥ samaprakṛtayaḥ prajāḥ // (24.2)
	srakṣyāmīty eva niścitya pātāle tapasi sthitaḥ / (25.1)
	evaṃ tapasyatas tasya vyatīyāya yugatrayam // (25.2)
	tato vilambitaṃ dṛṣṭvā rudrasargaṃ prajāpatiḥ / (26.1)
	tatra kāraṇam anvicchan yathāpūrvaṃ sasarja ha // (26.2)
	śrutvedaṃ prakṛter vākyāt tayā rudraḥ sahotthitaḥ / (27.1)
	dadṛśe medinīṃ viṣvagāvṛtāṃ cakravatsthitām // (27.2)
	kruddhaḥ provāca tāṃ mārgam udgantuṃ mama darśaya / (28.1)
	bhūr uvāca / (28.2)
	viśvātman praṇatāṃ trāhi strībhāvād api vihvalām // (28.3)
	nirāgasaṃ bhedayituṃ nocitaṃ te kṛpānidhe / (29.1)
	karṇacchidreṇa me deva hrasvo bhūtvordhvam utpata // (29.2)
	tatheti tāṃ samāśvāsyāṅguṣṭhamātravapurdharaḥ ūrdhvam etyāha giriśo bhadre matsaṅgamā tvayi / (30.1)
	bhavitā tanayaś caṇḍavīryo 'ṅgārakanāmakaḥ // (30.2)
	gośabdenākhyāyase tvaṃ karṇasthānam idaṃ hy ataḥ / (31.1)
	gokarṇam iti nāmnā tat trailokye parigīyate // (31.2)
	pātālād ūrdhvagamane yonikalpam abhūd yataḥ / (32.1)
	rudrayonir iti khyātaṃ puṇyakṣetreṣu gaṇyate // (32.2)
	kalpānte 'py āvartarūpaṃ varuṇālayamadhyabhāk / (33.1)
	na vinaśyati tasmāc ca varuṇāvartasaṃjñakam // (33.2)
	ye smariṣyanti gokarṇaṃ prātar utthāya nityaśaḥ / (34.1)
	teṣāṃ rātrikṛtaṃ pāpaṃ tatkṣaṇād eva naśyati // (34.2)
	evaṃ divā kṛtaṃ pāpam aparāhṇe praṇaśyati / (35.1)
	pañcakrośaparīmāṇaṃ kṣetrāṇām uttamottamam // (35.2)
	ye paśyanti mahākṣetraṃ gokarṇaṃ surapūjitam / (36.1)
	janmaprabhṛti yat teṣāṃ kṛtaṃ pāpaṃ vinaśyati // (36.2)
	snānaṃ kurvanti ye tatra gokarṇe tīrthasattame / (37.1)
	teṣāṃ trijanmācaritaṃ pāpaṃ naśyaty asaṃśayaḥ // (37.2)
	yathā kailāsaśikhare yathā mandaramūrdhani / (38.1)
	nivasāmi sadā devyā tathā gokarṇamaṇḍale // (38.2)
	śrīśailakāśyor api rāmasetau gokarṇamukhyeṣv api puṇyabhūmiṣu / (39.1)
	ārabhya tasmād api mukhyadeśāt sad yojanair dvādaśabhiś ca maṇḍalam // (39.2)
	ity uktvā dharaṇīṃ rudro dṛṣṭvā sṛṣṭiṃ svayaṃbhuvaḥ / (40.1)
	kruddhaḥ san tāṃ saṃjihīrṣuḥ kiṃcid īśānabhāgagaḥ // (40.2)
	pramathān sasṛje tatra tāvad viṣṇur upāgataḥ / (41.1)
	uvāca rudraṃ bho deva kṣamyatāṃ brahmaṇā kṛtam // (41.2)
	ante tvam eva saṃhartā krodham adyopasaṃhara / (42.1)
	tataḥ prasanno bhagavān prāhānugrāhakaṃ vacaḥ // (42.2)
	didhakṣor mama saṃsthityā bhūyād īśānabhūr iyam / (43.1)
	rudrabhūmir iti khyātā mokṣadā dahanān nṛṇām // (43.2)
	prakṛtyā pārṣadaiḥ sākam atra viṣṇo vasāmy aham / (44.1)
	ity ukte śambhunā viṣṇus tathāstv iti yayau divam // (44.2)
	sūta uvāca / (45.1)
	gokarṇe sarvadā vāsaṃ maraṇam muktimaṇḍape / (45.2)
	rudrabhūmyāṃ tu dahanaṃ kāṅkṣante vibudhā api // (45.3)
	tataḥ rudraś cintayitvā bhaviṣyat kāryaṃ gauravam / (46.1)
	svapratijñā vṛthā mā bhūd iti brahmādiṣu svayam // (46.2)
	sthāvarānteṣu yat sattvaṃ tad ākṛṣya maheśvaraḥ / (47.1)
	tena sattvena sauvarṇaṃ nirmame mṛgam uttamam // (47.2)
	svasya viṣṇor brahmaṇaś ca sattvāc chṛṅgatrayaṃ tathā / (48.1)
	aśeṣaprāṇināṃ sattvāc catuṣpādaṃ trinetrakam // (48.2)
	tanmṛgeṇa hy agāt svāmī saha kailāsamandiram / (49.1)
	tataḥ prajāḥ prajāpālāḥ sarve sattvavivarjitāḥ // (49.2)
	surāsuramanuṣyāś ca brahmāṇaṃ śaraṇaṃ yayuḥ / (50.1)
	tataḥ svayaṃbhuvā sārdhaṃ surā viṣṇum upāgaman // (50.2)
	uvāca viṣṇus tān devān sattvaṃ rudreṇa vai hṛtam / (51.1)
	taṃ gatvā vibudhaśreṣṭhā yāciṣyāmaḥ kṛpānidhim // (51.2)
	iti niścitya te sarve kailāsam agaman surāḥ / (52.1)
	pūjayitvā mahādevaṃ prārthayāmāsur īhitam // (52.2)
	tatas tuṣṭaḥ śivaḥ prādāt tebhyaḥ sattvamayaṃ mṛgam / (53.1)
	prītāḥ surās tad vibhajya jagṛhuḥ sattvam ātmanaḥ // (53.2)
	brahmā viṣṇuś ca rudraś ca svīyaṃ śṛṅgaṃ samādade / (54.1)
	tacchṛṅgaṃ pūjayann āste rudro bhaktyānukampayā // (54.2)
	etad eva prāṇaliṅgaṃ cātmaliṅgam itīryate / (55.1)
	ya etat pūjayed bhaktyā sa prāpnotīpsitaṃ dhruvam // (55.2)
	śṛṇudhvam asya māhātmyaṃ sarvalokaikaviśrutam / (56.1)
	trailokyasyādhipatyāya devadānavayoḥ purā // (56.2)
	yuddho babhūva tumulo devās tatra parājitāḥ / (57.1)
	dānavair hṛtasarvasvā īśvaraṃ śaraṇaṃ yayuḥ // (57.2)
	tadādeśāl liṅgam etad ārādhya punar āhave / (58.1)
	nirjitya dānavān devā alabhanta manorathān // (58.2)
	evamādikathāḥ śrutvā rāvaṇo 'sya jighṛkṣayā / (59.1)
	kailāsaparvataṃ gatvā tapa ugraṃ samācarat // (59.2)
	atrāntare nāradas tu gataḥ kailāsaparvatam / (60.1)
	tatra rāvaṇam anvīkṣya svargam etyāha vajriṇam // (60.2)
	kailāse rāvaṇas tīvraṃ tapaś carati duścaram / (61.1)
	mṛgaśṛṅgaṃ netukāmaḥ purā yuṣmābhir arcitam // (61.2)
	nīte liṅge rāvaṇena niḥśrīkāḥ syur dhruvaṃ surāḥ / (62.1)
	tasmād vighnaṃ kuru kṣipram ity uktvāgād yathāgatam // (62.2)
	tataḥ saṃmantrya guruṇā sākaṃ śakro 'maraiḥ saha / (63.1)
	paulastyatapase vighnaṃ kartuṃ kailāsam abhyagāt // (63.2)
	atrāntare rāvaṇo 'pi toṣayitvā maheśvaram / (64.1)
	pañcabrahmātmakam idaṃ na nidhehi bhuvi kvacit // (64.2)
	iti tenāpy anujñāto liṅgaṃ sampādya sādaram / (65.1)
	svapuraṃ pādacāreṇa niryayau dakṣiṇāmukhaḥ // (65.2)
	athātra devāḥ kailāsagirim etya maheśvaram / (66.1)
	papracchur abhivādyeśaṃ rāvaṇasya viceṣṭitam // (66.2)
	sa tān uvāca bho devā liṅgaṃ sampādya rāvaṇaḥ / (67.1)
	niragāt pādacāreṇa laṅkāṃ dakṣiṇadiṅmukhaḥ // (67.2)
	gānena tapasā lubdho liṅgaṃ tasmai dadāv aham / (68.1)
	parantv ayam apātro 'tiduṣṭātmā iti niścayaḥ // (68.2)
	atas tasya karāl liṅgaṃ galitaṃ sad yathā bhuvi / (69.1)
	pratiṣṭhitaṃ bhavec chīghraṃ prayatadhvaṃ tathā surāḥ // (69.2)
	ity avāpya śivasyājñām anviṣyanto daśānanam / (70.1)
	saptakoṭīśvaraṃ jagmur devā nāradabodhitāḥ // (70.2)
	daśānano 'pi madhyāhne tatra sthitvāhnikaṃ caran / (71.1)
	pūjāṃ nirvartayāmāsa saptakoṭīśvarasya ca // (71.2)
	tatra devān nirīkṣyaiva śaṅkitaḥ krodhamūrchitaḥ / (72.1)
	gokarṇābhimukhaḥ prāgād rāvaṇo meghagarjanaḥ // (72.2)
	tam anvayātāḥ śakrādyā liṅgasya sthāpane tadā / (73.1)
	viṣṇuṃ samprārthayāmāsur upāyaś cintyatām iti // (73.2)
	tato viṣṇuḥ kāryam etad upadiśya gajānanam / (74.1)
	prāhiṇod rāvaṇam anu svayaṃ cāgāt sahāyakṛt // (74.2)
	pitṛsthālīsamīpe tu yāvad gacchati rāvaṇaḥ / (75.1)
	tāvac ca viṣṇur bhagavān alaukikyātmamāyayā // (75.2)
	cakreṇa sūryam ācchādya sandhyām evāvabhāsayat / (76.1)
	astaṃ gato 'rka ity evaṃ manyamāno daśānanaḥ // (76.2)
	kathaṃ sandhyām upāsiṣye dhṛtvā liṅgaṃ kare 'dhunā / (77.1)
	iti cintākulas tāvad gajāsyaṃ baṭurūpiṇam // (77.2)
	dṛṣṭvā tasya kare liṅgaṃ ditsur evam uvāca tam / (78.1)
	muniputra gṛhāṇedaṃ liṅgaratnaṃ muhūrtakam // (78.2)
	nadyām asyāṃ vidhiṃ sāndhyaṃ samāpyādāsya āśv aham / (79.1)
	pañcabrahmātmakaṃ caitan na nidhehi mahītale // (79.2)
	evam ukto rāvaṇena gajāsyaḥ pratyuvāca tam / (80.1)
	aho gurutamaṃ hy etatkāryārthī vaṭur apy aham // (80.2)
	avaśyam eva vāhyaṃ cen na saheya ciraṃ kvacit / (81.1)
	tad ahaṃ tvāṃ trir āhūya nidhāsye tvayy anāgate // (81.2)
	ity uktvā jagṛhe liṅgaṃ sandhyāṃ dadhyau ca rāvaṇaḥ / (82.1)
	tāvad devāḥ prāṇaliṅge trilokībhāram ādadhuḥ // (82.2)
	tadā voḍhum aśaktas trir uccair udghuṣya rāvaṇam / (83.1)
	gajāsyaḥ sthāpayāmāsa liṅgaṃ śleṣmātake vane // (83.2)
	vaivasvate 'ntare rājan trayoviṃśatime śubhe / (84.1)
	tretāyuge tv īśvarābde śaratkāle nṛpottama // (84.2)
	ūrjaśuklapratipadi vyatīpāte raverdine / (85.1)
	tulāmāse mīnalagne viśākhāyāṃ pratiṣṭhitam // (85.2)
	tato viṣṇuḥ svakaṃ cakram upasaṃhṛtavān raveḥ / (86.1)
	atha devagaṇā liṅge puṣpāṇi vavṛṣur mudā // (86.2)
	śaśaṃsuśca stavair divyaiḥ śivaṃ viṣṇuṃ gajānanam / (87.1)
	vīṇāveṇumṛdaṅgādyair nanṛtuś cāpsarogaṇāḥ // (87.2)
	iti śrīskānde gokarṇakhaṇḍe śrīgokarṇamāhātmye sāroddhāre prathamo 'dhyāyaḥ / // (88.1)
