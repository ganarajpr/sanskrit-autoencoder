janmādyasya yato 'nvayāditarataś cārtheṣv abhijñaḥ svarāṭ / (1.1)
	tene brahma hṛdā ya ādikavaye muhyanti yat sūrayaḥ / (1.2)
	tejovārimṛdāṃ yathā vinimayo yatra trisargo 'mṛṣā / (1.3)
	dhāmnā svena sadā nirastakuhakaṃ satyaṃ paraṃ dhīmahi // (1.4)
	dharmaḥ projjhitakaitavo 'tra paramo nirmatsarāṇāṃ satām / (2.1)
	vedyaṃ vāstavam atra vastu śivadaṃ tāpatrayonmūlanam / (2.2)
	śrīmadbhāgavate mahāmunikṛte kiṃ vā parair īśvaraḥ / (2.3)
	sadyo hṛdy avarudhyate 'tra kṛtibhiḥ śuśrūṣubhis tatkṣaṇāt // (2.4)
	nigamakalpatarorgalitaṃ phalam / (3.1)
	śukamukhād amṛtadravasaṃyutam / (3.2)
	pibata bhāgavataṃ rasam ālayam / (3.3)
	muhuraho rasikā bhuvi bhāvukāḥ // (3.4)
	naimiṣe 'nimiṣakṣetre īśayaḥ śaunakādayaḥ / (4.1)
	sattraṃ svargāya lokāya sahasrasamam āsata // (4.2)
	ta ekadā tu munayaḥ prātarhutahutāgnayaḥ / (5.1)
	satkṛtaṃ sūtam āsīnaṃ papracchur idam ādarāt // (5.2)
	ṛṣaya ūcuḥ / (6.1)
	tvayā khalu purāṇāni setihāsāni cānagha / (6.2)
	ākhyātāny apy adhītāni dharmaśāstrāṇi yāny uta // (6.3)
	yāni vedavidāṃ śreṣṭho bhagavān bādarāyaṇaḥ / (7.1)
	anye ca munayaḥ sūta parāvaravido viduḥ // (7.2)
	vettha tvaṃ saumya tat sarvaṃ tattvatas tadanugrahāt / (8.1)
	brūyuḥ snigdhasya śiṣyasya guravo guhyam apyuta // (8.2)
	tatra tatrāñjasāyuṣman bhavatā yadviniścitam / (9.1)
	puṃsām ekāntataḥ śreyastan naḥ śaṃsitum arhasi // (9.2)
	prāyeṇālpāyuṣaḥ sabhya kalāv asmin yuge janāḥ / (10.1)
	mandāḥ sumandamatayo mandabhāgyā hy upadrutāḥ // (10.2)
	bhūrīṇi bhūrikarmāṇi śrotavyāni vibhāgaśaḥ / (11.1)
	ataḥ sādho 'tra yat sāraṃ samuddhṛtya manīṣayā / (11.2)
	brūhi bhadrāya bhūtānāṃ yenātmā suprasīdati // (11.3)
	sūta jānāsi bhadraṃ te bhagavān sātvatāṃ patiḥ / (12.1)
	devakyāṃ vasudevasya jāto yasya cikīrṣayā // (12.2)
	tan naḥ śuśrūṣamāṇānām arhasy aṅgānuvarṇitum / (13.1)
	yasyāvatāro bhūtānāṃ kṣemāya ca bhavāya ca // (13.2)
	āpannaḥ saṃsṛtiṃ ghorāṃ yan nāma vivaśo gṛṇan / (14.1)
	tataḥ sadyo vimucyeta yadbibheti svayaṃ bhayam // (14.2)
	yatpādasaṃśrayāḥ sūta munayaḥ praśamāyanāḥ / (15.1)
	sadyaḥ punanty upaspṛṣṭāḥ svardhunyāpo 'nusevayā // (15.2)
	ko vā bhagavatastasya puṇyaślokeḍyakarmaṇaḥ / (16.1)
	śuddhikāmo na śṛṇuyādyaśaḥ kalimalāpaham // (16.2)
	tasya karmāṇyudārāṇi parigītāni sūribhiḥ / (17.1)
	brūhi naḥ śraddadhānānāṃ līlayā dadhataḥ kalāḥ // (17.2)
	athākhyāhi harerdhīmann avatārakathāḥ śubhāḥ / (18.1)
	īlā vidadhataḥ svairam īśvarasyātmamāyayā // (18.2)
	vayaṃ tu na vitṛpyāma uttamaślokavikrame / (19.1)
	yacchṛṇvatāṃ rasajñānāṃ svādu svādu pade pade // (19.2)
	kṛtavān kila karmāṇi saha rāmeṇa keśavaḥ / (20.1)
	atimartyāni bhagavān gūḍhaḥ kapaṭamānuṣaḥ // (20.2)
	kalim āgatam ājñāya kṣetre 'smin vaiṣṇave vayam / (21.1)
	āsīnā dīrghasattreṇa kathāyāṃ sakṣaṇā hareḥ // (21.2)
	tvaṃ naḥ saṃdarśito dhātrā dustaraṃ nistitīrṣatām / (22.1)
	kaliṃ sattvaharaṃ puṃsāṃ karṇadhāra ivārṇavam // (22.2)
	brūhi yogeśvare kṛṣṇe brahmaṇye dharmavarmaṇi / (23.1)
	svāṃ kāṣṭhām adhunopete dharmaḥ kaṃ śaraṇaṃ gataḥ // (23.2)
