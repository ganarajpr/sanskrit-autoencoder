	nāḍī dhatte marutkope jalaukāsarpayorgatim / (2.1)
	kuliṅgakākamaṇḍūkagatiṃ pittasya kopataḥ // (2.2)
	haṃsapārāvatagatiṃ dhatte śleṣmaprakopataḥ / (3.1)
	lāvatittiravartīnāṃ gamanaṃ saṃnipātataḥ // (3.2)
	kadācin mandagamanā kadācidvegavāhinī / (4.1)
	dvidoṣakopato jñeyā hanti ca sthānavicyutā // (4.2)
	sthitvā sthitvā calati yā sā smṛtā prāṇanāśinī / (5.1)
	atikṣīṇā ca śītā ca jīvitaṃ hantyasaṃśayam // (5.2)
	jvarakopeṇa dhamanī soṣṇā vegavatī bhavet / (6.1)
	kāmakrodhādvegavahā kṣīṇā cintābhayaplutā // (6.2)
	mandāgneḥ kṣīṇadhātośca nāḍī mandatarā bhavet / (7.1)

	laghvī vahati dīptāgnestathā vegavatī matā / (8.1)
	sukhitasya sthirā jñeyā tathā balavatī smṛtā // (8.2)
	capalā kṣudhitasyāpi tṛptasya vahati sthirā / (9.1)
