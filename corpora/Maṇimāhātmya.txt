kailāsaśikharāsīnaṃ devadevaṃ jagatpatim / (1.1)
	papraccha pārvatī devī tattvaṃ paramadurlabham // (1.2)
	pārvaty uvāca / (2.1)
	maṇīnāṃ lakṣaṇaṃ deva kathayasva prasādataḥ / (2.2)
	yena siddhiṃ labhante 'tra sādhakā gatakalmaṣāḥ // (2.3)
	mahādeva mahāghora kurvanti ripumardanam / (3.1)
	kavitvaṃ dīrghajīvitvaṃ kurute 'tra yathā prabho // (3.2)
	aṣṭau guṇāḥ phalaṃ yatra tvatprasādān maheśvara / (4.1)
	jñānamārgaṃ ca mokṣam ca śūlarogaṃ ca dāruṇam // (4.2)
	cakṣūrogaṃ śirorogaṃ viṣopadravam eva ca / (5.1)
	sphuṭaṃ vada yathā prabho prasādān me maheśvara // (5.2)
	uvāca śaṃkaro devi yat tvayā paripṛcchyate / (6.1)
	yan na kasyacid ākhyātaṃ tad vadāmi varānane // (6.2)
	purāhaṃ viṣṇunā yukto brahmaṇā saha sundari / (7.1)
	śuklatīrthe gato devi revātīre suśobhane // (7.2)
	ratnaparvatanāmā ca tatra tiṣṭhati bhūdharaḥ / (8.1)
	indreṇa sthāpito devi sarvadevasukhaṃkaraḥ // (8.2)
	tasya darśanamātreṇa sarvapāpaiḥ pramucyate / (9.1)
	rogī rogavinirmukto jāyate nātra saṃśayaḥ // (9.2)
	devyā āyatane ye tu citāṃ dahanti mānavāḥ / (10.1)
	te yānti paramaṃ sthānaṃ śivadarśanasaṃyutam // (10.2)
	aṣṭamyāṃ snāti yaḥ kuṇḍe pūjayitvā tataḥ śivam / (11.1)
	sarvapāpavinirmukto mama lokaṃ sameti saḥ // (11.2)
	itthaṃ devagaṇāḥ sarve kuṇḍe snātvā kṣaṇaṃ sthitāḥ / (12.1)
	pavitradehāḥ śrīmantaḥ punaḥ kedāram āgatāḥ // (12.2)
	garutmatsthāpitaṃ liṅgaṃ sarvapāpavimocakam / (13.1)
	tasya darśanamātraṃ hi brahmahatyāṃ vyapohati // (13.2)
	aṣṭamyāṃ ca caturdaśyāṃ pūrṇamāsyāṃ viśeṣataḥ / (14.1)
	yaḥ pūjayati puṇyātmā mama lokaṃ sa gacchati // (14.2)
	kedāraṃ pūjayed yas tu puṇyātmā bhāgyabhājanam / (15.1)
	sarvārthasiddhisampannaṃ prāpnoti paramaṃ padam // (15.2)
	indreṇa sthāpitaṃ vajraṃ kośaś ca dhanadena tu / (16.1)
	mayāpi sthāpitā mantrāḥ kathitaṃ te varānane // (16.2)
	garutmataḥ samudgārān maṇikalā mahānadī / (17.1)
	viniḥsṛtā mahātejāḥ sarvapāpapraṇāśinī // (17.2)
	tasyāḥ prabhāvato devi maṇayaḥ śubhalakṣaṇāḥ / (18.1)
	bhogadā mokṣadāś caiva rogadoṣavighātakāḥ // (18.2)
	śrīdevy uvāca / (19.1)
	maṇīnāṃ lakṣaṇaṃ brūhi yathāsti vṛṣabhadhvaja / (19.2)
	kenopāyena te grāhyā devapūjā kathaṃ vibho // (19.3)
	kīdṛśaṃ ca vrataṃ kāryaṃ kiṃ dānaṃ kasya pūjanam / (20.1)
	kā ca bhaktikriyā kāryā deva me vada bhairava // (20.2)
	śrībhairava uvāca / (21.1)
	kedārabhavanaṃ gatvā kalaśānāṃ śatāṣṭakam / (21.2)
	śrīmatkedāranāthāya manasā kṛtabhāvanaḥ // (21.3)
	kṣetrapālaṃ yathāśaktyā upahārair anuttamaiḥ / (22.1)
	pūjayitvā prayatnena sādhakaḥ phalakāṅkṣayā // (22.2)
	evaṃ pūjya mahābhaktyā praṇamya ca punaḥ punaḥ / (23.1)
	baliṃ dadyād vidhānena dikṣu sarvāsu yatnataḥ // (23.2)
	śivasthāne tu kartavyo japaḥ surasamarcite / (24.1)
	tato gatvā mahānadyāṃ maṇiratnāni vīkṣate // (24.2)
	mantrasaṃnaddhakāyaś ca gojihvālepabhūṣitaḥ / (25.1)
	atha teṣāṃ maṇīnāṃ tu kartavyaṃ suparīkṣaṇam // (25.2)
	gopitaṃ yan mayā pūrvaṃ tan me nigadataḥ śṛṇu / (26.1)
	sutaptahemavarṇābho nīlarekhāsamanvitaḥ // (26.2)
	śvetarekhādharo nityaṃ pītarekhāsamāyutaḥ / (27.1)
	āraktarekhāsaṃyuktaḥ kṛṣṇarekhāvibhūṣitaḥ // (27.2)
	etaiś cihnaiḥ samāyukto nīlakaṇṭha iti smṛtaḥ / (28.1)
	dadāti vipulān bhogān jñānamārgaṃ sudurlabham // (28.2)
	kavitvaṃ dīrghajīvitvaṃ kurute nātra saṃśayaḥ / (29.1)
	tārābho hemavarṇābhaḥ caturbinduvibhūṣitaḥ // (29.2)
	kṛṣṇabindudharo yas tu biḍālasamarocanaḥ / (30.1)
	sa bhaveddhanalābhāya nātra kāryā vicāraṇā // (30.2)
	raktapādapavarṇābha indranīlasamadyutiḥ / (31.1)
	śvetarekhāsamāyukto hy arthakārye mahādyutiḥ // (31.2)
	sa viṣṇur iti vikhyātaḥ sarvaiśvaryaphalapradaḥ / (32.1)
	śuddhasphaṭikasaṃkāśo nīlarekhāvibhūṣitaḥ // (32.2)
	kṛṣṇabindudharaḥ śuklaḥ sa maṇiḥ sarvakāmadaḥ / (33.1)
	pītaś ca śvetarekhaś ca maṇiḥ svacchaś ca dṛśyate / (33.2)
	guṇānām ākaraḥ so hi bahurogān nihanti ca // (33.3)
	yaḥ pārāvatakaṇṭhābhaḥ samprāpto bindubhiḥ sitaiḥ / (34.1)
	āstīkasya kulotpannaḥ sa maṇir viṣadarpahā // (34.2)
	sāraṅgākṣisamo mahādyutidharo mattebhanetrākṛtiḥ / (35.1)
	śvetair bindubhir anvito varatanur bhāsvān maṇir bindukaḥ / (35.2)
	tatprakṣālitavāripānavidhinā naśyed viṣaṃ dāruṇaṃ / (35.3)
	yat sarvaṃ vinatāsuto bahuvidhaṃ hanyād viṣaṃ dāruṇam / (35.4)
	saṃgrāme jayate ripūn bahuvidhān bhogān maṇir yacchati // (35.5)
	kiṃcin nīlapadas tato 'ruṇaruciḥ kiṃcicca vidyutprabhaḥ / (36.1)
	kiṃcil locanasuprabho bahuvidharekhāyuto vartulaḥ / (36.2)
	vikhyātaḥ sa mahāmaṇir viṣaharo baddho narāṇāṃ kare / (36.3)
	bhūtaṃ nāśayatīha somasadṛśas tasmāt pṛthivyāṃ priyaḥ // (36.4)
	nānāratnasamadyutir bahuvidhai rekhāgaṇair aṅkitaḥ / (37.1)
	śuddho bindugaṇair yutaḥ savimalo nāgendradarpāpahaḥ / (37.2)
	satyaṃ kāñcanavittalābhakaraṇe sṛṣṭo mayāsau maṇiḥ / (37.3)
	prakhyātaś ca sa siddhajanmajananaiḥ puṇyaiḥ satāṃ gocaraḥ // (37.4)
	nīlavarṇo bhaved yas tu bindupañcakabhūṣitaḥ / (38.1)
	viśuddhāṅgo 'ruṇo vṛttaḥ prasiddho vinatāsutaḥ // (38.2)
	sindūravarṇasaṃkāśo yasyāṅge rekhā kāśitā / (39.1)
	kṛṣṇavarṇas tu vijñeyo niḥśeṣaviṣamardanaḥ // (39.2)
	kāṃsyavarṇo bhaved yas tu nānārekhāsamākulaḥ / (40.1)
	nānābindusamākīrṇo jvaratāpaṃ vyapohati // (40.2)
	pītavarṇo bhaved yas tu dvirekhaḥ sitabindukaḥ / (41.1)
	sujīrṇavṛścikasyāpi viṣaṃ hanti sudāruṇam // (41.2)
	śvetā pītā samā rekhā indranīlasamadyutiḥ / (42.1)
	netrarogaṃ ca śūlaṃ ca jalapānād vyapohati // (42.2)
	haridvarṇo bhaved yas tu śvetarekhāvibhūṣitaḥ / (43.1)
	pītarekhāsamāyukto niḥśeṣagaralāpahaḥ // (43.2)
	pītagodhūmavarṇo yo gajanetrākṛtiḥ punaḥ / (44.1)
	śvetabindudharo nityaṃ bhūtasyājīrṇanāśakaḥ // (44.2)
	raktāṅgaḥ śuddharekhaś ca ardhāṅge kṛṣṇa eva ca / (45.1)
	sa maṇī raktaśūlaṃ ca netraśūlaṃ vyapohati // (45.2)
	śuddhasphaṭikasaṃkāśaḥ kiṃciccāraktapītakaḥ / (46.1)
	vṛścikānāṃ viṣaṃ hanti sa maṇiḥ sarvakāmadaḥ // (46.2)
	ratkam ardhaṃ ca kṛṣṇārdhaṃ śvetaṃ kiṃcid bhaved yadi / (47.1)
	evaṃrūpo bhaved yas tu sarpādiviṣanāśanaḥ // (47.2)
	raktāṅgo raktarekhaś ca āvartaiḥ śobhanair yutaḥ / (48.1)
	sa maṇir garuḍo jñeyaḥ sarpādiviṣanāśanaḥ // (48.2)
	pītāṅgaḥ kṛṣṇarekhaś ca nānābindusamākulaḥ / (49.1)
	evaṃrūpo bhaved yas tu mahāsarpaviṣāpahaḥ // (49.2)
	pītāṅgaḥ pītarekhaś ca raktarekhāvibhūṣitaḥ / (50.1)
	sarvavyādhiharaḥ śvetaḥ kathitas tu varānane // (50.2)
	kūṣmāṇḍīpuṣpasaṃkāśo nānārūpas tu bindubhiḥ / (51.1)
	sarvavyādhiharo jñeyaḥ samastaviṣamardanaḥ // (51.2)
	raktavarṇā bhavantīha nānābindusamākulāḥ / (52.1)
	tejasvino 'tirūpāś ca sarve te viṣamardakāḥ // (52.2)
	bindunābho mahākāntiḥ kṛṣṇabinduvibhūṣitaḥ / (53.1)
	sarvarogavināśo 'yam kathitas te varānane // (53.2)
	māñjiṣṭhapītavarṇābhas tāmrabindusamanvitaḥ / (54.1)
	sarvavyādhiharo nityaṃ bhūtajvaravināśanaḥ // (54.2)
	dāḍimīpuṣpasaṃkāśaḥ kṛṣṇabinduvibhūṣitaḥ / (55.1)
	saubhāgyajananaḥ śrīmān bhramarekhāyutaḥ priye // (55.2)
	kundapuṣpapratīkāśas tulyatve vartulaḥ priye / (56.1)
	evaṃrūpeṇa saṃyuktaḥ samastaviṣamardakaḥ // (56.2)
	gajanetrākṛtir yas tu biḍālākṣisamaprabhaḥ / (57.1)
	tārkṣyatulyamahātejāḥ pūjanīyo yathārcitaḥ // (57.2)
	tīrthakaraḥ sutejāśca dyutimān iti dṛśyate / (58.1)
	samastaviṣaho jñeyaḥ sa maṇir jīyate dhruvam // (58.2)
