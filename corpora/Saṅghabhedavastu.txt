buddho bhagavān kapilavastuni viharati nyagrodhārāme // (1.1)
	atha sambahulānāṃ kāpilavāstavānāṃ śākyānāṃ saṃsthāgāre saṃniṣaṇṇānāṃ saṃnipatitānām ayam evaṃrūpo 'bhūd antarākathāsamudāhāraḥ // (2.1)
	kuto nirjātā bhavantaḥ śākyāḥ kimagraṇyāḥ kimanvayāḥ kaś ca śākyānāṃ paurāṇakulavaṃśaś ca // (3.1)
	sacetkaścid asmākam upasaṃkramyaivaṃ pṛcchet kuto nirjātā bhavantaḥ śākyāḥ kimagraṇyāḥ kimanvayāḥ kaś ca teṣām paurāṇaḥ kulavaṃśa iti // (4.1)
	evaṃ pṛṣṭā vayaṃ kiṃ vyākuryāmaḥ // (5.1)
	na ca punar jānīmaḥ kuto nirjātāḥ śākyāḥ kimagraṇyāḥ kimanvayāḥ kaśca teṣāṃ paurāṇaḥ kulavaṃśa iti // (6.1)
	ete vayaṃ yena bhagavāṃstenopasaṃkrāmāmaḥ // (7.1)
	upasaṃkramya bhagavantam etam evārthaṃ paripṛcchāmaḥ yathā cāsmākaṃ bhagavān vyākaroti tathainaṃ dhārayiṣyāmaḥ iti // (8.1)
	atha sambahulāḥ kāpilavāstavāḥ śākyā yena bhagavāṃs tenopasaṃkrāntāḥ // (9.1)
	upasaṃkramya bhagavataḥ pādau śirasā vanditvā ekānte niṣaṇṇāḥ // (10.1)
	ekānte niṣaṇṇāḥ sambahulāḥ kāpilavāstavāḥ śākyā bhagavantam idam avocan // (11.1)
	ihāsmākam bhadanta sambahulānāṃ kāpilavāstavānāṃ śākyānāṃ saṃsthāgāre saṃniṣaṇṇānāṃ saṃnipatitānām ayam evaṃrūpo 'bhūd antarākathāsamudāhāraḥ // (12.1)
	kuto nirjātāḥ śākyāḥ kimagraṇyāḥ kimanvayāḥ kaśca teṣāṃ paurāṇaḥ kulavaṃśaḥ // (13.1)
	sacet kaścid asmākam upasaṃkramyaivaṃ pṛcchet kuto nirjātā bhavantaḥ śākyāḥ kimagraṇyāḥ kimanvayāḥ kaśca teṣāṃ paurāṇaḥ kulavaṃśa iti evaṃ pṛṣṭā vayaṃ kiṃ vyākuryāmaḥ // (14.1)
	na ca punar jānīmaḥ kuto nirjātāḥ śākyāḥ kimagraṇyāḥ kimanvayāḥ kaś ca paurāṇaḥ kulavaṃśa iti // (15.1)
	ete vayaṃ yena bhagavāṃs tenopasaṃkramāmaḥ // (16.1)
	upasaṃkramya bhagavantam etam evārthaṃ paripṛcchāmaḥ / (17.1)
	yathā cāsmākaṃ bhagavān vyākariṣyati tathainaṃ dhārayiṣyāma iti // (17.2)
	te vayam etam evārthaṃ paripṛcchāmaḥ kuto bhagavan nirjātāḥ śākyāḥ kimagraṇyāḥ kimanvayāḥ kaś ca śākyānāṃ paurāṇaḥ kulavaṃśa iti // (18.1)
	atha bhagavata etad abhavat // (19.1)
	saced ahaṃ śākyānāṃ paurāṇaṃ kulavaṃśam ārabhya dharmyāṃ kathāṃ kuryāṃ sthānam etad vidyate yad anyatīrthikaparivrājakā evaṃ vadeyuḥ ātmaślāghī śramaṇo gautamo yad icchati tad vyākarotīti // (20.1)
	atha ko nu mama śrāvakaḥ pratibalaḥ syād yaḥ śākyānāṃ paurāṇaṃ kulavaṃśam ārabhya bhikṣūṇāṃ dharmyāṃ kathāṃ kuryāt tena khalu samayenāyuṣmān mahāmaudgalyāyanastasyām eva pariṣadi saṃniṣaṇṇo 'bhūt saṃnipatitaḥ // (21.1)
	bhavati gautamā sa samayo yad ayaṃ lokaḥ saṃvartate saṃvartamāne loke yadbhūyasā sattvā ābhāsvare devanikāye upapadyante te tatra bhavanti rūpiṇo manomayāḥ avikalā ahīnendriyāḥ sarvāṅgapratyaṅgopetāḥ śubhā varṇasthāyinaḥ svayamprabhā vihāyasaṃgamāḥ prītibhakṣāḥ prītyāhārāḥ dīrghāyuṣo dīrgham adhvānaṃ tiṣṭhanti / (22.1)
	tena khalu samayeneyaṃ mahāpṛthivī ekodakā bhavatyekārṇavā // (22.2)
	yaḥ khalu ekodakāyā mahāpṛthivyā ekārṇavāyā upari vāyunā saraḥ saṃgacchati saṃmūrchati saṃtanoti tadyathā payasaḥ pakvasya śītībhūtasya upari vāyunā saraḥ saṃgacchati saṃmūrchati saṃtanoti / (23.1)
	evam ekodakāyā mahāpṛthivyā ekārṇavāyā upari vāyunā saraḥ saṃgacchati saṃmūrchati saṃtanoti // (23.2)
	sa bhavati pṛthivīraso varṇasampanno gandhasampanno rasasampannaḥ evaṃrūpo varṇena tadyathā navanītam evaṃrūpo rasena tadyathā kṣaudramadhv aneḍakam // (24.1)
	bhavati gautamā sa samayo yad ayaṃ loko vivartate vivartamāne loke tata eke sattvā āyuḥkṣayāt karmakṣayāt puṇyakṣayāt ābhāsvarād devanikāyāccyutvā ittham āgacchanti mānuṣyāṇāṃ sabhāgatāyām // (25.1)
	te iha bhavanti rūpiṇo manomayā avikalā ahīnendriyāḥ sarvāṅgapratyaṅgopetāḥ śubhā varṇasthāyinaḥ svayamprabhā vihāyasaṃgamāḥ prītibhakṣāḥ prītyāhārā dīrghāyuṣo dīrgham adhvānaṃ tiṣṭhanti // (26.1)
	tena khalu samayena na sūryācandramasor loke prādurbhāvo bhavati na nakṣatrāṇām na kṣaṇalavamuhūrtānām na rātriṃdivasānām na māsārdhamāsṛtusaṃvatsarāṇāṃ loke prādurbhāvo bhavati // (27.1)
	na strī prajñāyate na puruṣo nānyatra sattvaḥ sattva iti saṃkhyā gacchati // (28.1)
	athānyatamo lolupajātīyaḥ sattvaḥ pṛthivīrasam aṅgulyagreṇāsvādayati yathā yathāsvādayati tathā tathā rocayate yathā yathā rocayate tathā tathā kavaḍīkārāhāropakrameṇa paribhuṅkte // (29.1)
	adrākṣur anye'pi sattvāstaṃ sattvaṃ pṛthivīrasam aṅgulyagreṇāsvādayamānam yathā yathāsvādayati tathā tathā rocayate yathā yathā rocayate tathā tathā kavaḍīkāropakrameṇa paribhuktavān iti // (30.1)
	dṛṣṭvā ca punas te sattvāḥ pṛthivīrasam aṅgulyagreṇa āsvādayitum ārabdhāḥ yathā yathāsvādayanti tathā tathā rocayante yathā yathā rocayante tataḥ kavaḍīkāropakrameṇa paribhuktavantaḥ yataś ca te sattvāḥ pṛthivīrasaṃ kavaḍīkārāhāropakrameṇa paribhuktās tatas teṣāṃ sattvānāṃ kharatvaṃ ca gurutvaṃ ca kāye 'vakrāntam // (31.1)
	teṣāṃ yāsau śubhā varṇanibhā sāntarhitā // (32.1)
	andhakāraṃ loke prādurbhūtam // (33.1)
	dharmatā khalu gautamā andhakārasya loke prādurbhāvāt sūryācandramasor loke prādurbhāvo bhavati nakṣatrāṇām kṣaṇalavamuhūrtānāṃ rātriṃdivasānāṃ māsārdhamāsartusaṃvatsarāṇāṃ loke prādurbhāvo bhavati // (34.1)
	te tadbhakṣās tadāhārā dīrghāyuṣo dīrgham adhvānaṃ tiṣṭhanti // (35.1)
	teṣāṃ yo 'lpataram āhāram āharati sa varṇavān bhavati yaḥ prabhūtataram āhāram āharati sa durvarṇo bhavati ity āhāradvimātratāṃ pratītya varṇadvimātratā prajñāyate // (36.1)
	varṇadvimātratāyāṃ satyāṃ sattvaḥ sattvam avamanyate haṃbhoḥ sattva varṇavān ahaṃ durvarṇas tvam iti // (37.1)
	teṣāṃ varṇābhimānikānāṃ satāṃ teṣām eva pāpakānām akuśalānāṃ dharmāṇāṃ samādānahetoḥ pṛthivīraso 'ntarhitaḥ // (38.1)
	antarhite pṛthivīrase te sattvāḥ saṃgamya samāgamya śocanti klāmyanti paridevante // (39.1)
	evaṃ cāhur aho rasa aho rasa iti // (40.1)
	tadyathaitarhi manuṣyāḥ kiṃcid eva svādu subhojanaṃ bhuktvā tad eva purāṇam akṣarapadavyañjanam anusmaranta evam āhur aho rasa aho rasa iti // (41.1)
	evaṃ te sattvā antarhite gautamā pṛthivīrase saṃgamya samāgamya śocanti klāmyanti paridevante // (42.1)
	evaṃ cāhur aho rasa aho rasa iti // (43.1)
	arthaṃ cāsya na jānanti ayam asya bhāṣitasyārtho 'yam asya bhāṣitasyārtha iti // (44.1)
	antarhite pṛthivīrase teṣāṃ sattvānāṃ pṛthivīparpaṭakaḥ prādurbhūto varṇasampanno gandhasampanno rasasampanna evaṃrūpo varṇena tadyathā karṇikārapuṣpam evaṃrūpo rasena tadyathā kṣaudramadhvaneḍakam // (45.1)
	te tadbhakṣās tadāhārāḥ dīrghāyuṣo dīrgham adhvānaṃ tiṣṭhanti // (46.1)
	teṣāṃ yo 'lpataram āhāram āharati sa varṇavān bhavati yaḥ prabhūtataram āhāram āharati sa durvarṇa ityāhāradvimātratāṃ pratītya varṇadvimātratā prajñāyate // (47.1)
	varṇadvimātratāyāṃ satyāṃ sattvaḥ sattvam avamanyate haṃbhoḥ sattva varṇavān aham asmi durvarṇas tvam iti // (48.1)
	teṣāṃ varṇābhimānikānāṃ satāṃ teṣām eva pāpakānām akuśalānāṃ dharmāṇāṃ samādānahetoḥ pṛthivīparpaṭako 'ntarhitaḥ // (49.1)
	antarhite pṛthivīparpaṭake te sattvāḥ saṃgamya samāgamya śocanti klāmyanti paridevante // (50.1)
	evaṃ cāhur aho bata aho bateti // (51.1)
	tadyathaitarhi manuṣyāḥ kenacid eva duḥkhadaurmanasyena spṛṣṭāḥ tāny eva purāṇāny akṣarapadavyañjanāny anuvyavaharanta evam āhur aho bata aho bateti // (52.1)
	evam eva te sattvā antarhite pṛthivīparpaṭake saṃgamya samāgamya śocanti klāmyanti paridevante // (53.1)
	evaṃ cāhur aho bata aho bateti // (54.1)
	arthaṃ cāsya na jānanty ayam asya bhāṣitasyārtho 'yam asya bhāṣitasyārtha iti // (55.1)
	antarhite pṛthivīparpaṭake teṣāṃ sattvānāṃ vanalatā prādurbhūtā varṇasampannā gandhasampannā rasasampannā evaṃrūpā varṇena tadyathā kadambakāpuṣpam evaṃrūpā rasena tadyathā kṣaudramadhvaneḍakam // (56.0)
	te tadbhakṣās tadāhārā dīrghāyuṣo dīrgham adhvānaṃ tiṣṭhanti // (57.1)
	teṣāṃ yo 'lpataram āhāram āharati sa varṇavān bhavati yaḥ prabhūtam āhāram āharati sa durvarṇo bhavatīty āhāradvimātratāṃ pratītya varṇadvimātratā prajñāyate // (58.1)
	varṇadvimātratāyāṃ satyāṃ sattvaḥ sattvam avamanyate haṃbhoḥ sattva varṇavān aham durvarṇas tvam iti // (59.1)
	teṣāṃ varṇābhimānikānāṃ satāṃ teṣām eva pāpakānām akuśalānāṃ dharmāṇāṃ samādānahetoḥ vanalatā antarhitā // (60.1)
	paribhuktavantaḥ // (108.1)tato 'sya śāleḥ kuṇaś ca tuṣaś ca taṇḍulaṃ paryavanahyati // (109.1)
	lūno lūno na prativirohaty abalaś ca prajñāyate // (110.1)
	ṣaṇḍavanaṣaṇḍeṣu vyavasthitaḥ śāliḥ // (111.1)
	yan nu vayaṃ saṃgamya samāgamya kṣetrāṇi māpayema sīmāṃ badhnīyāma maryādāṃ sthāpayema idaṃ tava idaṃ mameti te saṃgamya samāgamya kṣetrāṇi māpitavantaḥ sīmāṃ ca baddhavantaḥ maryādāṃ sthāpitavantaḥ // (112.1)
	ayaṃ gautamā purāṇo 'graṇīḥ maryādākarmāntānāṃ loke prādurbhāvo bhavati // (113.1)
	tac ca dharmeṇa nādharmeṇa // (114.1)
	tatrāyaṃ dharmo dharmaḥ śreṣṭho jinendrāṇām // (115.1)
	athānyatamaḥ sattvaḥ tiṣṭhati sve śālau parakīyaṃ śālim adattam ādatte // (116.1)
	adrākṣīd anyataraḥ sattvaḥ taṃ sattvaṃ tiṣṭhati sve śālau parakīyaṃ śālim adattam ādadānaṃ dṛṣṭvā ca punas taṃ sattvam idam avocat kasmāt tvam bhoḥ sattva tiṣṭhati sve śālau parakīyaṃ śālim ādatse gaccha bhos tvam sattva mā bhūya evaṃ kārṣir dvir api trir api sa sattvaḥ tiṣṭhati sve śālau parakīyaṃ śālim adattam ādatte // (117.1)
	adrākṣīt sa sattvaḥ taṃ sattvaṃ dvir api trir api tiṣṭhati sve śālau parakīyaṃ śālim adattam ādadānaṃ dṛṣṭvā ca punas taṃ sattvam idam avocat // (118.1)
	kasmāt tvaṃ bhoḥ sattva tiṣṭhati sve śālau parakīyaṃ śālim adattam ādatse // (119.1)
	sa tam ākarṣati parākarṣati yāvat parṣanmadhye 'py avatārayati ayaṃ bhavantaḥ sattvaḥ tiṣṭhati sve śālau parakīyaṃ śālim ādatta iti // (120.1)
	atha te sattvās taṃ sattvam idam avocan kasmāt tvaṃ bhoḥ sattva tiṣṭhati sve śālau yāvat trir api parakīyaṃ śālim adattam ādatse gaccha tvaṃ bhoḥ sattva mā bhūya evaṃ kārṣir iti // (121.1)
	atha sa sattvas tān sattvān idam avocat anenāsmi bhavantaḥ sattvena śālikāraṇād ākṛṣṭaḥ parākṛṣṭo yāvat parṣanmadhye api avadhyāyitaḥ // (122.1)
	atha te sattvāḥ sattvam idam avocan kasmāt tvaṃ bhoḥ sattva sattvaṃ śālikāraṇād ākarṣasi parākarṣasi yāvat parṣanmadhye 'py avatarayasi gaccha tvaṃ bhoḥ sattva mā bhūya evaṃ kārṣīr iti // (123.1)
	atha teṣāṃ sattvānām etad abhavat dṛśyante khalu bhavantaḥ śālikāraṇād ākarṣaṇam api parākarṣaṇam api yāvatparṣanmadhye 'py avatāraṇam // (124.0)
	yan nu vayaṃ saṃgamya samāgamya yo 'smākaṃ sattvo 'bhirūpataraśca darśanīyataraś ca prāsādikataraś ca maheśākhyataraś ca taṃ vayaṃ kṣetrāṇām adhipatiṃ sthāpayema yo 'smākaṃ nigṛhītavyāṃśca nigrahīṣyati pragṛhītavyāṃś ca pragrahīṣyati // (125.0)
	yac cāsmākaṃ kṣetrebhyaḥ sampatsyate tato 'smai dharmyāṃ kṣitim anupradāsyāma iti te saṃgamya samāgamya yas teṣāṃ sattvo 'bhirūpataraśca darśanīyataraś ca prāsādikataraś ca maheśākhyataraś ca taṃ kṣetrāṇām adhipatiṃ sthāpayanti // (126.0)
	evaṃ cāhuḥ ehi tvaṃ bhoḥ sattva asmān nigṛhītavyāṃś ca nigṛhāṇa pragṛhītavyāṃś ca pragṛhāṇa yaccāsmākaṃ kṣetrebhyaḥ sampatsyate tatas te vayaṃ dharmyāṃ kṣitim anupradāsyāma iti // (127.0)
	sa teṣāṃ nigṛhītavyāṃś ca nigṛhṇāti pragṛhītavyāṃś ca pragṛhṇāti // (128.0)
	yac ca teṣām kṣetrebhyaḥ sampadyate tato 'smai dharmyāṃ kṣitim anuprayacchanti mahājanena saṃmato mahāsaṃmata iti mahāsaṃmato mahāsaṃmata iti saṃjñodapādi // (129.0)
	kṣetrāṇām adhipatiḥ kṣatāc ca trāyata iti kṣatriyaḥ kṣatriya iti saṃjñodapādi // (130.0)
	dharmeṇa prajā rañjayati śīlavṛttasamudācāreṇa prajñāvṛttasamudācāreṇeti rājā rājeti saṃjñodapādi mahāsammatasya gautamā rājño manuṣyāṇāṃ sattvā sattvā iti saṃjñābhūt // (131.0)
	mahāsammatasya gautamā rājño rocaḥ putraḥ rocasya rājño mānuṣyāṇām ehikā ehikā iti saṃjñodapādi rocasya gautamā kalyāṇaḥ putraḥ kalyāṇasya rājño mānuṣyāṇām tilakās tilakā iti saṃjñodapādi kalyāṇasya gautamā varakalyāṇaḥ putraḥ varakalyāṇasya gautamā rājño mānuṣyāṇām abhrakaṇṭhā abhrakaṇṭhā iti saṃjñodapādi // (132.0)
	varakalyāṇasya upoṣadhaḥ putraḥ // (133.0)
