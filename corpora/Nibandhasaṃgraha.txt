idānīṃ sarvendriyādhiṣṭhānatvena prāṇādhiṣṭhānatvena ca śiraḥ pradhānamataḥ prathamaṃ śiraḥsthān vyādhīn kathayāmīti nirdiśannāha mahata ityādi // (1.0)
	tasya uttaratantrasyādau uttamāṅgasthān rogān abhidadhāmīti saṃbandhaḥ // (2.0)
	ambudheriva samudrasyeva mahataḥ ativistīrṇasya durgādhasya agādhasya // (3.0)
	saṃkhyayā ṣaṭsaptatyā videhādhipaproktayā lakṣaṇaiḥ vyañjanaiḥ cakāreṇa hetucikitsābhyāṃ sādhyāsādhyakrameṇa ceti sādhyāsādhyaparipāṭyā // (4.0)
	athavā lakṣaṇaiḥ hetupūrvarūpopaśayasaṃprāptibhiḥ cakāreṇa cikitsitam evānuktaṃ samuccīyate kecicca kramaśabdenaiva cikitsitaṃ gṛhṇanti // (5.0)
