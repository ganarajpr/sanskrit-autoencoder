māhātmyametadvṛṣabhadhvajasya śrutvā muner gandhavatīsutasya / (1.1)
	vaco mahārthaṃ munayaśca sarve papracchur ugraṃ praṇipatya sūtam // (1.2)
	munaya ūcuḥ / (2.1)
	kimandhakārirbhagavān pinākī kasyānvaye vīryavataḥ pṛthivyām / (2.2)
	jāto mahātmā balavān pradhāne kaścāndhakaḥ kasya sutaḥ kathaṃ vā // (2.3)
	etat samagraṃ sarahasyamadya bravīhi sūta bhagavatprasādāt / (3.1)
	vyāsāt tvayā vai viditaṃ hi samyag yuddhaṃ yathābhūd anayor hataḥ saḥ // (3.2)
	teṣāṃ tadāsau vacanaṃ niśamya provāca sūtastu munīṃstadānīm / (4.1)
	pūrvaṃ dvijā mandaraśailasaṃsthā kaparddinaścaṇḍaparākramasya // (4.2)
	cakre tato netranimīlanaṃ tu sā pārvvatī narmayutaṃ salīlam / (5.1)
	pravālahemābjadhṛtaprabhābhyāṃ karāmbujābhyāṃ nimimīla netre // (5.2)
	harasya netreṣu nimīliteṣu kṣaṇena jātaḥ sumahāndhakāraḥ / (6.1)
	tatsparśayogācca maheśvarasya karācca tasyāḥ skhalitaṃ madāmbhaḥ // (6.2)
	śambhorlalāṭekṣaṇavahnitapto vinirgato bhūrijalasya binduḥ / (7.1)
	garbho babhūvātha karānanasya bhayaṅkaraḥ krodhaparaḥ kṛtaghnaḥ // (7.2)
	anyo virūpī jaṭilaḥ śmaśrulaśca kṛṣṇo naro vai vikṛtaḥ suromā / (8.1)
	gāyan hasan prarudan nṛtyamāno vilelihāno ghanaghoraghoṣaḥ // (8.2)
	jātena tenādbhutadarśanena gaurīṃ bhavo'sau smitapūrvam āha / (9.1)
	nimīlya netrāṇi kṛtaṃ ca narma vibheṣase māṃ dayite kathaṃ tvam // (9.2)
	gaurī harāt tadvacanaṃ niśamya vihasyamānā pramumoca netre / (10.1)
	jāte prakāśe sati ghorarūpo jāto'ndhakārādapi netrahīnaḥ // (10.2)
	papraccha ko'yaṃ bhagavan virūpas tādṛgvidhaṃ tacca nirīkṣya bhūtam / (11.1)
	vadasva satyaṃ mama kiṃ nimittaṃ sṛṣṭo'thavā kena ca kasya puttraḥ // (11.2)
	śrutvā harastadvacanaṃ priyāyā utpanna eṣo'dbhutacaṇḍavīryyaḥ / (12.1)
	nimīlite cakṣuṣī me bhavatyā sa svedajaścāndhakanāmadheyaḥ // (12.2)
	tato'sya kartāsmi yathānurūpaṃ tvayā sasakhyā sadayaṃ gaṇebhyaḥ / (13.1)
	sa rakṣitavyas tvayanaṃ hi caikaṃ vicārya buddhyā karaṇīyam ārye // (13.2)
	gaurī tato bharttṛvaco niśamya kāruṇyabhāvāt sahitā sakhībhiḥ / (14.1)
	nānāprakārair bahubhir hyupāyaiścakāra rakṣāṃ svasutasya yadvat // (14.2)
	kāle'tha yasmin bhuvi cāndhakastu hiraṇyanetrastvatha putrakāmaḥ / (15.1)
	araṇyam āśritya tapaścakāra jātastadā kaśyapajaḥ sutārtham // (15.2)
	kāṣṭhopamo'sau jitaroṣadoṣaḥ saṃdarśanārthaṃ tu maheśvarasya / (16.1)
	tuṣṭaḥ pinākī tapasāsya samanyag varapradānāya yayau dvijendrāḥ // (16.2)
	tuṣṭastamāhākulitendriyaṃ taṃ kimartham etadvratam āśritaṃ te / (17.1)
	prabrūhi kāmaṃ varado bhavo'haṃ yad icchasi tvaṃ sakalaṃ dadāmi // (17.2)
	puttrastu me candralalāṭa nāsti suvīryavān daityakulānurūpī / (18.1)
	tadarthametad vratam āsthito'haṃ taṃ dehi deveśa suvīryyavantam // (18.2)
	yasmācca madbhrāturanantavīryyāḥ prahrādapūrvā api pañca puttrāḥ / (19.1)
	mameha nāstīti niranvayo'haṃ yo māmakaṃ rājyamidaṃ bubhūṣet // (19.2)
	rāṣṭraṃ parasya svabalena hṛtvā bhuṅkte'thavā svaṃ pitureva diṣṭam / (20.1)
	sa procyate puttra ihāpyamutra puttrī sa tenāpi bhavet pitāsau // (20.2)
	ūrdhvaṃ gatiḥ puttravatāṃ niruktā manīṣibhir dharmavatāṃ variṣṭhaiḥ / (21.1)
	sarvāṇi bhūtāni tadarthameva mṛtau pravartanti haṭhāt svabhāvataḥ // (21.2)
	niranvayasyāpi na santi lokās tadarthamicchanti sutān svabhūmne / (22.1)
	krītādayaḥ santi sutāstathāṣṭau icchāmi tasmāt sutam ekameva // (22.2)
	etadbhavastadvacanaṃ niśamya kṛpākaro daityanṛpasya tuṣṭaḥ / (23.1)
	tamāha daityādhipa nāsti puttras tvadvīryyajaḥ kiṃtu dadāmi puttram // (23.2)
	mamātmajaṃ tvandhakanāmadheyaṃ tvattulyavīryyaṃ hyaparājitaṃ ca / (24.1)
	vṛṇīṣva puttraṃ sakalaṃ vihāya duḥkhaṃ pratīcchasva sukhaṃ hi cemam // (24.2)
	ityevamuktvā pradadau sa tasmai hiraṇyanetrāya sutaṃ prasannaḥ / (25.1)
	harastu gauryā sahito mahātmā bhūtādhināthastripurārirugraḥ // (25.2)
	tato harāt prāpya sutaṃ sa daityaḥ pradakṣiṇīkṛtya yathākrameṇa / (26.1)
	stotrair anekairabhipūjya rudraṃ hṛṣṭaḥ svarājyaṃ gatavān mahātmā // (26.2)
	tatastu puttraṃ giriśād avāpya rasātalaṃ caṇḍaparākramastu / (27.1)
	imāṃ dharitrīmanayat svadeśaṃ daityo vijitya tridaśānaśeṣān // (27.2)
	tatastu devairmunibhiśca sarvaiḥ sarvātmakaṃ yajñamayaṃ karālam / (28.1)
	vārāhamāśritya vapuḥ pradhānam ārādhito viṣṇur anantavīryyaḥ // (28.2)
	ghoṇāprahārair vividhairdharitrīṃ vidārya pātālatalaṃ praviśya / (29.1)
	tuṇḍena daityān śataśo vicūrṇya daṃṣṭrābhir agryābhir akhaṇḍitābhiḥ // (29.2)
	pādaprahārair aśaniprakāśair unmathya sainyāni niśācarāṇām / (30.1)
	mārtaṇḍakoṭipratimena paścāt sudarśanenādbhutacaṇḍatejāḥ // (30.2)
	hiraṇyanetrasya śiro jvalantaṃ cicheda daityāṃśca dadāha duṣṭān / (31.1)
	tataḥ prahṛṣṭo ditijendrarājye tamandhakaṃ tatra sa cābhyaṣiñcat // (31.2)
	svasthānam āsādya tato dharitrīṃ daṃṣṭrāṅkureṇādadharat prahṛṣṭaḥ / (32.1)
	bhūmiṃ ca pātālatalānmahātmā pupoṣa bhāgaṃ tvatha pūrvakaṃ tu // (32.2)
	pratāpya lokānakhilāṃstato'sau samāgataḥ padmayoniṃ dadarśa / (33.1)
	varaṃ pradātuṃ pravadantameva varaṃ vṛṇīṣveti sa mūḍhabuddhiḥ // (33.2)
	mṛtyor bhayaṃ me bhagavan sadaiva pitāmahābhūnna kadācidevam / (34.1)
	śastrāstrapāśāśaniśuṣkavṛkṣagirīndratoyāgniripuprahāraiḥ // (34.2)
	svarge dharaṇyāṃ divase niśāyāṃ daivāhataḥ prāha na so'pi sandhau / (35.1)
	tasyaitadīdṛgvacanaṃ niśamya daityendra tuṣṭo'smi labhasva sarvam // (35.2)
	praṇamya viṣṇuṃ manasā tamāha bhayānvito'sāvapi padmayoniḥ / (36.1)
	tataḥ sa daityaḥ paripūrṇakāmastvam aṣṭakoṭyastvatha ṣaṇṇavatyaḥ // (36.2)
	uttiṣṭha rājyaṃ kuru dānavānāṃ śrutvā giraṃ tāṃ sa sukhī babhūva / (37.1)
	rājye'bhiṣiktaḥ prapitāmahena trailokyanāśāya matiṃ cakāra // (37.2)
	utsādya dharmmān sakalān pramatto jitvāhave so'pi surān samastān / (38.1)
	tato bhayād indramukhāśca devāḥ kṣīrodadhau yatra haristu śete // (38.2)
	ārādhayāmāsur atīva gatvā tatastu hṛṣṭaḥ pradadau varāṃstu / (39.1)
	utthāya tasmācchayanād upendro yathānurūpair vividhair vacobhiḥ // (39.2)
	āśvāsya devānakhilān munīndrā ravīnduvaiśvānaratulyatejāḥ / (40.1)
	āśritya rūpaṃ jaṭilaṃ karālaṃ daṃṣṭrāyudhaṃ tīkṣṇanakhaṃ sunāsam // (40.2)
	mṛgendratulyaṃ ca vidāritāsyaṃ mārtaṇḍakoṭipratimaṃ sughoram / (41.1)
	yugāntakālāgnisamaprabhāvaṃ jaganmayaṃ kiṃ bahubhirvacobhiḥ // (41.2)
	astaṃ ravau so'pi hi gacchamāne gato'surāṇāṃ nagarīṃ mahātmā / (42.1)
	kṛtvā ca yuddhaṃ prabalaiḥ sahāyair hatvātha tān daityagaṇān grahītum // (42.2)
