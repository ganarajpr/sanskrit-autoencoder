mahātapā uvāca / (1.1)
	pūrvamāsīnmahādaityo balavānandhako bhuvi / (1.2)
	sa devān vaśamāninye brahmaṇo varadarpitaḥ // (1.3)
	tenātmavān surāḥ sarve tyājitā meruparvatam / (2.1)
	brahmāṇaṃ śaraṇaṃ jagmurandhakasya bhayārditāḥ // (2.2)
	tān āgatāṃstadā brahmā uvāca surasattamān / (3.1)
	kimāgamanakṛtyaṃ vo devā brūta kimāsyate // (3.2)
	devā ūcuḥ / (4.1)
	andhakenārditāḥ sarve vayaṃ devā jagatpate / (4.2)
	trāhi sarvāṃś caturvaktra pitāmaha namo'stu te // (4.3)
	brahmovāca / (5.1)
	andhakānnaiva śakto'haṃ trātuṃ vai surasattamāḥ / (5.2)
	bhavaṃ śarvaṃ mahādevaṃ vrajāma śaraṇārthinaḥ // (5.3)
	kiṃtu pūrvaṃ mayā datto varastasya surottamāḥ / (6.1)
	avadhyastvaṃ hi bhavitā na śarīraṃ spṛśenmahīm // (6.2)
	tasyaivaṃ balinastveko hantā rudraḥ paraṃtapaḥ / (7.1)
	tatra gacchāmahe sarve kailāsanilayaṃ prabhum // (7.2)
	evam uktvā yayau brahmā sadevo bhavasannidhau / (8.1)
	tasya saṃdarśanād rudraḥ pratyutthānādikāḥ kriyāḥ / (8.2)
	kṛtvābhyuvāca deveśo brahmāṇaṃ bhuvaneśvaram // (8.3)
	śaṃbhuruvāca / (9.1)
	kiṃ kāryaṃ devatāḥ sarvā āgatā mama sannidhau / (9.2)
	yenāhaṃ tatkaromyāśu ājñā kāryā hi satvaram // (9.3)
	devā ūcuḥ / (10.1)
	rakṣasva deva balinastvandhakād duṣṭacetasaḥ // (10.2)
	yāvadevaṃ surā sarve śaṃsanti parameṣṭhinaḥ / (11.1)
	tāvat sainyena mahatā tatraivāndhaka āyayau // (11.2)
	balena caturaṅgeṇa hantukāmo bhavaṃ mṛdhe / (12.1)
	tasya bhāryāṃ girisutāṃ hartumicchan sasādhanaḥ // (12.2)
	taṃ dṛṣṭvā sahasāyāntaṃ devaśakraprahāriṇam / (13.1)
	saṃnahya sahasā devā rudrasyānucarā bhavan // (13.2)
	rudro'pi vāsukiṃ dhyātvā takṣakaṃ ca dhanaṃjayam / (14.1)
	valayaṃ kaṭisūtraṃ ca cakāra parameśvaraḥ // (14.2)
	nīlanāmā ca daityendro hastī bhūtvā bhavāntikam / (15.1)
	āgatastvaritaḥ śakrahastīvoddhatarūpavān // (15.2)
	sa jñāto nandinā daityo vīrabhadrāya darśitaḥ / (16.1)
	vīrabhadro'pi siṃhena rūpeṇāhatya ca drutam // (16.2)
	tasya kṛttiṃ vidāryāśu kariṇastvañjanaprabhām / (17.1)
	rudrāyārpitavān so'pi tam evāmbaram ākarot / (17.2)
	tataḥ prabhṛti rudro'pi gajacarmapaṭo 'bhavat // (17.3)
	gajacarmapaṭo bhūtvā bhujaṃgābharaṇojjvalaḥ / (18.1)
	ādāya triśikhaṃ bhīmaṃ sagaṇo'ndhakamanvayāt // (18.2)
	tataḥ pravṛtte yuddhe ca devadānavayormahat / (19.1)
	indrādyā lokapālāstu skandaḥ senāpatis tathā / (19.2)
	sarve devagaṇāś cānye yuyudhuḥ samare tadā // (19.3)
	taṃ dṛṣṭvā nārado yuddhaṃ yayau nārāyaṇaṃ prati / (20.1)
	śaśaṃsa ca mahad yuddhaṃ kailāse dānavaiḥ saha // (20.2)
	tacchrutvā cakramādāya garuḍastho janārdanaḥ / (21.1)
	tamevaṃ deśam āgamya yuyudhe dānavaiḥ saha // (21.2)
	āgatya ca tato devā hariṇāpyāyitā raṇe / (22.1)
	viṣaṇṇavadanāḥ sarve palāyanaparā bhavan // (22.2)
	tatra bhagneṣu deveṣu svayaṃ rudro'ndhakaṃ yayau / (23.1)
	tatra tena mahad yuddham abhavallomaharṣaṇam // (23.2)
	tatra devo 'pyasau daityaṃ triśūlenāhanad bhṛśam / (24.1)
	tasyāhatasya yad raktamapatad bhūtale kila / (24.2)
	tatrāndhakā asaṃkhyātā babhūvurapare bhṛśam // (24.3)
	tad dṛṣṭvā mahadāścaryaṃ rudraḥ śūle'ndhakaṃ mṛdhe / (25.1)
	gṛhītvā triśikhāgreṇa nanarta parameśvaraḥ // (25.2)
	itare 'pyandhakāḥ sarve cakreṇa parameṣṭhinā / (26.1)
	nārāyaṇena nihatāstatra ye'nye samutthitāḥ // (26.2)
	asṛgdhārātuṣāraistu śūlaprotasya cāsakṛt / (27.1)
	anārataṃ samuttasthus tato rudro ruṣānvitaḥ // (27.2)
	tasya krodhena mahatā mukhājjvālā viniryayau / (28.1)
	tadrūpadhāriṇī devī yāṃ tāṃ yogeśvarīṃ viduḥ // (28.2)
	svarūpadhāriṇī cānyā viṣṇunāpi vinirmitā / (29.1)
	brahmaṇā kārttikeyena indreṇa ca yamena ca / (29.2)
	varāheṇa ca devena viṣṇunā parameṣṭhinā // (29.3)
	pātāloddharaṇaṃ rūpaṃ tasyā devyā vinirmame / (30.1)
	māheśvarī ca rājendra ityetā aṣṭamātaraḥ // (30.2)
	kāraṇaṃ tāni yatproktaṃ kṣetrajñenāvadhāraṇam / (31.1)
	śarīrād devatānāṃ tu tadidaṃ kīrtitaṃ mayā // (31.2)
	kāmaḥ krodhastathā lobho mado moho'tha pañcamaḥ / (32.1)
	mātsaryaṃ ṣaṣṭhamityāhuḥ paiśunyaṃ saptamaṃ tathā / (32.2)
	asūyā cāṣṭamī jñeyā ityetā aṣṭamātaraḥ // (32.3)
	kāmaṃ yogeśvarīṃ viddhi krodho māheśvarīṃ tathā / (33.1)
	lobhastu vaiṣṇavī proktā brahmāṇī mada eva ca // (33.2)
	mohaḥ svayaṃbhūḥ kaumārī mātsaryaṃ cendrajaṃ viduḥ / (34.1)
	yamadaṇḍadharā devī paiśunyaṃ svayameva ca / (34.2)
	asūyā ca varāhākhyā ityetāḥ parikīrtitāḥ // (34.3)
	kāmādigaṇa eṣo'yaṃ śarīre parikīrtitaḥ / (35.1)
	jagrāha mūrtiṃ tu yathā tathā te kīrtitaṃ mayā // (35.2)
	etābhir devatābhiśca tasya rakte'tiśoṣite / (36.1)
	kṣayaṃ gatāsurī māyā sa ca siddho'ndhako'bhavat / (36.2)
	etat te sarvamākhyātam ātmavidyāmṛtaṃ mayā // (36.3)
	ya etacchṛṇuyānnityaṃ mātṝṇām udbhavaṃ śubham / (37.1)
	tasya tāḥ sarvato rakṣāṃ kurvantyanudinaṃ nṛpa // (37.2)
	yaścaitat paṭhate janma mātṝṇāṃ puruṣottama / (38.1)
	sa dhanyaḥ sarvadā loke śivalokaṃ ca gacchati // (38.2)
	tāsāṃ ca brahmaṇā dattā aṣṭamī tithiruttamā / (39.1)
	etāḥ sampūjayed bhaktyā bilvāhāro naraḥ sadā / (39.2)
	tasya tāḥ parituṣṭāstu kṣemārogyaṃ dadanti ca // (39.3)
