he bhasmāṅgaviraktirūpaguṇadaṃ taṃ preraṇādaṃ śivaṃ gaṅgābhūṣitaśekharaṃ smaraharaṃ śaktisvarūpaṃ prabho / (1.1)
	tvāmīśaṃ karuṇārṇavaṃ śaraṇadaṃ vidyānidhiṃ nirguṇaṃ bhūteśaṃ girijāpatiṃ śaśidharaṃ māṅgalyadevaṃ namaḥ / (1.2)
	namāmi devaṃ suravṛndapūjitaṃ gaṇādhipaṃ vighnavināśakārakam / (1.3)
	smaranti ye nityam udāracetasaḥ kaṣṭāni te nānubhavanti satyam / (1.4)
	siddhinandanamiśreṇa śuddhāṃ ca sāragarbhitām / (1.5)
	rasendracintāmaṇeśca ṭīkāṃ kurve śivājñayā / (1.6)
	idānīṃ kālanāthaśiṣyaḥ śrīḍhuṇḍhukanāthāhvayo rasendracintāmaṇigrantham ārabhamāṇas tanmūladevate śrīmadambikāmaheśvarau sakalajagadutpattisthitipralayanidānaṃ viśeṣasiddhāntagarbhavācā arthaprakāśakāsāravimarśāmbujinīmayam / (2.1)
	saccidānandavibhavaṃ śivayorvapurāśraye // (2.2)
	laghīyaḥ parimāṇatayā nikhilarasajñānadāyitvāccintāmaṇiriva cintāmaṇiḥ // (3.0)
	aśrauṣaṃ bahuviduṣāṃ mukhād apaśyaṃ śāstreṣu sthitam akṛtaṃ na tallikhāmi / (4.1)
	yatkarma vyaracayamagrato gurūṇāṃ prauḍhānāṃ tadiha vadāmi vītaśaṅkaḥ // (4.2)
	adhyāpayanti yad darśayituṃ kṣamante sūtendrakarma guravo guravasta eva / (5.1)
	śiṣyāsta eva racayanti guroḥpuro ye śeṣāḥ punastadubhayābhinayaṃ bhajante // (5.2)
	saṃskārāḥ paratantreṣu ye gūḍhāḥ siddhisūcitāḥ / (6.1)
	tāneva prakaṭīkartum udyamaṃ kila kurmahe // (6.2)
	granthād asmādāharanti prayogān svīyaṃ vāsminnāma ye nikṣipanti / (7.1)
	gotrāṇyeṣām asmadīyaśramoṣmā bhasmīkurvannāyugaṃ bobhavītu // (7.2)
	saṃskārāḥ śivajanuṣo bahuprakārās tulyā ye laghubahulaprayāsaṃsādhyāḥ / (8.1)
	yadyekaṃ sukaram udāharāmi teṣāṃ vyāhāraiḥ kimihaphalaṃ tataḥ pareṣām // (8.2)
	iha khalu puruṣeṇa duḥkhasya nirupādhidveṣaviṣayatvāt tadabhāvaścikīrṣitavyo bhavati sukhamapi nirupādhipremāspadatayā gaveṣaṇīyam tadetatpuruṣārthadvayam / (9.1)
	abhāvasyānyatvāt // (9.2)
	kiṃca srakcandanavanitādiviṣayāṇāṃ satyapi tatkāraṇatve nāntarīyakaduḥkhasambhedād anarthaparamparāparicitatvān mūrkhāṇāṃ kośāṇḍakavad ābhāsamānatvād anaikāntikatvād virodhināṃ yugapadadṛśyamānatvād atyantatāvirahitatvācca pariharaṇīyatvam // (10.0)
	ekāntātyantataśca punaste hyupāyāḥ khalu hariharabrahmāṇa iva tulyā eva sambhavanti / (11.1)
	jñānayogaḥ pavanayogo rasayogaśceti // (11.2)
	nanu kathameṣāṃ tulyatetyapekṣāyāṃ brūmaḥ mokṣopāye bṛhadvāsiṣṭhādau bhuśuṇḍopākhyāne vasiṣṭhavākyam / (12.1)
	asādhyaḥ kasyacidyogaḥ kasyacijjñānaniścayaḥ / (12.2)
	dvau prakārau tato devo jagāda paramaḥ śivaḥ / (12.3)
	prāṇānāṃ vā nirodhena vāsanānodanena vā / (12.4)
	no cetsaṃvidam ucchāṇāṃ karoṣi tadayogavān / (12.5)
	dvāveva hi samau rāma jñānayogāvimau smṛtau // (12.6)
	tatrādyayoḥ kevalaṃ pakvakaṣāyāṇāmapi kathañcana sādhyatvāccarame tu punarbhogalolupānām apyadhikāritvāttābhyāṃ samīcīno'yamiti kasya na pratibhāti // (13.0)
	kiṃca asya bhagavanniryāsatayā sevakānāṃ svasambhūtasakaladhātutvāpādakasya bhagavato rasarājasya guṇasindhoḥ kiyantaḥ pṛṣatāḥ prasaṅgāllikhyante // (14.0)
	acirājjāyate devi śarīramajarāmaram / (15.1)
	manasaśca samādhānaṃ rasayogādavāpyate // (15.2)
	sattvaṃ ca labhate devi vijñānaṃ jñānapūrvakam / (16.1)
	satyaṃ mantrāśca sidhyanti yo'śnāti mṛtasūtakam // (16.2)
	yāvanna śaktipātastu na yāvatpāśakṛntanam / (17.1)
	tāvattasya kutaḥ śuddhirjāyate mṛtasūtake // (17.2)
	yāvanna harabījaṃ tu bhakṣayetpāradaṃ rasam / (18.1)
	tāvattasya kuto muktiḥ kutaḥ piṇḍasya dhāraṇam // (18.2)
	svadehe khecaratvaṃ vai śivatvaṃ yena labhyate / (19.1)
	tādṛśe tu rasajñāne nityābhyāsaṃ kuru priye // (19.2)
	tvaṃ mātā sarvabhūtānāṃ pitā cāhaṃ sanātanaḥ / (20.1)
	dvayośca yo raso devi mahāmaithunasambhavaḥ // (20.2)
	darśanāt sparśanāt tasya bhakṣaṇāt maraṇātpriye / (21.1)
	pūjanādrasadānācca dṛśyate ṣaḍvidhaṃ phalam // (21.2)
	kedārādīni liṅgāni pṛthivyāṃ yāni kānicit / (22.1)
	tāni dṛṣṭvā ca yatpuṇyaṃ tatpuṇyaṃ rasadarśanāt // (22.2)
	candanāgurukarpūrakuṅkumāntargato rasaḥ / (23.1)
	mūrchitaḥ śivapūjā sā śivasāṃnidhyasiddhaye // (23.2)
	bhakṣaṇātparameśāni hanti tāpatrayaṃ rasaḥ / (24.1)
	durlabhaṃ brahmaviṣṇvādyaiḥ prāpyate paramaṃ padam // (24.2)
	hṛtpadmakarṇikāntaḥsthaṃ rasendraṃ parameśvari / (25.1)
	smaranvimucyate pāpaiḥ sadyo janmāntarārjitaiḥ // (25.2)
	svayambhūliṅgasāhasrairyatphalaṃ samyagarcanāt / (26.1)
	tatphalaṃ koṭiguṇitaṃ rasaliṅgārcanādbhavet // (26.2)
	rogibhyo yo rasaṃ datte śuddhipākasamanvitam / (27.1)
	tulādānāśvamedhānāṃ phalaṃ prāpnoti śāśvatam // (27.2)
	rasavidyā parā vidyā trailokye'pi ca durlabhā / (28.1)
	bhuktimuktikarī yasmāttasmājjñeyā guṇānvitā // (28.2)
	brahmajñānena so'yukto yaḥ pāpī rasanindakaḥ / (29.1)
	nāhaṃ trātā bhave tasya janmakoṭiśatairapi // (29.2)
	ālāpaṃ gātrasaṃsparśaṃ yaḥ kuryādrasanindakaiḥ / (30.1)
	yāvajjanmasahasrāṇi sa bhavetpāpapīḍitaḥ // (30.2)
	hemajīrṇo bhasmasūto rudratvaṃ bhakṣito diśet / (31.1)
	viṣṇutvaṃ tārajīrṇastu brahmatvaṃ bhāskareṇa tu // (31.2)
	tīkṣṇajīrṇo dhaneśatvaṃ sūryatvaṃ cāpi tālake / (32.1)
	rājare tu śaśāṅkatvamamaratvaṃ ca rohaṇe // (32.2)
	sāmānyena tu tīkṣṇena naraḥ śakratvamāpnuyāt // (33.1)
	doṣahīno raso brahmā mūrchitastu janārdanaḥ / (34.1)
	mārito rudrarūpī syād baddhaḥ sākṣāt sadāśivaḥ // (34.2)
	īdṛśasya guṇānāṃ paryavasānamambujasambhavo'pi mahākalpairapi vacobhirnāsādayitumalamityalaṃ bahunā // (35.0)
	yadyanmayākriyata kārayituṃ ca śakyaṃ sūtendrakarma tadiha prathayāṃbabhūve / (36.1)
	adhyāpayanti ya idaṃ natu kārayanti kurvanti nedamadhiyantyubhaye mṛṣārthāḥ // (36.2)
