kalyāṇaṃ kamalāpatirdiśatu me yaḥ kaustubhe rādhayā vīkṣya svaṃ pratibimbitaṃ pratiyuvatyeṣeti tarkākulam / (1.1)
	āśleṣonmukhayāpi mānaparayā manvānayā kaitavaṃ tiryagvakritakaṃdharaṃ valitayā sāsūyam ālokitaḥ // (1.2)
	so 'vyānmām aravindanābha udayad yannābhipadmālayā padmā padmavimīlamīlanavidhiprācīṇyavannetrayoḥ / (2.1)
	yogād ardhanimīlitāmbujavaśāt sambhogabhogetarāvasthādvaṃdvabhavānubhūtijanitakrīḍāsukhāny anvabhūt // (2.2)
	diśyānme 'rdhaśivātanuḥ sa bhagavānnityoditāṃ sampadaṃ śambhur viśvajayaśriyaḥ paravaśīkāraikasatkārmaṇam / (3.1)
	yatraikāṅganavīnavibhramarasād ekāṅgagupter anāsvādāt sambhavadadbhutaikaparamā jāgarti hemādrijā // (3.2)
	natvā mataṃgabharatapramukhān sugītasaṃgītaśastranipuṇāñjayadevavācām / (4.1)
	śrīkumbhakarṇanṛpatirvivṛtiṃ tanoti gānaṃ nidhāya sarasaṃ rasikapriyāhvām // (4.2)
	śrīvaijayāyenasagotravaryaḥ śrībappanāmā dvijapuṃgavo'bhūt / (5.1)
	haraprasādād apasādarājyaprājyopabhogāya nṛpo'bhavadyaḥ // (5.2)
	yadanvaye nirjitatarkavādipadaḥ padaṃ tatparamā vyabhāti / (6.1)
	śrīmedapāṭe guhilapradhāne yatrābhavanbhūpatayaḥ prasūtāḥ // (6.2)
	tatra kramād bhavyaparamparāḍhye hammīranāmā nṛpatirbabhūva / (7.1)
	candrādiratnaprakarakrameṇa ratnākare kalpataruryathāsīt // (7.2)
	dānāni saṃgatavanīpakamātrapātramāsādya yo dadiranantaguṇāni kāmam / (8.1)
	pañcānano viṣamadhāḍiṣu yaḥ prasiddhaścakre mṛdhānyakhilaśatrubhayāvahāni // (8.2)
	tasmādabhūttanayaḥ prabhūtanayaḥ parāyaḥ sadayaḥ śatāyuḥ / (9.1)
	śrīkṣetrasiṃhaḥ pratipakṣanārāsiṃhaḥ parāhaṃkṛtimattasiṃhaḥ // (9.2)
	tato'bhavallakṣa udatilakṣavipakṣayakṣakṣapakāradakṣaḥ / (10.1)
	gayāvimokṣāsthitadharmarakṣaḥ svakṣaḥ kṛtatryakṣasamastalakṣyaḥ // (10.2)
	tannandano nirjitapūrvarājacāritrasampāditamedinīkaḥ / (11.1)
	śrīmokalendraḥ praṇatārimaulimāṇikyabhābhāsitapādapadmaḥ // (11.2)
	śrīkumbhakarṇastadanu kṣitīndraḥ kṣitiṃ bibhartīndrasamānasāraḥ / (12.1)
	śeṣādikebhyo dharaṇe dharitryā bharasya viśrāṇitaviśramaḥ san // (12.2)
	sa śrīśabhaktipravaṇaḥ pravīṇaḥ saṃgītaśāstre'khilaśākhavettā / (13.1)
	śrīgītagovindasugītakasya navyākṛtiṃ vyākṛtimātanoti // (13.2)
	jitvā tu pṛthivīṃ kṛtvā tatpatīnkaradāyinaḥ / (14.1)
	rādhāmādhavasārasya rasiko ramate'dhunā // (14.2)
	pratyajñāyi prabandho yo jayadevena dhīmatā / (15.1)
	na tasya vidyate lakṣma sarvāṅgairupalakṣitam // (15.2)
	ataḥ svarādibhiḥ ṣaḍbhiraṅgaiḥ saṃyojya tathyatām / (16.1)
	nītvā gītvā tadā hitvā kuṭīkāsu pravartyate // (16.2)
	śṛṅgāre saprapañce rasa iha ruciraucityayuktau prakṛṣṭe 'laṃkāre nāyikāyā guṇagaṇagaṇane varṇane nāyakasya / (17.1)
	gītau prītau ca vṛttau layamanu rasikāḥ kautukaṃ cet tademā doṣairmuktā guṇāḍhyāḥ śṛṇuta narapateḥ kumbhakarṇasya vācaḥ // (17.2)
	sa kiṃ bandhaḥ ślāghyo vrajati śithilībhāvamasakṛdvicāreṇākṣipto nanu bhavati ṭīkāpi kimu sā / (18.1)
