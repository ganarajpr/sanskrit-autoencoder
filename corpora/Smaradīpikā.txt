	samyag ārādhitaḥ kāmaḥ sugandhikusumādibhiḥ / (2.1)
	vidadhāti varastrīṇāṃ mānagranthivimocanam // (2.2)
	smaran nirjitya rudreṇa paścād uddīpitaḥ smaraḥ / (3.1)
	tena tannāmadheyena nirmitā smaradīpikā // (3.2)
	anekakāmaśāstrāṇāṃ sāram ākṛṣya yatnataḥ / (4.1)
	bālavyutpattaye strīṇāṃ cittasaṃtoṣaṇāya ca / (4.2)
	prabodhāya varastrīṇāṃ tuṣṭyai ratisukhāya ca // (4.3)
	gargeṇābhāṣitāṃ samyag vakṣyāmi smaradīpikām / (5.1)
	yasya vijñānamātreṇa mūrkho 'pi ratiraṅgadhīḥ // (5.2)
	bālikā taruṇī prauḍhā vṛddhā ceti viśeṣataḥ / (6.1)
	jñātavyo hy anvitaḥ kāmo dhruvaṃ śṛṅgāram icchatā // (6.2)
	kāmaśāstrasya tattvajñā jāyante sundarīpriyāḥ / (7.1)
	kāmaśāstram ajānanto ramante paśuvat striyam // (7.2)
	nānānibandhaiḥ suratopacāraiḥ krīḍāsukhaṃ janmaphalaṃ narāṇām / (8.1)
	kiṃ saurabheyīśatamadhyavartī vṛṣo 'pi saṃbhogasukhaṃ na bhuṅkte // (8.2)
	svanārīrakṣaṇaṃ puṃsāṃ paranāryanurañjanam / (9.1)
	bandhabhedeṅgitajñānam etat phalam udāhṛtam // (9.2)
	yena saṃvatsaro dṛṣṭaḥ sakṛt kāmaḥ susevitaḥ / (10.1)
	tena sarvam idaṃ dṛṣṭaṃ punar āvartitaṃ jagat // (10.2)
	prathamaṃ lakṣaṇaṃ puṃsāṃ strīṇāṃ ca tadanantaram / (11.1)
	dhvajasya lakṣaṇaṃ proktaṃ bhagalakṣaṇasaṃyutam // (11.2)
	kāmasthānāni saṃlakṣya punaḥ samyakpracālanam / (12.1)
	punaḥ ṣoḍaśa bandhāś ca tathaivādhomukhāś ca ṣaṭ // (12.2)
	dvau bandhau sundarīṇāṃ ca paścān mukharataṃ tathā / (13.1)
	bāhyaṃ rataṃ tataḥ kuryād rataṃ deśaviśeṣajam // (13.2)
	iṅgitasya parijñānaṃ dūtyāś ca tadanantaram / (14.1)
	tathāṣṭanāyikāyāś ca mantrauṣadhisutodayaḥ // (14.2)
	śaśo mṛgo vṛṣaś caiva caturthas tu hayas tathā / (15.1)
	kathayāmi kramāt puṃsām etaj jāticatuṣṭayam // (15.2)
	mṛducapalasuśīlaḥ komalāṅgaḥ suveṣaḥ sakalaguṇanidhānaṃ satyavādī śaśo 'sau // (16.1)
	vadati madhuravāṇīṃ nṛtyagītānuraktaḥ / (17.1)
	dvijasuragurubhakto bandhuyukto dhanāḍhyaḥ // (17.2)
	strījito gāyanaś caiva nārīsattvaparaḥ sukhī / (18.1)
	ṣaḍaṅgulo bhaven meḍhraḥ śrīmāṃś ca śaśako mataḥ // (18.2)
	vadati madhuravāṇīṃ raktanetraḥ suśīlaḥ calamatiratibhīruḥ śīghragāmī mṛgo 'sau // (19.1)
	udarakaṭikṛśaḥ syād dīrghabimbādharauṣṭho daśanavadanadīrgho dīrghabāhuḥ pratāpī // (20.1)
	alpabhug dhārmikaś caiva satyavādī priyaṃvadaḥ / (21.1)
	aṣṭāṅgulo bhaven meḍhro rūpayukto mṛgo mataḥ // (21.2)
	vṛṣo yathā udarakaṭikṛśāsyaḥ śīghragāmī natāṃsaḥ kanakaruciradehaḥ kaṣṭavādī vṛṣo 'sau // (22.1)
	vyasanakṛpaṇabuddhiḥ strīvaśaḥ strīvilāso bahuguṇabahutejāḥ dīrghanetro 'bhimānī // (23.1)
	upakāraparo nityaṃ strīvaśī śleṣmalas tathā / (24.1)
	lubdhaś ca kṛpaṇaś caiva mithyāvādī ca nirbhayaḥ / (24.2)
	daśāṅgulaśarīras tu medasvī vṛṣabho mataḥ // (24.3)
