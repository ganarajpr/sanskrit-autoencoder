vijñaptimātramevedamasadarthāvabhāsanāt / (1.1)
	yadvat taimirikasyāsatkeśoṇḍrakādidarśanam // (1.2)
	na deśakālaniyamaḥ saṃtānāniyamo na ca / (2.1)
	na ca kṛtyakriyā yuktā vijñaptiryadi nārthataḥ // (2.2)
	deśādiniyamaḥ siddhaḥ svapnavat pretavatpunaḥ / (3.1)
	saṃtānāniyamaḥ sarvaiḥ pūyanadyādidarśane // (3.2)
	svapnopaghātavatkṛtyakriyā narakavatpunaḥ / (4.1)
	narakapālādidarśane taiśca bādhane // (4.2)
	tiraścāṃ saṃbhavaḥ svarge yathā ca narake tathā / (5.1)
	na pretānāṃ yatastajjaṃ duḥkhaṃ nānubhavanti te // (5.2)
	yadi tatkarmabhistatra bhūtānāṃ saṃbhavastathā / (6.1)
	iṣyate pariṇāmaśca kiṃ vijñānasya neṣyate // (6.2)
	karmaṇo vāsanānyatra phalamanyatra kalpyate / (7.1)
	tatraiva neṣyate yatra vāsanā kiṃ nu kāraṇam // (7.2)
	rūpādyāyatanāstitvaṃ tadvineyajanaṃ prati / (8.1)
	abhiprāyavaśāduktamupapādukasattvavat // (8.2)
	nāstīha sattva ātmā vā dharmāstvete sahetukāḥ / (9.1)
	yataḥ svabījād vijñaptiryadābhāsā pravartate / (9.2)
	dvividhāyatanatvena te tasyā munirabravīt // (9.3)
	tathā pudgalanairātmyapraveśo hi anyathā punaḥ / (10.1)
	deśanā dharmanairātmyapraveśaḥ kalpitātmanā // (10.2)
	na tadekaṃ na cānekaṃ viṣayaḥ paramāṇuśaḥ / (11.1)
	na ca te saṃhatā yasmātparamāṇurna sidhyati // (11.2)
	ṣaṭkena yugapadyogātparamāṇoḥ ṣaḍaṃśatā / (12.1)
	ṣaṇṇāṃ samānadeśatvātpiṇḍaḥ syādaṇumātrakaḥ // (12.2)
	paramāṇorasaṃyoge tatsaṃghāte'sti kasya saḥ / (13.1)
	na cānavayavatvena tatsaṃyogānna sidhyati // (13.2)
	digbhāgabhedo yasyāsti tasyaikatvaṃ na yujyate / (14.1)
	chāyāvṛtī kathaṃ vā anyo na piṇḍaś cen na tasya te // (14.2)
	ekatve na krameṇetir yugapan na grahāgrahau / (15.1)
	vicchinnānekavṛttiśca sūkṣmānīkṣā ca no bhavet // (15.2)
	pratyakṣabuddhiḥ svapnādau yathā sā ca yadā tadā / (16.1)
	na so'rtho dṛśyate tasya pratyakṣatvaṃ kathaṃ matam // (16.2)
	uktaṃ yathā tadābhāsā vijñaptiḥ smaraṇaṃ tataḥ / (17.1)
	svapnadṛgviṣayābhāvaṃ nāprabuddho 'vagacchati // (17.2)
	anyonyādhipatitvena vijñaptiniyamo mithaḥ / (18.1)
	middhenopahataṃ cittaṃ svapne tenāsamaṃ phalam // (18.2)
	maraṇaṃ paravijñaptiviśeṣādvikriyā yathā / (19.1)
	smṛtilopādikānyeṣāṃ piśācādimanovaśāt // (19.2)
	kathaṃ vā daṇḍakāraṇyaśūnyatvamṛṣikopataḥ / (20.1)
	manodaṇḍo mahāvadyaḥ kathaṃ vā tena sidhyati // (20.2)
	paracittavidāṃ jñānamayathārthaṃ kathaṃ yathā / (21.1)
	svacittajñānam ajñānādyathā buddhasya gocaraḥ // (21.2)
	vijñaptimātratāsiddhiḥ svaśaktisadṛśī mayā / (22.1)
	kṛteyaṃ sarvathā sā tu na cintyā buddhagocaraḥ // (22.2)
