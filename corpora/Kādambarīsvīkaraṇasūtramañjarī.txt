ananyajasaptatantau anutarṣasvīkaraṇasyātyāvaśyakatvāt // (1.1)
	ratyānandasyodbodhe anirvacanīyānandasya kāraṇabhūtatvāt // (2.1)
	udañjyabhyutthāne pracetasaḥ kanyāyāḥ svīkaraṇasya paramakāraṇatvam // (3.1)
	prayojyaprādhānyakartṛtvābhāve ratitantraṃ vidhātum aśakyatvāt // (4.1)
	prayojye narmavyāpāravistāre abhyutthānaviśiṣṭodañjyabhāve sarvāṅgīṇavyāpāropalakṣitaṃ ratitantraṃ kartum adhikārābhāvāt // (5.1)
	ratitantravilāse anirvacanīyarasotpattau sīdhusaṃgrahaṇasya paramakāraṇatvam // (6.1)
	lalitavibhramabandhavilāse puṣkarādhipater ātmajāyāḥ naisargikasvabhāvaḥ // (7.1)
	upaśyāmayā saha madhuvāra udañjidhārṣṭye prayojakaḥ // (8.1)
	śyāmayā saha nirveśanaṃ yāmadvayasopalakṣitadiṣṭāvadhiretaḥstambhane paramakāraṇam // (9.1)
	nistanūruhavarāṅgasambhede yoṣāyā anuprāśanasyātyāvaśyakatvāt // (10.1)
	kriyamāṇe 'nuprāśane nirbhedavyathā nānubhūyate // (11.1)
	uttarasmin ghasre asaṃbheditam iva varāṅgaṃ paridṛśyate // (12.1)
	kriyamāṇe narmavyāpāre udañjer dhārṣṭyasya phalegrahiḥ paridṛśyate // (13.1)
	upaśyāmayā saha vyānatādibandhe samupasthitau pāśina ātmajāyāḥ svīkaraṇe anirvacanīyasukhānubhave hetuḥ kāraṇatvam // (14.1)
	aprāptayauvanābhiḥ saha samprayoge aṇumātraprāśanena caritārthatvāt // (15.1)
	tīyapratyayasya prakṛtibhūtāyām avasthāyāṃ kaśyasvīkaraṇasyātyāvaśyakatvaṃ sati pūrvarūpasaṃyoge // (16.1)
	taditarāvasthāyāṃ tu nādhikārakatvam // (17.1)
	ekatra parijñānābhāvāt // (18.1)
	itaratra pāñcabhautikāvayavānāṃ kṣīṇataratvāc ca // (19.1)
	haripriyāvatāṃ janānām etac chākheṇānuvartanam atyāvaśyakam // (20.1)
	taditareṣām etacchākhe pravṛttir na vidhīyate // (21.1)
	pāriśeṣyāj jaladhisambhavayā yukteṣv etacchāstrasya pravṛttiḥ // (22.1)
	gotrāpatyaghasre anutarṣasvīkaraṇasyātyāvaśyakatvam iti mārkaṇḍeyādiḥ // (23.1)
	kāvyaghasreṣv atyāvaśyakatamam iti surathavaiśyau // (24.1)
	vākyadvayasya śaktyupāsanāvatāṃ viniyogāt // (25.1)
	taditareṣāṃ yathākālopadeśaḥ // (26.1)
	dvitīyavarṇe sārvakālikam abhyanujñānam // (27.1)
	yāgahetunā prathamavarṇasya yathāvācanikābhyanujñānam // (28.1)
	khaṇḍamaṇḍalādhipatyādīnām indirāvatāṃ janānām api sīdhugrahaṇasya vilāsaḥ sārvakālikaḥ // (29.1)
	svakīyapriyasāhacaryeṇa yuvatīnām abhyanujñānam // (30.1)
	tatrāpy ananyajamakheṣv eva netaratra vidhānam // (31.1)
	yoṣāyāḥ āsyapadmena prāśanaṃ bhāṣāyāḥ prabodhe avyabhicaritakāraṇam // (32.1)
	śyāmāyā āsyasahasrapattrād anuprāśanam aparokṣānubhavena mūlaprakṛteḥ svarūpasyānudarśanam // (33.1)
