	vṛṣabhaṃ vṛṣabhaṃ vande vṛṣabhāṅkaṃ vṛṣārcitam / (2.1)

	parameṣṭhipadāptānām parameṣṭhipadāptaye / (3.1)
	parameṣṭhipadau vande satyaṃ ca parameṣṭhinam // (3.2)
	ārhatī bhāratī pūjyā lokālokapradīpikā / (4.1)
	rajo vidhūya no nityaṃ tanotu vimalām matim // (4.2)
	sveṣṭārthasiddhikaraṇāś caraṇāḥ santu gauravāḥ / (5.1)
	gauravāptāḥ sucaraṇais taraṇair me bhavāmbudheḥ // (5.2)
	śaktyā hīno 'pi vakṣye 'haṃ gurubhaktyā praṇoditaḥ / (6.1)
	śrībhadrabāhucaritam yathā jñātaṃ gurūktitaḥ // (6.2)
	yacchrutaṃ mugdhabuddhīnām mithyāmohamahātamaḥ / (7.1)
	dhunute tanute śuddhajainamārge 'malāṃ matim // (7.2)
	athātra bhārate varṣe viṣaye magadhābhidhe / (8.1)
	puraṃ rājagṛhaṃ bhāti puraṃdarapuropamam // (8.2)
	natāśeṣanṛpaśreṇiḥ śreṇikaḥ śreyasāṃ nidhiḥ / (9.1)
	bhāvukaḥ pālakas tasya celanī mahiṣīśitā // (9.2)
	ekadāsau viśāṃ nātho viditvā vanapālataḥ / (10.1)
	vipulādrau mahāvīrasamavasṛtim āgatām // (10.2)
	parānandathum āpanno 'caladdevaṃ vivandiṣuḥ / (11.1)
	tauryatrikavarārāvabadhirīkṛtadiṅmukhaḥ // (11.2)
	nirīkṣya surasaṃsevyaṃ kevalojjvalarociṣam / (12.1)
	nutvā natvā samabhyarcya tasthivān narasaṃsidi // (12.2)
	dvidhādharmaṃ jinodgītam aśrāvīt praśrayānvitaḥ / (13.1)
	praṇipatya tato 'prākṣīt karau mukulayan nṛpaḥ // (13.2)
	devātra duḥṣame kāle kevalaśrutabodhakaḥ / (14.1)
	kiyanto 'gre bhaviṣyanti kiṃ kiṃ cāto bhaviṣyati // (14.2)

	mayi muktim ite rājan gautamākhyaḥ sudharmavāk / (16.1)
