ataḥ paraṃ gṛhasthasya karmācāraṃ kalau yuge / (1.1)
	dharmaṃ sādhāraṇaṃ śaktyā cāturvarṇyāśramāgatam // (1.2)
	taṃ pravakṣyāmy ahaṃ pūrvaṃ parāśaravaco yathā / (2.1)
	ṣaṭkarmasahito vipraḥ kṛṣikarma ca kārayet // (2.2)
	kṣudhitaṃ tṛṣitaṃ śrāntaṃ balīvardaṃ na yojayet / (3.1)
	hīnāṅgaṃ vyādhitaṃ klībaṃ vṛṣaṃ vipro na vāhayet // (3.2)
	sthirāṅgaṃ nīrujaṃ tṛptaṃ sunardaṃ ṣaṇḍhavarjitam / (4.1)
	vāhayed divasasyārdhaṃ paścāt snānaṃ samācaret // (4.2)
	japyaṃ devārcanaṃ homaṃ svādhyāyaṃ caivam abhyaset / (5.1)
	ekadvitricaturviprān bhojayet snātakān dvijaḥ // (5.2)
	svayaṃkṛṣṭe tathā kṣetre dhānyaiś ca svayamarjitaiḥ / (6.1)
	nirvapet pañca yajñāṃś ca kratudīkṣāṃ ca kārayet // (6.2)
	tilā rasā na vikreyā vikreyā dhānyatatsamāḥ / (7.1)
	viprasyaivaṃvidhā vṛttis tṛṇakāṣṭhādivikrayaḥ // (7.2)
	brāhmaṇaś cet kṛṣiṃ kuryāt tanmahādoṣam āpnuyāt / (8.1)
	saṃvatsareṇa yat pāpaṃ matsyaghātī samāpnuyāt // (8.2)
	ayomukhena kāṣṭhena tad ekāhena lāṅgalī / (9.1)
	pāśako matsyaghātī ca vyādhaḥ śākunikas tathā // (9.2)
	adātā karṣakaś caiva sarve te samabhāginaḥ / (10.1)
	vṛkṣaṃ chittvā mahīṃ bhittvā hatvā ca kṛmikīṭakān // (10.2)
	karṣakaḥ khalayajñena sarvapāpaiḥ pramucyate / (11.1)
	yo na dadyād dvijātibhyo rāśimūlam upāgataḥ // (11.2)
	sa cauraḥ sa ca pāpiṣṭho brahmaghnaṃ taṃ vinirdiśet / (12.1)
	rājñe dattvā tu ṣaḍbhāgaṃ devānāṃ caikaviṃśakam // (12.2)
	viprāṇāṃ triṃśakaṃ bhāgaṃ sarvapāpaiḥ pramucyate / (13.1)
	kṣatriyo 'pi kṛṣiṃ kṛtvā devān viprāṃś ca pūjayet // (13.2)
	vaiśyaḥ śūdras tathā kuryāt kṛṣivāṇijyaśilpakam / (14.1)
	vikarma kurvate śūdrā dvijaśuśrūṣayojjhitāḥ // (14.2)
	bhavanty alpāyuṣas te vai nirayaṃ yānty asaṃśayaṃ / (15.1)
	caturṇām api varṇānām eṣa dharmaḥ sanātanaḥ // (15.2)
