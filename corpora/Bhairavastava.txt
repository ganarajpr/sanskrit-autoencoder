	bhairavanātham anāthaśaraṇyaṃ tvanmayacittatayā hṛdi vande // (1.2)
	tvanmayam etad aśeṣam idānīṃ bhāti mama tvadanugrahaśaktyā / (2.1)
	tvaṃ ca maheśa sadaiva mamātmā svātmamayaṃ mama tena samastam // (2.2)
	svātmani viśvagaye tvayi nāthe tena na saṃsṛṣṭibhītikathāsti / (3.1)
	satsv api durdharaduḥkhavimohatrāsavidhāyiṣu karmagaṇeṣu // (3.2)
	antaka māṃ prati mā dṛśam enaṃ krodhakarālatamāṃ vinidhehi / (4.1)
	śaṅkarasevanacintanadhīro bhīṣaṇabhairavaśaktimayo 'smi // (4.2)
	ittham upoḍhabhavanmayasaṃviddīdhitidāritabhūritamisraḥ / (5.1)
	mṛtyuyamāntakakarmapiśācair nātha namo 'stu na jātu bibhemi // (5.2)
	proditasatyavibodhamarīciprekṣitaviśvapadārthasatattvaḥ / (6.1)
	bhāvaparāmṛtanirbharapūrṇe tvayy aham ātmani nirvṛtim emi // (6.2)
	mānasagocaram eti yadaiva kleśadaśā tanutāpavidhātrī / (7.1)
	nātha tadaiva mama tvadabhedastotraparāmṛtavṛṣṭir udeti // (7.2)
	śaṅkara satyam idaṃ vratadānasnānatapo bhavatāpavidāri / (8.1)
	tāvakaśāstraparāmṛtacintā syandati cetasi nirvṛtidhārām // (8.2)
	nṛtyati gāyati hṛṣyati gāḍhaṃ saṃvid iyaṃ mama bhairavanātha / (9.1)
	tvāṃ priyam āpya sudarśanam ekam durlabham anyajanaiḥ samayajñam // (9.2)
	vasurasapauṣe kṛṣṇadaśamyām abhinavaguptaḥ stavam imam akarot / (10.1)
	yena vibhur bhavamarusantāpaṃ śamayati janasya jhaṭiti dayāluḥ // (10.2)
	samāptaṃ stavam idam abhinavākhyaṃ padyanavakam // (11.1)
