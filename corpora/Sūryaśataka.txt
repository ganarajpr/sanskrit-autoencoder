jambhārātībhakumbhodbhavamiva dadhataḥ sāndrasindūrareṇuṃ raktāḥ siktā ivaughairudayagiritaṭīdhātudhārādravasya / (1.1)
	āyāntyā tulyakālaṃ kamalavanarucevāruṇā vo vibhūtyai bhūyāsurbhāsayanto bhuvanamabhinavā bhānavo bhānavīyāḥ // (1.2)
	bhaktiprahvāya dātuṃ mukulapuṭakuṭīkoṭarakroḍalīnāṃ lakṣmīm ākraṣṭukāmā iva kamalavanodghāṭanaṃ kurvate ye / (2.1)
	kālākārāndhakārānanapatitajagatsādhvasadhvaṃsakalyāḥ kalyāṇaṃ vaḥ kriyāsuḥ kisalayarucayaste karā bhāskarasya // (2.2)
	garbheṣvambhoruhāṇāṃ śikhariṣu ca śitāgreṣu tulyaṃ patantaḥ prārambhe vāsarasya vyuparatisamaye caikarūpāstathaiva / (3.1)
	niṣparyāyaṃ pravṛttāstribhuvanabhavanaprāṅgaṇe pāntu yuṣmān ūṣmāṇaṃ saṃtatādhvaśramajamiva bhṛśaṃ bibhrato bradhnapādāḥ // (3.2)
	prabhraśyatyuttarīyatviṣi tamasi samudvīkṣya vītāvṛtīnprāgjantūṃstantūn yathā yānatanu vitanute tigmarocirmarīcīn / (4.1)
	te sāndrībhūya sadyaḥ kramaviśadadaśāśādaśālīviśālaṃ śaśvat sampādayanto 'mbaram amalam alaṃ maṅgalaṃ vo diśantu // (4.2)
	nyakkurvannoṣadhīśe muṣitaruci śucevauṣadhīḥ proṣitābhā bhāsvadgrāvodgatena prathamamiva kṛtābhyudgatiḥ pāvakena / (5.1)
	pakṣacchedavraṇāsṛksruta iva dṛṣado darśayanprātaradrer ātāmrastīvrabhānor anabhimatanude stādgabhastyudgamo vaḥ // (5.2)
	śīrṇaghrāṇāṅghripāṇīn vraṇibhir apaghanair ghargharāvyaktaghoṣān dīrghāghrātān aghaughaiḥ punarapi ghaṭayatyeka ullāghayan yaḥ / (6.1)
	gharmāṃśor yasya vo'ntardviguṇaghanaghṛṇānighnanirvighnavṛtter dattārghāḥ siddhasaṃghair vidadhatu ghṛṇayaḥ śīghramaṃhovighātam // (6.2)
	bibhrāṇā vāmanatvaṃ prathamamatha tathaivāṃśavaḥ prāṃśavo vaḥ krāntākāśāntarālāstadanu daśadiśaḥ pūrayantastato'pi / (7.1)
	dhvāntād ācchidya devadviṣa iva balito viśvamāśvaśnuvānāḥ kṛcchrāṇyucchrāyahelopahasitaharayo hāridaśvā harantu // (7.2)
	udgāḍhenāruṇimnā vidadhati bahulaṃ ye'ruṇasyāruṇatvaṃ mūrdhodbhūtau khalīnakṣatarudhiraruco ye rathāśvānaneṣu / (8.1)
	śailānāṃ śekharatvaṃ śritaśikhariśikhāstanvate ye diśantu preṅkhantaḥ khe kharāṃśoḥ khacitadinamukhāste mayūkhāḥ sukhaṃ vaḥ // (8.2)
	dattānandāḥ prajānāṃ samucitasamayākṛṣṭasṛṣṭaiḥ payobhiḥ pūrvāhṇe viprakīrṇā diśi diśi viramatyahni saṃhārabhājaḥ / (9.1)
	dīptāṃśor dīrghaduḥkhaprabhavabhavabhayodanvaduttāranāvo gāvo vaḥ pāvanānāṃ paramaparimitāṃ prītimutpādayantu // (9.2)
	bandhadhvaṃsaikahetuṃ śirasi natirasābaddhasaṃdhyāñjalīnāṃ lokānāṃ ye prabodhaṃ vidadhati vipulāmbhojakhaṇḍāśayeva / (10.1)
	yuṣmākaṃ te svacittaprathitapṛthutaraprārthanākalpavṛkṣāḥ kalpantāṃ nirvikalpaṃ dinakarakiraṇāḥ ketavaḥ kalmaṣasya // (10.2)
	dhārā rāyo dhanāyāpadi sapadi karālambabhūtāḥ prapāte tattvālokaikadīpāstridaśapatipuraprasthitau vīthya eva / (11.1)
	nirvāṇodyogiyogipragamanijatanudvāri vetrāyamāṇās trāyantāṃ tīvrabhānor divasamukhasukhā raśmayaḥ kalmaṣādvaḥ // (11.2)
	prāci prāgācarantyo 'naticiram acale cārucūḍāmaṇitvaṃ muñcantyo rocanāmbhaḥ pracuramiva diśāmuccakaiścarcanāya / (12.1)
	cāṭūtkaiś cakranāmnāṃ caturamavicalairlocanair arcyamānāś ceṣṭantāṃ cintitānām ucitam acaramāś caṇḍarocīruco vaḥ // (12.2)
	ekaṃ jyotirdṛśau dve trijagati gaditānyabjajāsyaiścaturbhir bhūtānāṃ pañcamaṃ yānyalamṛtuṣu tathā ṣaṭsu nānāvidhāni / (13.1)
	yuṣmākaṃ tāni saptatridaśamuninutāny aṣṭadigbhāñji bhānor yānti prāhṇe navatvaṃ daśa dadhatu śivaṃ dīdhitīnāṃ śatāni // (13.2)
	āvṛttibhrāntaviśvāḥ śramamiva dadhataḥ śoṣiṇaḥ svoṣmaṇeva grīṣme dāvāgnitaptā iva rasamasakṛdye dharitryā dhayanti / (14.1)
	te prāvṛṣyāttapānātiśayaruja ivodvāntatoyā himartau mārtaṇḍasyāpracaṇḍāściramaśubhabhide 'bhīśavo vo bhavantu // (14.2)
	tanvānā digvadhūnāṃ samadhikamadhurālokaramyām avasthām ārūḍhaprauḍhileśotkalitakapilimālaṃkṛtiḥ kevalaiva / (15.1)
	ujjṛmbhāmbhojanetradyutini dinamukhe kiṃcidudbhidyamānā śmaśruśreṇīva bhāsāṃ diśatu daśaśatī śarma gharmatviṣo vaḥ // (15.2)
	maulīndormaiṣa moṣīd dyutim iti vṛṣabhāṅkena yaḥ śaṅkineva pratyagrodghāṭitāmbhoruhakuharaguhāsusthiteneva dhātrā / (16.1)
	kṛṣṇena dhvāntakṛṣṇasvatanuparibhavatrasnuneva stuto'laṃ trāṇāya stāttanīyānapi timiraripoḥ sa tviṣāmudgamo vaḥ // (16.2)
	vistīrṇaṃ vyoma dīrghāḥ sapadi daśa diśo vyastavelāmbhaso 'bdhīn kurvadbhir dṛśyanānānaganagaranagābhogapṛthvīṃ ca pṛthvīm / (17.1)
	padminyucchvāsyate yairuṣasi jagadapi dhvaṃsayitvā tamisrām usrā visraṃsayantu drutamanabhimataṃ te sahasratviṣo vaḥ // (17.2)
