pracaṇḍasūryaḥ spṛhaṇīyacandramāḥ sadāvagāhakṣatavārisaṃcayaḥ / (1.1)
	dināntaramyo 'bhyupaśāntamanmatho nidāghakālo'yamupāgataḥ priye // (1.2)
	niśāḥ śaśāṅkakṣatanīlarājayaḥ kvacidvicitraṃ jalayantramandiram / (2.1)
	maṇiprakārāḥ sarasaṃ ca candanaṃ śucau priye yānti janasya sevyatām // (2.2)
	suvāsitaṃ harmyatalaṃ manoharaṃ priyāmukhocchvāsavikampitaṃ madhu / (3.1)
	sutantrigītaṃ madanasya dīpanaṃ śucau niśīthe'nubhavanti kāminaḥ // (3.2)
	nitambabimbaiḥ sadukūlamekhalaiḥ stanaiḥ sahārābharaṇaiḥ sacandanaiḥ / (4.1)
	śiroruhaiḥ snānakaṣāyavāsitaiḥ striyo nidāghaṃ śamayanti kāminām // (4.2)
	nitāntalākṣārasarāgarañjitair nitambinīnāṃ caraṇaiḥ sanūpuraiḥ / (5.1)
	pade pade haṃsarutānukāribhirjanasya cittaṃ kriyate samanmatham // (5.2)
	payodharāścandanapaṅkacarcitās tuṣāragaurārpitahāraśekharāḥ / (6.1)
	nitambadeśāśca sahemamekhalāḥ prakurvate kasya mano na sotsukam // (6.2)
	samudgatasvedacitāṅgasaṃdhayo vimucya vāsāṃsi gurūṇi sāmpratam / (7.1)
	staneṣu tanvaṃśukamunnatastanā niveśayanti pramadāḥ sayauvanāḥ // (7.2)
	sacandanāmbuvyajanodbhavānilaiḥ sahārayaṣṭistanamaṇḍalārpaṇaiḥ / (8.1)
	savallakīkākaligītanisvanair vibodhyate supta ivādya manmathaḥ // (8.2)
	siteṣu harmyeṣu niśāsu yoṣitāṃ sukhaprasuptāni mukhāni candramāḥ / (9.1)
	vilokya nūnaṃ bhṛśam utsukaściraṃ niśākṣaye yāti hriyeva pāṇḍutām // (9.2)
	asahyavātoddhatareṇumaṇḍalā pracaṇḍasūryātapatāpitā mahī / (10.1)
	na śakyate draṣṭumapi pravāsibhiḥ priyāviyogānaladagdhamānasaiḥ // (10.2)
	mṛgāḥ pracaṇḍātapatāpitā bhṛśaṃ tṛṣā mahatyā pariśuṣkatālavaḥ / (11.1)
	vanāntare toyamiti pradhāvitā nirīkṣya bhinnāñjanasaṃnibhaṃ nabhaḥ // (11.2)
	savibhramaiḥ sasmitajihmavīkṣitair vilāsavatyo manasi pravāsinām / (12.1)
	anaṅgasaṃdīpanamāśu kurvate yathā pradoṣāḥ śaśicārubhūṣaṇāḥ // (12.2)
	ravermayūkhair abhitāpito bhṛśaṃ vidahyamānaḥ pathi taptapāṃsubhiḥ / (13.1)
	avāṅmukho jihmagatiḥ śvasanmuhuḥ phaṇī mayūrasya tale niṣīdati // (13.2)
	tṛṣā mahatyā hatavikramodyamaḥ śvasanmuhur dūravidāritānanaḥ / (14.1)
	na hantyadūre'pi gajānmṛgeśvaro vilolajihvaścalitāgrakesaraḥ // (14.2)
	viśuṣkakaṇṭhodgatasīkarāmbhaso gabhastibhir bhānumato 'nutāpitāḥ / (15.1)
	pravṛddhatṛṣṇopahatā jalārthino na dantinaḥ kesariṇo'pi bibhyati // (15.2)
	hutāgnikalpaiḥ saviturgabhastibhiḥ kalāpinaḥ klāntaśarīracetasaḥ / (16.1)
	na bhoginaṃ ghnanti samīpavartinaṃ kalāpacakreṣu niveśitānanam // (16.2)
	sabhadramustaṃ pariśuṣkakardamaṃ saraḥ khanann āyatapotṛmaṇḍalaiḥ / (17.1)
	ravermayūkhairabhitāpito bhṛśaṃ varāhayūtho viśatīva bhūtalam // (17.2)
	vivasvatā tīkṣṇatarāṃśumālinā sapaṅkatoyāt saraso 'bhitāpitaḥ / (18.1)
	utplutya bhekastṛṣitasya bhoginaḥ phaṇātapatrasya tale niṣīdati // (18.2)
	samuddhṛtāśeṣamṛṇālajālakaṃ vipannamīnaṃ drutabhītasārasam / (19.1)
	parasparotpīḍanasaṃhatairgajaiḥ kṛtaṃ saraḥ sāndravimardakardamam // (19.2)
	raviprabhodbhinnaśiromaṇiprabho vilolajihvādvayalīḍhamārutaḥ / (20.1)
	viṣāgnisūryātapatāpitaḥ phaṇī na hanti maṇḍūkakulaṃ tṛṣākulaḥ // (20.2)
	saphenalālāvṛtavaktrasampuṭaṃ viniḥsṛtālohitajihvamunmukham / (21.1)
	tṛṣākulaṃ niḥsṛtamadrigahvarādavekṣamāṇaṃ mahiṣīkulaṃ jalam // (21.2)
	paṭutaradavadāhocchuṣkasasyaprarohāḥ paruṣapavanavegotkṣiptasaṃśuṣkaparṇāḥ / (22.1)
	dinakaraparitāpakṣīṇatoyāḥ samantādvidadhati bhayamuccair vīkṣyamāṇā vanāntāḥ // (22.2)
	śvasiti vihagavargaḥ śīrṇaparṇadrumasthaḥ kapikulamupayāti klāntamadrer nikuñjam / (23.1)
	bhramati gavayayūthaḥ sarvatastoyamicchañśarabhakulamajihmaṃ proddharatyambu kūpāt // (23.2)
	vikacanavakusumbhasvacchasindūrabhāsā prabalapavanavegodbhūtavegena tūrṇam / (24.1)
	taṭaviṭapalatāgrāliṅganavyākulena diśi diśi paridagdhā bhūmayaḥ pāvakena // (24.2)
	jvalati pavanavṛddhaḥ parvatānāṃ darīṣu sphuṭati paṭuninādaiḥ śuṣkavaṃśasthalīṣu / (25.1)
	prasarati tṛṇamadhye labdhavṛddhiḥ kṣaṇena glapayati mṛgavargaṃ prāntalagno davāgniḥ // (25.2)
	bahutara iva jātaḥ śālmalīnāṃ vaneṣu sphurati kanakagauraḥ koṭareṣu drumāṇām / (26.1)
	pariṇatadalaśākhānutpatanprāṃśuvṛkṣānbhramati pavanadhūtaḥ sarvato'gnirvanānte // (26.2)
	gajagavayamṛgendrā vahnisaṃtaptadehāḥ suhṛda iva sametā dvaṃdvabhāvaṃ vihāya / (27.1)
	hutavahaparikhedād āśu nirgatya kakṣādvipulapulinadeśāṃ nimnagāṃ saṃviśanti // (27.2)
	kamalavanacitāmbuḥ pāṭalāmodaramyaḥ sukhasalilaniṣekaḥ sevyacandrāṃśuhāraḥ / (28.1)
	vrajatu tava nidāghaḥ kāminībhiḥ sameto niśi sulalitagīte harmyapṛṣṭhe sukhena // (28.2)
