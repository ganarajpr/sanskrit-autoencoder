vimalakalāśrayābhinavasṛṣṭimahā jananī bharitatanuś ca pañcamukhaguptarucir janakaḥ / (1.1)
	tadubhayayāmalasphuritabhāvavisargamayaṃ hṛdayam anuttarāmṛtakulaṃ mama saṃsphuratāt // (1.2)
	vitatas tantrāloko vigāhituṃ naiva śakyate sarvaiḥ / (2.1)
	ṛjuvacanaviracitam idaṃ tu tantrasāraṃ tataḥ śṛṇuta // (2.2)
	śrīśambhunāthabhāskaracaraṇanipātaprabhāpagatasaṃkocam / (3.1)
	abhinavaguptahṛdambujam etad vicinuta maheśapūjanahetoḥ // (3.2)
	iha jñānaṃ mokṣakāraṇaṃ bandhanimittasya ajñānasya virodhakatvāt dvividhaṃ ca ajñānaṃ buddhigataṃ pauruṣaṃ ca tatra buddhigatam aniścayasvabhāvaṃ viparītaniścayātmakaṃ ca // (4.0)
	pauruṣaṃ tu vikalpasvabhāvaṃ saṃkucitaprathātmakaṃ tad eva ca mūlakāraṇaṃ saṃsārasya iti vakṣyāmo malanirṇaye // (5.0)
	tatra pauruṣam ajñānaṃ dīkṣādinā nivartetāpi kiṃ tu dīkṣāpi buddhigate anadhyavasāyātmake ajñāne sati na sambhavati heyopādeyaniścayapūrvakatvāt tattvaśuddhiśivayojanārūpāyā dīkṣāyā iti // (6.0)
	tatra adhyavasāyātmakaṃ buddhiniṣṭham eva jñānaṃ pradhānam tad eva ca abhyasyamānaṃ pauruṣam api ajñānaṃ nihanti vikalpasaṃvidabhyāsasya avikalpāntatāparyavasānāt // (7.0)
	vikalpāsaṃkucitasaṃvitprakāśarūpo hy ātmā śivasvabhāva iti sarvathā samastavastuniṣṭhaṃ samyaṅniścayātmakaṃ jñānam upādeyam // (8.0)
	tac ca śāstrapūrvakam // (9.0)
	śāstraṃ ca parameśvarabhāṣitam eva pramāṇam // (10.0)
	aparaśāstroktānām arthānāṃ tatra vaiviktyena abhyupagamāt tadarthātiriktayuktisiddhanirūpaṇāc ca tena aparāgamoktaṃ jñānaṃ tāvata eva bandhāt vimocakam na sarvasmāt sarvasmāt tu vimocakaṃ parameśvaraśāstraṃ pañcasrotomayaṃ daśāṣṭādaśavasvaṣṭabhedabhinnam // (11.0)
	tato 'pi sarvasmāt sāraṃ ṣaḍardhaśāstrāṇi // (12.0)
	tebhyo 'pi mālinīvijayam // (13.0)
	tadantargataś cārthaḥ saṃkalayyāśakyo nirūpayitum // (14.0)
	na ca anirūpitavastutattvasya muktatvaṃ mocakatvaṃ vā śuddhasya jñānasyaiva tathārūpatvāt iti // (15.0)
	svabhyastajñānamūlatvāt parapuruṣārthasya tatsiddhaye idam ārabhyate // (16.0)
	ajñānaṃ kila bandhahetur uditaḥ śāstre malaṃ tat smṛtaṃ pūrṇajñānakalodaye tad akhilaṃ nirmūlatāṃ gacchati // (17.0)
	dhvastāśeṣamalātmasaṃvidudaye mokṣaś ca tenāmunā śāstreṇa prakaṭīkaromi nikhilaṃ yaj jñeyatattvaṃ bhavet // (18.0)
	tatra iha svabhāva eva paramopādeyaḥ sa ca sarvabhāvānāṃ prakāśarūpa eva aprakāśasya svabhāvatānupapatteḥ sa ca nānekaḥ prakāśasya taditarasvabhāvānupraveśāyoge svabhāvabhedābhāvāt deśakālāv api ca asya na bhedakau tayor api tatprakāśasvabhāvatvāt iti eka eva prakāśaḥ sa eva ca saṃvit arthaprakāśarūpā hi saṃvit iti sarveṣām atra avivāda eva // (19.0)
	sa ca prakāśo na paratantraḥ prakāśyataiva hi pāratantryam prakāśyatā ca prakāśāntarāpekṣitaiva na ca prakāśāntaraṃ kiṃcit asti iti svatantra ekaḥ prakāśaḥ svātantryād eva ca deśakālākārāvacchedavirahāt vyāpako nityaḥ sarvākāranirākārasvabhāvaḥ tasya ca svātantryam ānandaśaktiḥ taccamatkāra icchāśaktiḥ prakāśarūpatā cicchaktiḥ āmarśātmakatā jñānaśaktiḥ sarvākārayogitvaṃ kriyāśaktiḥ ity evaṃ mukhyābhiḥ śaktibhiḥ yukto 'pi vastuta icchājñānakriyāśaktiyuktaḥ anavacchinnaḥ prakāśo nijānandaviśrāntaḥ śivarūpaḥ sa eva svātantryāt ātmānaṃ saṃkucitam avabhāsayan aṇur iti ucyate // (20.0)
	punar api ca svātmānaṃ svatantratayā prakāśayati yena anavacchinnaprakāśaśivarūpatayaiva prakāśate // (21.0)
	tatrāpi svātantryavaśāt anupāyam eva svātmānaṃ prakāśayati sopāyaṃ vā sopāyatve 'pi icchā vā jñānaṃ vā kriyā vā abhyupāya iti traividhyaṃ śāmbhavaśāktāṇavabhedena samāveśasya tatra caturvidham api etad rūpaṃ krameṇa atra upadiśyate // (22.0)
	ātmā prakāśavapur eṣa śivaḥ svatantraḥ svātantryanarmarabhasena nijaṃ svarūpam / (23.1)
	saṃchādya yat punar api prathayeta pūrṇaṃ tac ca kramākramavaśād athavā tribhedāt // (23.2)
