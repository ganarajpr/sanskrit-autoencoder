	om namo buddhāya / (1.1)
	yaḥ sarvathā sarvahatāndhakāraḥ saṃsārapaṅkājjagadujjahāra / (1.2)
	tasmai namaskṛtya yathārthaśāstre śāstraṃ pravakṣyāmyabhidharmakośam // (1.3)
	prajñāmalā sānucarābhidharmaḥ tatprāptaye yāpi ca yacca śāstram / (2.1)
	tasyārthato'smin samanupraveśāt sa cāśrayo 'syetyabhidharmakośam // (2.2)
	dharmāṇāṃ pravicayamantareṇa nāsti kleśānāṃ yata upaśāntaye'bhyupāyaḥ / (3.1)
	kleśaiśca bhramati bhavārṇave'tra lokastaddhetorata uditaḥ kilaiṣa śāstrā // (3.2)
	sāsravānāsravā dharmāḥ saṃskṛtā mārgavarjitāḥ / (4.1)
	sāsravāḥ āsravāsteṣu yasmāt samanuśerate // (4.2)
	anāsravā mārgasatyaṃ trividhaṃ cāpyasaṃskṛtam / (5.1)
	ākāśaṃ dvau nirodhau ca tatrākāśam anāvṛtiḥ // (5.2)
	pratisaṃkhyānirodho yo visaṃyogaḥ pṛthak pṛthak / (6.1)
	utpādātyantavighno'nyo nirodho'pratisaṃkhyayā // (6.2)
	te punaḥ saṃskṛtā dharmā rūpādiskandhapañcakam / (7.1)
	ta evādhvā kathāvastu saniḥsārāḥ savastukāḥ // (7.2)
	ye sāsravā upādānaskandhāste saraṇā api / (8.1)
	duḥkhaṃ samudayo loko dṛṣṭisthānaṃ bhavaśca te // (8.2)
	rūpaṃ pañcendriyāṇyarthāḥ pañcāvijñaptireva ca / (9.1)
	tadvijñānāśrayā rūpaprasādāścakṣurādayaḥ // (9.2)
	rūpaṃ dvidhā viṃśatidhā śabdastvaṣṭavidhaḥ rasaḥ / (10.1)
	ṣoḍhā caturvidho gandhaḥ spṛśyamekādaśātmakam // (10.2)
	vikṣiptācittakasyāpi yo'nubandhaḥ śubhāśubhaḥ / (11.1)
	mahābhūtānyupādāya sa hyavijñaptirucyate // (11.2)
	bhūtāni pṛthividhāturaptejovāyudhātavaḥ / (12.1)
	dhṛtyādikarmasaṃsiddhāḥ kharasnehoṣṇateraṇāḥ // (12.2)
	pṛthivī varṇasaṃsthānamucyate lokasaṃjñayā / (13.1)
	āpastejaśca vāyustu dhātureva tathāpi ca // (13.2)
	indriyārthāsta eveṣṭā daśāyatanadhātavaḥ / (14.1)
	vedanānubhavaḥ saṃjñā nimittodgrahaṇātmikā // (14.2)
	caturbhyo'nye tu saṃskāraskandhaḥ ete punastrayaḥ / (15.1)
	dharmāyatanadhātvākhyāḥ sahāvijñaptyasaṃskṛtaiḥ // (15.2)
	vijñānaṃ prativijñaptiḥ mana āyatanaṃ ca tat / (16.1)
	dhātavaḥ sapta ca matāḥ ṣaḍ vijñānānyatho manaḥ // (16.2)
	ṣaṇṇām anantarātītaṃ vijñānaṃ yaddhi tanmanaḥ / (17.1)
	ṣaṣṭhāśrayaprasiddhyarthaṃ dhātavo 'ṣṭādaśa smṛtāḥ // (17.2)
	sarvasaṃgraha ekena skandhenāyatanena ca / (18.1)
	dhātunā ca svabhāvena parabhāvaviyogataḥ // (18.2)
	jātigocaravijñānasāmānyādekadhātutā / (19.1)
	dvitve'pi cakṣurādīnāṃ śobhārthaṃ tu dvayodbhavaḥ // (19.2)
	rāśyāyadvāragotrārthāḥ skandhāyatanadhātavaḥ / (20.1)
	mohendriyarucitraidhāt tisraḥ skandhādideśanāḥ // (20.2)
	vivādamūlasaṃsārahetutvāt kramakāraṇāt / (21.1)
	caittebhyo vedanāsaṃjñe pṛthakskandhau niveśitau // (21.2)
	skandheṣvasaṃskṛtaṃ noktamarthāyogāt kramaḥ punaḥ / (22.1)
	yathaudārikasaṃkleśabhājanādyarthadhātutaḥ // (22.2)
	prāk pañca vārttamānārthyāt bhautikārthyāccatuṣṭayam / (23.1)
	dūrāśutaravṛttyānyat yathāsthānaṃ kramo'thavā // (23.2)
	viśeṣaṇārthaṃ prādhānyād bahudharmāgrasaṃgrahāt / (24.1)
	ekamāyatanaṃ rūpamekaṃ dharmākhyamucyate // (24.2)
	dharmaskandhasahasrāṇi yānyaśītiṃ jagau muniḥ / (25.1)
	tāni vāṅnāma vetyeṣāṃ rūpasaṃskārasaṃgrahaḥ // (25.2)
	śāstrapramāṇā ityeke skandhādīnāṃ kathaikaśaḥ / (26.1)
	caritapratipakṣastu dharmaskandho'nuvarṇitaḥ // (26.2)
	tathānye'pi yathāyogaṃ skandhāyatanadhātavaḥ / (27.1)
	pratipādyā yathokteṣu sampradhārya svalakṣaṇam // (27.2)
	chidramākāśadhātvākhyam ālokatamasī kila / (28.1)
	vijñānadhāturvijñānaṃ sāsravaṃ janmaniśrayāḥ // (28.2)
	sanidarśana eko'tra rūpaṃ sapratighā daśa / (29.1)
	rūpiṇaḥ avyākṛtā aṣṭau ta evārūpaśabdakāḥ // (29.2)
	tridhānye kāmadhātvāptāḥ sarve rūpe caturdaśa / (30.1)
	vinā gandharasaghrāṇajihvāvijñānadhātubhiḥ // (30.2)
	ārūpyāptā manodharmamanovijñānadhātavaḥ / (31.1)
	sāsravānāsravā ete trayaḥ śeṣāstu sāsravāḥ // (31.2)
	savitarkavicārā hi pañca vijñānadhātavaḥ / (32.1)
	antyāstrayastriprakārāḥ śeṣā ubhayavarjitāḥ // (32.2)
	nirūpaṇānusmaraṇavikalpenāvikalpakāḥ / (33.1)
	tau prajñā mānasī vyagrā smṛtiḥ sarvaiva mānasī // (33.2)
	sapta sālambanāścittadhātavaḥ ardhaṃ ca dharmataḥ / (34.1)
	navānupāttās te cāṣṭau śabdaśca anye nava dvidhā // (34.2)
	spraṣṭavyaṃ dvividhaṃ śeṣā rūpiṇo nava bhautikāḥ / (35.1)
	dharmadhātvekadeśaśca saṃcitā daśa rūpiṇaḥ // (35.2)
	chinatti chidyate caiva bāhyaṃ dhātucatuṣṭayam / (36.1)
	dahyate tulayatyevaṃ vivādo dagdhṛtulyayoḥ // (36.2)
	vipākajaupacayikāḥ pañcādhyātmaṃ vipākajaḥ / (37.1)
	na śabdaḥ apratighā aṣṭau naiḥṣyandikavipākajāḥ // (37.2)
	tridhānye dravyavānekaḥ kṣaṇikāḥ paścimāstrayaḥ / (38.1)
	cakṣurvijñānadhātvoḥ syāt pṛthak lābhaḥ sahāpi ca // (38.2)
	dvādaśādhyātmikāḥ hitvā rūpādīn dharmasaṃjñakaḥ / (39.1)
	sabhāgaḥ tatsabhāgāśca śeṣāḥ yo na svakarmakṛt // (39.2)
	daśa bhāvanayā heyāḥ pañca ca antyāstrayastridhā / (40.1)
	na dṛṣṭiheyamakliṣṭaṃ na rūpaṃ nāpyaṣaṣṭhajam // (40.2)
	cakṣuśca dharmadhātośca pradeśo dṛṣṭiḥ aṣṭadhā / (41.1)
	pañcavijñānasahajā dhīrna dṛṣṭiratīraṇāt // (41.2)
	cakṣuḥ paśyati rūpāṇi sabhāgaṃ na tadāśritam / (42.1)
	vijñānaṃ dṛśyate rūpaṃ na kilāntaritaṃ yataḥ // (42.2)
	ubhābhyāmapi cakṣurbhyāṃ paśyati vyaktadarśanāt / (43.1)
	cakṣuḥśrotramano'prāptaviṣayaṃ trayamanyathā // (43.2)
	tribhirghrāṇādibhistulyaviṣayagrahaṇaṃ matam / (44.1)
	caramasyāśrayo'tītaḥ pañcānāṃ sahajaśca taiḥ // (44.2)
	tadvikāravikāritvād āśrayāś cakṣurādayaḥ / (45.1)
	ato'sādhāraṇatvāddhi vijñānaṃ tairnirucyate // (45.2)
	na kāyasyādharaṃ cakṣuḥ ūrdhvaṃ rūpaṃ na cakṣuṣaḥ / (46.1)
	vijñānaṃ ca asya rūpaṃ tu kāyasyobhe ca sarvataḥ // (46.2)
	tathā śrotraṃ trayāṇāṃ tu sarvameva svabhūmikam / (47.1)
	kāyavijñānamadharasvabhūmi aniyataṃ manaḥ // (47.2)
	pañca bāhyā dvivijñeyāḥ nityā dharmā asaṃskṛtāḥ / (48.1)
	dharmārdhamindriyaṃ ye ca dvādaśādhyātmikāḥ smṛtāḥ // (48.2)
	abhidharmakośabhāṣye dhātunirdeśo nāma prathamaṃ kośasthānaṃ samāptamiti / (49.1)
