	itarattu prāsaṅgikam aupodghātikaṃ coktam // (2.1)
	ihedānīṃ taddhetukaṃ kaivalyaṃ vyutpādanīyam // (3.1)
	na caitatkaivalyabhāgīyaṃ cittaṃ paralokaṃ ca paralokinaṃ vijñānātiriktaṃ cittakaraṇasukhādyātmakaśabdādyupabhoktāram ātmānaṃ ca prasaṃkhyānaparamakāṣṭhāṃ ca vinā vyutpādya śakyaṃ vaktumiti tad etat sarvam atra pāde vyutpādanīyam itaracca prasaṅgād upodghātād vā // (4.1)
	tatra prathamaṃ siddhacitteṣu kaivalyabhāgīyaṃ cittaṃ nirdhārayitukāmaḥ pañcatayīṃ siddhim āha janmauṣadhimantratapaḥsamādhijāḥ siddhayaḥ // (5.1)
	vyācaṣṭe dehāntariteti // (6.1)
	svargopabhogabhāgīyāt karmaṇo manuṣyajātīyācaritāt kutaścin nimittāl labdhaparipākāt kvacid devanikāye jātamātrasyaiva divyadehāntaritā siddhiraṇimādyā bhavatīti // (7.1)
	oṣadhisiddhim āha asurabhavaneṣviti // (8.1)
	manuṣyo hi kutaścinnimittād asurabhavanam upasaṃprāptaḥ kamanīyābhir asurakanyābhir upanītaṃ rasāyanam upayujyājarāmaraṇatvam anyāśca siddhīr āsādayati // (9.1)
	ihaiva vā rasāyanopayogena yathā māṇḍavyo munī rasopayogād vindhyavāsīti // (10.1)
	mantrasiddhim āha mantrair iti // (11.1)
	tapaḥsiddhimāha tapaseti // (12.1)
	saṃkalpasiddhim āha kāmarūpīti // (13.1)
	yadeva kāmayate'ṇimādi tadekapade 'sya bhavatīti // (14.1)
	yatra kāmayate śrotuṃ vā mantuṃ vā tatra tadeva śṛṇoti manute veti // (15.1)
	ādiśabdāddarśanādayaḥ saṃgṛhītā iti // (16.1)
	samādhijāḥ siddhayo vyākhyātā adhastane pāde // (17.1)
