śrīdevy uvāca / (1.1)
	brūhi me jagatāṃ nātha sarvavidyāmaya prabho / (1.2)
	mahāvidyāsu sarvāsu pūjyāsu bhuvanatraye // (1.3)
	etāsāṃ dakṣiṇe bhāge nānārūpaṃ pinākadhṛk / (2.1)
	pṛthak pṛthak mahādeva kathayasva mayi prabho // (2.2)
	śrīśiva uvāca / (3.1)
	śṛṇu cārvaṅgi subhage kālikāyāśca bhairavam / (3.2)
	mahākālaṃ dakṣiṇāyā dakṣabhāge prapūjayet // (3.3)
	mahākālena vai sārdhaṃ dakṣiṇā ramate sadā / (4.1)
	tārāyā dakṣiṇe bhāge akṣobhyaṃ paripūjayet // (4.2)
	samudramathane devi kālakūṭaṃ samutthitam / (5.1)
	sarve devāḥ sadārāśca mahākṣobham avāpnuyuḥ // (5.2)
	kṣobhādirahitaṃ yasmāt pītaṃ hālāhalaṃ viṣam / (6.1)
	ata eva maheśāni akṣobhyaḥ parikīrtitaḥ // (6.2)
	tena sārdhaṃ mahāmāyā tāriṇī ramate sadā / (7.1)
	mahātripurasundaryā dakṣiṇe pūjayecchivam // (7.2)
	pañcavaktraṃ trinetraṃ ca prativaktre sureśvari / (8.1)
	tena sārdhaṃ mahādevī sadā kāmakutūhalā // (8.2)
	ata eva maheśāni pañcamīti prakīrtitā / (9.1)
	śrīmadbhuvanasundaryā dakṣiṇe tryambakaṃ yajet // (9.2)
	svarge martye ca pātāle yā cādyā bhuvaneśvarī / (10.1)
	etāsu ramate yena tryambakastena kathyate // (10.2)
	saśaktiśca samākhyātaḥ sarvatantraprapūjitaḥ / (11.1)
	bhairavyā dakṣiṇe bhāge dakṣiṇāmūrtisaṃjñakam // (11.2)
	pūjayet parayatnena pañcavaktraṃ tameva hi / (12.1)
	chinnamastādakṣiṇāṃśe kabandhaṃ pūjayecchivam // (12.2)
	kabandhapūjanāddevi sarvasiddhīśvaro bhavet / (13.1)
	dhūmāvatī mahāvidyā vidhavārūpadhāriṇī // (13.2)
	vagalāyā dakṣabhāge ekavaktraṃ prapūjayet / (14.1)
	mahārudreti vikhyātaṃ jagatsaṃhārakārakam // (14.2)
	mātaṃgīdakṣiṇāṃśe ca mataṃgaṃ pūjayecchivam / (15.1)
	tameva dakṣiṇāmūrtiṃ jagadānandarūpakam // (15.2)
	kamalāyā dakṣiṇāṃśe viṣṇurūpaṃ sadāśivam / (16.1)
	pūjayet parameśāni sa siddho nātra saṃśayaḥ // (16.2)
	pūjayedannapūrṇāyā dakṣiṇe brahmarūpakam / (17.1)
	mahāmokṣapradaṃ devaṃ daśavaktraṃ maheśvaram // (17.2)
	durgāyā dakṣiṇe deśe nāradaṃ paripūjayet / (18.1)
	nākāraḥ sṛṣṭikartā ca dakāraḥ pālakaḥ sadā // (18.2)
	rephaḥ saṃhārarūpatvānnāradaḥ parikīrtitaḥ / (19.1)
	anyāsu sarvavidyāsu ṛṣir yaḥ parikīrtitaḥ // (19.2)
	sa eva tasyā bhartā ca dakṣabhāge prapūjayet / (20.1)
	śrīdevī uvāca / (20.2)
	yā cādyā paramā vidyā dvitīyā bhairavī parā / (20.3)
	trailokyajananī nityā sā kathaṃ śavavāhanā // (20.4)
	śrīśiva uvāca / (21.1)
	yā cādyā parameśāni svayaṃ kālasvarūpiṇī / (21.2)
	śrīśivasya hṛdambhoje sthitā saṃhārarūpiṇī // (21.3)
	ata eva mahākālo jagatsaṃhārakārakaḥ / (22.1)
	saṃhārarūpiṇī kālī yadā vyaktasvarūpiṇī // (22.2)
	tadaiva sahasā devi śavarūpaḥ sadāśivaḥ / (23.1)
	tatkṣaṇāc cañcalāpāṅgi sā devī śavavāhanā // (23.2)
	śrīdevī uvāca / (24.1)
	śavarūpo mahādeva mṛtadehaḥ sadāśivaḥ / (24.2)
	mṛtadehaṃ mahādeva salilaṃ vā kathaṃ nahi // (24.3)
	śrīśiva uvāca / (25.1)
	yasmin vyaktā mahākālī śaktihīnaḥ sadāśivaḥ / (25.2)
	śaktyā yukto yadā devi tadaiva śivarūpakaḥ / (25.3)
	śaktihīnaḥ śavaḥ sākṣāt puruṣatvaṃ na muñcati // (25.4)
