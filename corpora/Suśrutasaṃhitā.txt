athāto vedotpattimadhyāyaṃ vyākhyāsyāmaḥ // (1.1)
	yathovāca bhagavān dhanvantariḥ // (2.1)
	atha khalu bhagavantam amaravaram ṛṣigaṇaparivṛtam āśramasthaṃ kāśirājaṃ divodāsaṃ dhanvantarim aupadhenavavaitaraṇaurabhrapauṣkalāvatakaravīryagopurarakṣitasuśrutaprabhṛtaya ūcuḥ // (3.1)
	bhagavan śārīramānasāgantuvyādhibhir vividhavedanābhighātopadrutān sanāthān apy anāthavad viceṣṭamānān vikrośataś ca mānavānabhisamīkṣya manasi naḥ pīḍā bhavati teṣāṃ sukhaiṣiṇāṃ rogopaśamārthamātmanaś ca prāṇayātrārthaṃ prajāhitahetor āyurvedaṃ śrotum icchāma ihopadiśyamānam atrāyattam aihikam āmuṣmikaṃ ca śreyaḥ tadbhagavantam upapannāḥ smaḥ śiṣyatveneti // (4.1)
	tān uvāca bhagavān svāgataṃ vaḥ sarva evāmīmāṃsyā adhyāpyāś ca bhavanto vatsāḥ // (5.1)
	iha khalv āyurvedo nāmopāṅgam atharvavedasyānutpādya iva prajāḥ ślokaśatasahasram adhyāyasahasraṃ ca kṛtavān svayambhūḥ tato 'lpāyuṣṭvam alpamedhastvaṃ cālokya narāṇāṃ bhūyo 'ṣṭadhā praṇītavān // (6.1)
	tadyathā śalyaṃ śālākyaṃ kāyacikitsā bhūtavidyā kaumārabhṛtyam agadatantraṃ rasāyanatantraṃ vājīkaraṇatantram iti // (7.1)
	athāsya pratyaṅgalakṣaṇasamāsaḥ / (8.1)
	tatra śalyaṃ nāma vividhatṛṇakāṣṭhapāṣāṇapāṃśulohaloṣṭāsthivālanakhapūyāsrāvaduṣṭavraṇāntargarbhaśalyoddharaṇārthaṃ yantraśastrakṣārāgnipraṇidhānavraṇaviniścayārthaṃ ca / (8.2)
	śālākyaṃ nāmordhvajatrugatānāṃ śravaṇanayanavadanaghrāṇādisaṃśritānāṃ vyādhīnām upaśamanārtham / (8.3)
	kāyacikitsā nāma sarvāṅgasaṃśritānāṃ vyādhīnāṃ jvararaktapittaśoṣonmādāpasmārakuṣṭhamehātisārādīnām upaśamanārtham / (8.4)
	bhūtavidyā nāma devāsuragandharvayakṣarakṣaḥpitṛpiśācanāgagrahādyupasṛṣṭacetasāṃ śāntikarmabaliharaṇādigrahopaśamanārtham / (8.5)
	kaumārabhṛtyaṃ nāma kumārabharaṇadhātrīkṣīradoṣasaṃśodhanārthaṃ duṣṭastanyagrahasamutthānāṃ ca vyādhīnām upaśamanārtham / (8.6)
	agadatantraṃ nāma sarpakīṭalūtāmūṣakādidaṣṭaviṣavyañjanārthaṃ vividhaviṣasaṃyogopaśamanārthaṃ ca / (8.7)
	rasāyanatantraṃ nāma vayaḥsthāpanam āyurmedhābalakaraṃ rogāpaharaṇasamarthaṃ ca / (8.8)
	vājīkaraṇatantraṃ nāmālpaduṣṭakṣīṇaviśuṣkaretasām āpyāyanaprasādopacayajanananimittaṃ praharṣajananārthaṃ ca // (8.9)
	evam ayam āyurvedo 'ṣṭāṅga upadiśyate atra kasmai kim ucyatām iti // (9.1)
	ta ūcuḥ asmākaṃ sarveṣām eva śalyajñānaṃ mūlaṃ kṛtvopadiśatu bhagavān iti // (10.1)
	sa uvācaivam astv iti // (11.1)
	ta ūcur bhūyo 'pi bhagavantam asmākam ekamatīnāṃ matam abhisamīkṣya suśruto bhagavantaṃ prakṣyati asmai copadiśyamānaṃ vayam apy upadhārayiṣyāmaḥ // (12.1)
	sa hovācaivam astv iti // (13.1)
	vatsa suśruta iha khalv āyurvedaprayojanaṃ vyādhyupasṛṣṭānāṃ vyādhiparimokṣaḥ svasthasya rakṣaṇaṃ ca // (14.1)
	āyur asmin vidyate 'nena vāyur vindatīty āyurvedaḥ // (15.1)
	tasyāṅgavaramādyaṃ pratyakṣāgamānumānopamānair aviruddham ucyamānam upadhāraya // (16.1)
	etaddhy aṅgaṃ prathamaṃ prāg abhighātavraṇasaṃrohād yajñaśiraḥsaṃdhānāc ca / (17.1)
	śrūyate hi yathā rudreṇa yajñasya śiraśchinnamiti tato devā aśvināv abhigamyocur bhagavantau naḥ śreṣṭhatamau yuvāṃ bhaviṣyathaḥ bhavadbhyāṃ yajñasya śiraḥ saṃdhātavyam iti / (17.2)
	tāv ūcatur evam astv iti / (17.3)
	atha tayor arthe devā indraṃ yajñabhāgena prāsādayan / (17.4)
	tābhyāṃ yajñasya śiraḥ saṃhitam iti // (17.5)
	aṣṭāsv api cāyurvedatantreṣv etad evādhikam abhimatam āśukriyākaraṇādyantraśastrakṣārāgnipraṇidhānāt sarvatantrasāmānyāc ca // (18.1)
	tad idaṃ śāśvataṃ puṇyaṃ svargyaṃ yaśasyamāyuṣyaṃ vṛttikaraṃ ceti // (19.1)
	brahmā provāca tataḥ prajāpatir adhijage tasmād aśvinau aśvibhyām indraḥ indrād ahaṃ mayā tv iha pradeyam arthibhyaḥ prajāhitahetoḥ // (20.1)
	bhavati cātra / (21.1)
	ahaṃ hi dhanvantarirādidevo jarārujāmṛtyuharo 'marāṇām / (21.2)
	śalyāṅgamaṅgair aparair upetaṃ prāpto 'smi gāṃ bhūya ihopadeṣṭum // (21.3)
	asmin śāstre pañcamahābhūtaśarīrisamavāyaḥ puruṣa ity ucyate / (22.1)
	tasmin kriyā so 'dhiṣṭhānaṃ kasmāt lokasya dvaividhyāt loko hi dvividhaḥ sthāvaro jaṅgamaś ca dvividhātmaka evāgneyaḥ saumyaś ca tadbhūyastvāt pañcātmako vā tatra caturvidho bhūtagrāmaḥ saṃsvedajarāyujāṇḍajodbhijjasaṃjñaḥ tatra puruṣaḥ pradhānaṃ tasyopakaraṇamanyat tasmāt puruṣo 'dhiṣṭhānam // (22.2)
	tadduḥkhasaṃyogā vyādhaya ucyante // (23.1)
	te caturvidhāḥ āgantavaḥ śārīrāḥ mānasāḥ svābhāvikāśceti // (24.1)
	teṣāmāgantavo 'bhighātanimittāḥ / (25.1)
	śārīrās tv annapānamūlā vātapittakaphaśoṇitasaṃnipātavaiṣamyanimittāḥ / (25.2)
	mānasāstu krodhaśokabhayaharṣaviṣāderṣyābhyasūyādainyamātsaryakāmalobhaprabhṛtaya icchādveṣabhedair bhavanti / (25.3)
	svābhāvikāstu kṣutpipāsājarāmṛtyunidrāprakṛtayaḥ // (25.4)
	ta ete manaḥśarīrādhiṣṭhānāḥ // (26.1)
	teṣāṃ saṃśodhanasaṃśamanāhārācārāḥ samyakprayuktā nigrahahetavaḥ // (27.1)
	prāṇināṃ punarmūlam āhāro balavarṇaujasāṃ ca sa ṣaṭsu raseṣvāyattaḥ rasāḥ punardravyāśrayāḥ dravyāṇi punar oṣadhayaḥ / (28.1)
	tāstu dvividhāḥ sthāvarā jaṅgamāś ca // (28.2)
	tāsāṃ sthāvarāścaturvidhāḥ vanaspatayo vṛkṣā vīrudha oṣadhaya iti / (29.1)
	tāsu apuṣpāḥ phalavanto vanaspatayaḥ puṣpaphalavanto vṛkṣāḥ pratānavatyaḥ stambinyaś ca vīrudhaḥ phalapākaniṣṭhā oṣadhaya iti // (29.2)
	jaṅgamāḥ khalv api caturvidhāḥ jarāyujāṇḍajasvedajodbhijjāḥ / (30.1)
	tatra paśumanuṣyavyālādayo jarāyujāḥ khagasarpasarīsṛpaprabhṛtayo 'ṇḍajāḥ kṛmikīṭapipīlikāprabhṛtayaḥ svedajāḥ indragopamaṇḍūkaprabhṛtaya udbhijjāḥ // (30.2)
	tatra sthāvarebhyas tvakpattrapuṣpaphalamūlakandaniryāsasvarasādayaḥ prayojanavantaḥ jaṅgamebhyaś carmanakharomarudhirādayaḥ // (31.1)
	pārthivāḥ suvarṇarajatamaṇimuktāmanaḥśilāmṛtkapālādayaḥ // (32.1)
	kālakṛtāḥ pravātanivātātapachāyājyotsnātamaḥśītoṣṇavarṣāhorātrapakṣamāsartvayanādayaḥ saṃvatsaraviśeṣāḥ // (33.1)
	ta ete svabhāvata eva doṣāṇāṃ saṃcayaprakopapraśamapratīkārahetavaḥ prayojanavantaś ca // (34.1)
	bhavanti cātra ślokāḥ / (35.1)
	śārīrāṇāṃ vikārāṇām eṣa vargaś caturvidhaḥ / (35.2)
	prakope praśame caiva hetur uktaś cikitsakaiḥ // (35.3)
	āgantavas tu ye rogās te dvidhā nipatanti hi / (36.1)
	manasyanye śarīre 'nye teṣāṃ tu dvividhā kriyā // (36.2)
	śarīrapatitānāṃ tu śārīravad upakramaḥ / (37.1)
	mānasānāṃ tu śabdādiriṣṭo vargaḥ sukhāvahaḥ // (37.2)
	evam etat puruṣo vyādhir auṣadhaṃ kriyākāla iti catuṣṭayaṃ samāsena vyākhyātam tatra puruṣagrahaṇāt tatsambhavadravyasamūho bhūtādir uktas tadaṅgapratyaṅgavikalpāś ca tvaṅmāṃsāsthisirāsnāyuprabhṛtayaḥ vyādhigrahaṇād vātapittakaphaśoṇitasaṃnipātavaiṣamyanimittāḥ sarva eva vyādhayo vyākhyātāḥ oṣadhagrahaṇād dravyarasaguṇavīryavipākānām ādeśaḥ kriyāgrahaṇācchedyādīni snehādīni ca karmāṇi vyākhyātāni kālagrahaṇāt sarvakriyākālānām ādeśaḥ // (38.1)
	bhavati cātra / (39.1)
	bījaṃ cikitsitasyaitatsamāsena prakīrtitam / (39.2)
	saviṃśam adhyāyaśatamasya vyākhyā bhaviṣyati // (39.3)
	tac ca saviṃśam adhyāyaśataṃ pañcasu sthāneṣu sūtranidānaśārīracikitsitakalpeṣv arthavaśāt saṃvibhajya uttare tantre śeṣānarthān vyākhyāsyāmaḥ // (40.1)
	bhavati cātra / (41.1)
	svayambhuvā proktamidaṃ sanātanaṃ paṭheddhi yaḥ kāśipatiprakāśitam / (41.2)
	sa puṇyakarmā bhuvi pūjito nṛpair asukṣaye śakrasalokatāṃ vrajet // (41.3)
